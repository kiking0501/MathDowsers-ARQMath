Infinite set always has a countably infinite subset

I'm trying to show that one infinite always has a countably infinite subset, but I'm confused with something on the proof. Let S be one infinite set. In that case, to show it has one countably infinite subset means to find a subset S0⊂S such that there is a bijection between S0 and ℕ.

The procedure I considered is: pick one arbitrary a0∈S and consider S∖{a0}. This new set will also be infinite, so we can pick a1∈S∖{a0} and consider S∖{a0,a1}. This new set is again infinite. We can repeat this procedure n times, for every n∈ℕ and thus construct a set {a0,a1,…} which will clearly be in bijection to ℕ.

Now there is a point I'm quite unsure. This idea that: we pick a0 from S then we pick a1 from S∖{a0} and so forth. It seems to me that in the way I wrote, this is not fully rigorous.

Is that the case? If so, how can this idea be made rigorous?

Welcome to the wormhole.

A set which has a countably infinite subset is called Dedekind Infinite. It is not provable within Zermelo-Fraenkel set theory (ZF) that every infinite set is Dedekind infinite. This was demonstrated by Paul Cohen, who exhibited a model of ZF in which there is an infinite, Dedekind-finite set of real numbers. See Asaf's answer to my question for implications of this statement in analysis.

You do not need any form of the axiom of choice to prove that infinite equals Dedekind infinite, in the sense that one can construct models of ZF in which all infinite sets have countably infinite subsets but no form of the axiom of choice holds. However, if we do have some form of the axiom of choice available, even a weak form, then we can prove your statement.

Let me explain. The axiom of choice states that, given any collection (Ai:i∈I) of nonempty sets, indexed by some set I, there exists a choice function picking out, for each i∈I, an element ai∈Ai. In other words, we have a function f:I→⋃iAi such that f⁢(i)∈Ai for each i.

A well-known (but not easy to prove) consequence of the axiom of choice is the well-ordering theorem: given any set A, we can order the elements of A in such a way that any nonempty subset of A has a least element under the ordering. In fact, the well-ordering theorem is equivalent to the axiom of choice, since, given (Ai:i∈I), we can define f⁢(i) to be the least element of Ai, under this ordering, giving us a choice function.

With this form of the axiom of choice, one can prove your statement as follows: given your set S, order it according to the well-ordering theorem. Then let a0 be the least element of S, let a1 be the least element of S∖{a0} and so on. This process will never terminate; otherwise, S would be equal to {a0,…,an} for some n and therefore would be finite.

That's all well and good, but the well-ordering theorem is a non-obvious theorem in set theory, and certainly isn't implicit in your argument. We shouldn't need to use it to prove this statement.

Thankfully, there is another way, and it actually involves a strictly weaker form of the axiom of choice. The axiom of countable choice is the same as the axiom of choice, except that we require the set I to be countable. There are models of set theory in which the axiom of choice fails but still holds whenever the index set is countable, so countable choice is strictly weaker than full choice.

Here's how we prove your statement using only countable choice. Let S be an infinite set. Then, for each n=0,1,2,…, let Sn be the set of subsets of S of size 2n.

We can show that each Sn is non-empty using your argument (note that the axiom of choice is not required for a finite collection of sets, since we can proceed by induction): inductively pick elements a1,…,a2n from S; since S is infinite, this process will not terminate and since we are only choosing finitely many elements, there are no dodgy set-theory issues.

Now, there are countably many Sn, so the axiom of countable choice tells us that there is a sequence A0,A1,A2,… of subsets of S such that An has size 2n for each n.

Now, for each n, let Bn be the set of elements of An that are not contained in A1∪⋯∪An-1. Since An has size 2n, and A1,…,An-1 have at most 1+2+4+⋯+2n-1=2n-1 distinct elements between them, Bn must be non-empty. By the axiom of countable choice again, we can choose a sequence s0,s1,… with each sn∈Bn. In particular, the sn are distinct. We have found a countably infinite subset of S.
