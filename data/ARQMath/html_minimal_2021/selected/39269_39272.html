The idea, in a nutshell, is to assume by contradiction that the reals are countable. Due to that you can write them as ai for i∈ℕ. This list contains all the real numbers.

By taking the number produced by the diagonal argument you ensure at the n-th step that you are not any of a1,…,an. When the induction "ends" you have produced a real number which is none of the ai,i∈ℕ.

This could only mean one thing: the enumeration did not capture all the real numbers like you assumed it to.

Since the enumeration was not specific, but arbitrary, we deduce that any countable enumeration of the reals will not cover the entire real numbers. This, by definition, means that the real numbers are uncountable.

Addendum:

After reading more carefully most of this thread (on its comments, and so on) I think that I should add a few words about proof by contradiction - which is a very common method in mathematics.

"Mathematics is a science of deductions", said my linear algebra prof. on the first day of my B.Sc.

"We assume A,B,C and deduce D,E,F", he continued.

The important thing about the ability to deduce things, is to remain consistent. In simple words it means not to be able to prove both something and its negation. Why is this important? Because it is simple to prove anything from contradiction, in fact according to this xkcd it is even possible to derive phone numbers (although I never managed to do that myself).

Proof by contradiction exploits the assumption that we use a consistent axiomatic system. If we assumed one thing, and derived from it contradiction then our assumption was false.

Since something is either true or false, if it is false then its negation is true.

So what did I do in the proof above? I started by assuming mathematics as we know it is consistent, then added the assumption "The reals are countable". If they are countable, then by definition we can list them as above.

The diagonal argument shows that regardless to how you are going to list them, countably many indices is not enough, and for every list we can easily manufacture a real number not present on it. From this we deduce that there are no countable lists containing all the real numbers. This is to say that the reals are not countable, which contradicts our assumption that they are countable.

I should add that some of the proofs by contradiction do not actually require the contrapositive assumption. In this case, for example, I could have said "given an arbitrary countable list of real numbers we can procure a real number not on the list" and deduce from that that the real numbers are uncountable. Sometimes, however, contradiction is slightly more essential for the proof.

@johne: Note that (P→¬P)→¬P is a tautology. That means that if from an assumption P you can conclude that P is false, that means that P is false. The argument takes that form: Assume the list contains all reals (P="the list contains all reals"). Using the diagonal argument, you construct a real not on the list (¬⁢P="there is a real not on the list). So P→¬⁢P; hence the conclusion is that ¬⁢P is true (given a list of reals, there is a real not on that last).
@Arturo: (P→¬P)→¬P, or more specifically "P→¬⁢P ", does not reflect what it is you are asserting or trying to prove. What you are trying to prove is P∧¬⁢P : P="the list contains all reals", ¬⁢P="there is a real not on the list". If ¬⁢P is "a real not on the list", then P∪¬⁢P≠P, but P is "a list that contains all reals" if, and only if, P∪¬⁢P=P. Both must be true simultaneously- P∧¬⁢P, not if / then P→¬⁢P.
@johne: If from P you can prove ¬⁢P, then this proves ¬⁢P. This is a simple consequence of the fact that (P→¬P)→¬P is a tautology; that proving A→B is equivalent to assuming A is true and then using that to prove B; and modus ponens. While the "classic" argument by contradiction proceeds by showing that from P you conclude A∧¬⁢A *for some A*, if from P you can conclude ¬⁢P, then you have **both** P and ¬⁢P, which gives you the conjunction you want.
@johne: So that would mean that from P we can conclude P∧¬⁢P, and using this as an argument by contradiction this shows ¬⁢P. But this is all codified in the argument I noted: that if from P you can prove ¬⁢P, then this is equivalent to proving that P→¬⁢P holds (Deduction Theorem); and since (P→¬P)→¬P is a tautology, this and P→¬⁢P together, by Modus Ponens, yield ¬⁢P.
@Arturo: In fact, "hence the conclusion is that ¬⁢P is true" does means _precisely_ "given a list of reals, there is a real not on that last [sic]", which is not the same thing as "given a list of ***all*** reals, there is a real not on that last [sic]", which was the original premiss: P="the list contains ***all*** reals". This, I think, is why I have trouble with the Cantor's diagonal argument: I don't see how it proves anything if you drop the ***all*** qualification.
@johne: You are confusing a *part* of the argument with the argument as a whole. This particular **bit** of argument says "suppose you had a list that (allegedly) contains all reals; then by `blah` we show that the list is missing at least one real; therefore, you do not have a list that contains all reals." Now the *entire* theorem, which is "for all lists, if L is a list of reals, then there is at least one real not on L" follows; take a given list L. It either contains all reals, or not. If it doesn't, done. If it (allegedly) contained all reals, then it doesn't; therefore, it doesn't. Done

@Arturo: P→¬⁢P does not prove Cantors diagonal argument, it only proves that P=⊥. But in order for the proof to work, P ***must*** be ⊤- You can not have a "list of ***all*** reals" (P=⊤) ***and*** "a real that is not on the list" (¬⁢P=⊤). If you have a real that was not on the list, then by P→¬⁢P, you did not have a list of ***all*** the reals, which pretty much scuttles the whole proof since it doesn't work if P≠ "a list of ***all*** the reals".
@johne. There's "Cantor's Diagonal Argument", and then there's "Cantor's Theorem." Cantor's "Diagonal argument" is **only** the part of the proof that shows how, given any list of sequences, you can construct a sequence not on the list. Cantor's **Theorem** is that there is no list of all sequences. **Part** of the proof of the theorem, **not the entirety of it**, is the diagonal argument. **Part** of the proof of the theorem, **separate** from the diagonal argument, can be done by contradiction, assuming there *is* a list of all sequences; only that part is of the form P→¬⁢P.
