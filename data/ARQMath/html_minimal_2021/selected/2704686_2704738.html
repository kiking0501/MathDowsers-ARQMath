Let x1:=a>0 and xn+1:=xn+1/xn for n∈ℕ. Determine whether (xn) converges or diverges.

My Attempt:

We note that x1=a>0. Suppose that xk>0 for some k∈ℕ. Then xk+1=xk+1xk>0. Therefore xn>0⁢ for all ⁢n∈ℕ.

We note that for any natural number n, as xn>0, so we have xn+1=xn+1xn=12⁢[(xn+1xn)2+(xn-1xn)2]≥12⁢[(xn+1xn)2-(xn-1xn)2]=2. Thus we can conclude that xn≥2⁢ for all ⁢n∈ℕ⁢ such that ⁢n≥2.

Now suppose (xn) is convergent, with limit x in ℝ. Then as xn≥2 for all n∈ℕ and n≥2, so we must have x≥2. Then we must have x=x+1x, which implies that 1x=0, which is impossible.

Hence our sequence cannot converge in ℝ.

For each n∈ℕ, as xn>0, so we have xn+1=xn+1xn>xn. Thus our sequence is strictly increasing.

As our sequence is monotonically increasing and not convergent, so it is not bounded from above. Therefore we can conclude that limn→∞⁡xn=+∞.
