The problem is that the integral is like ∫ex2⁢𝑑x and we can't write this down. This doesn't mean that there isn't an antiderivative, just that we can't write it down with the usual functions available.
Note that we don't compute primitives, but rather find them. We compute definite integrals, though.

Note that exp⁡(x2⁢log⁡5)=5x2

You're looking at something of the form ∫eα⁢x2⁢𝑑x

which is the same as finding a primitive for ∫ex2⁢𝑑x

after a change of variables. But this has not a closed form solution in terms of elementary functions, so your original integral has not one either. You can use that if G⁢(x)=∫0xet2⁢𝑑t then G′⁢(x)=exp⁡x2 to find approximations using Taylor expansions.
