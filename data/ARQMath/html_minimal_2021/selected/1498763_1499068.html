The other answers have given you different ways to solve the problem at hand. I will below just add some notes on how one can evaluate general sums on the form ∑n=0∞nkn! where k is any integer using just algebraic manipulations (since you said you cannot use derivatives). I'm however assuming we already know that ∑n=0∞1n!=e.

We start by noting that we can always write nk on the form

nk=C1⁢n+C2⁢n⁢(n-1)+…+Ck⁢n⁢(n-1)⁢⋯⁢(n-k+1)≡∑j=1kCj⁢(nj)⁢j!

where Ci are some real numbers (see below for a simple algorithm to compute these numbers). For example

n2=n⁢(n-1)+n and n4=n⁢(n-1)⁢(n-2)⁢(n-3)+6⁢n⁢(n-1)⁢(n-2)+7⁢n⁢(n-1)+n

The reason this form is useful is that we can very easily evaluate the sums

∑n=0∞n⁢(n-1)⁢⋯⁢(n-i)n!=∑n=i+1∞1(n-i-1)!=∑m=0∞1m!=e

where we have simply shifted the summation index to m=n-i-1 in the last step. The sum above does not depend on i at all so the sum we are after becomes

∑n=0∞nkn!=∑n=0∞C1⁢n+C2⁢n⁢(n-1)+…+Ck⁢n⁢(n-1)⁢⋯⁢(n-k+1)n!=C1⁢e+C2⁢e+…+Ck⁢e=(∑j=1kCj)⁢e

so the problem reduces to finding the Ci's.

Here is a simple method to compute the Ci's. Lets take k=3 as an example and start by writing n3=\color⁢b⁢l⁢u⁢e⁢1⁢n3 Now we subtract \color⁢b⁢l⁢u⁢e⁢1⁢n⁢(n-1)⁢(n-2) to get n3-\color⁢b⁢l⁢u⁢e⁢1⁢n⁢(n-1)⁢(n-2)=\color⁢r⁢e⁢d⁢3⁢n2-2⁢n so C2=\color⁢r⁢e⁢d⁢3. Now subtract \color⁢r⁢e⁢d⁢3⁢n⁢(n-1) to get n3-n⁢(n-1)⁢(n-2)-\color⁢r⁢e⁢d⁢3⁢n⁢(n-1)=\color⁢p⁢i⁢n⁢k⁢1⁢n so C1=\color⁢p⁢i⁢n⁢k⁢1. We have now shown that n3=\color⁢b⁢l⁢u⁢e⁢1⁢n⁢(n-1)⁢(n-2)+\color⁢r⁢e⁢d⁢3⁢n⁢(n-1)+\color⁢p⁢i⁢n⁢k⁢1⁢n so using the formula above we have ∑n=0∞n3n!=(\color⁢b⁢l⁢u⁢e⁢1+\color⁢r⁢e⁢d⁢3+\color⁢p⁢i⁢n⁢k⁢1)⁢e=5⁢e

If you want to read more about this, the numbers Cj are the so-called Stirling numbers of the second kind, Cj≡S⁢(k,j), and the sum ∑j=1kCj=∑j=1kS⁢(k,j)≡Bk are the so-called k'th Bell number (and is always an integer). Using this the final result can be compactly written

∑n=0∞nkn!=e⁢Bk

which is known as Dobinski's formula. Knowing this there exist an even simpler method to compute the Bell numbers using a "Pascal's triangle" approach, see for example this page.
