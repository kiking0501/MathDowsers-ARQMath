Function taking only natural values

Let f:[0,1]→ℕ be a continuous function. Show that f is a constant function.

As f is continuous it follows the Ivp hence between two values f⁢(a) and f⁢(b) there must exist one f⁢(c) belongs to ℕ but this is not possible.

But the above argument is false since ℕ is discrete set. Please help!

Real analysis class

I think your argument suffices, although it's very difficult to tell without knowing your instructor and their expectations. In a real analysis class, the function f is just a function from ℝ to ℝ, whose range happens to lie in ℕ. The same logic works: if f is non-constant, then there exist distinct integers m,n∈ℕ such that f⁢(x)=m and f⁢(y)=n for some x,y∈ℝ. Note that m+0.1 or m-0.1 must lie between m and n, and is not an integer. By the IVT, there must be some z between x and y such that f⁢(z)=m±0.1, which is a contradiction.

If this were a topology class instead, it might be good to put a note about how the topology on ℕ is the subspace topology when considered as a subspace of ℝ. Consequently, we can extend f to a function g:ℝ→ℝ whose continuity is equivalent to the continuity of f. From there, the IVT argument works to show g is continuous, hence f is continuous.
