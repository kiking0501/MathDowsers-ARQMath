Function composition - solving for functions

I have a simple, continuous, real-valued function σ, whose functional form I know, and I know that two other invertible functions, f and g, satisfy the following relationships:

  • f∘σ∘σ∘f-1=1,
  • g∘σ∘σ∘g-1=1,
  • g-1∘σ∘f∘f∘σ∘g-1=1,
  • f-1∘σ∘g∘g∘σ∘f-1=1.

where (a∘b)⁢(x)=a⁢(b⁢(x)) is function composition, a-1 is the inverse of function a, and 1 is the identity function.

How do I go about finding f and g, or any further relationships between them? If σ=1 then f=g=h for any invertible function h seems to work, but how about more generally? Any suggestions of topics to investigate very warmly received.

function-and-relation-composition

We have that

f∘σ∘σ∘f-1=1⇔σ∘σ=1

g∘σ∘σ∘g-1=1⇔σ∘σ=1

g-1∘σ∘f∘f∘σ∘g-1=1⇔σ∘f∘f∘σ=g∘g

f-1∘σ∘g∘g∘σ∘f-1=1⇔σ∘g∘g∘σ=f∘f

If we assume σ=1 then we have

f∘f=g∘g. But we can get f,g from the relation above.

Some solutions:

  • If f=g we are done.
  • Assuming the domain is ℝ and that f∘f=g∘g=1 we can have f⁢(x)=x,g⁢(x)=k-x or f⁢(x)=a-x,g⁢(x)=b-x.
  • Assuming the domain is ℝ∖{0} and that f∘f=g∘g=1 we can have f⁢(x)=1x,g⁢(x)=k-x or f⁢(x)=a-x,g⁢(x)=b-x.

Note that f∘σ∘σ∘f-1=1⇔f∘σ∘σ∘f-1∘f=1∘f⇔f∘σ∘σ=f. And thus f∘σ∘σ=f⇔f-1∘f∘σ∘σ=f-1∘f⇔σ∘σ=1.
Assume σ⁢(x)=1-x. Then f⁢(x)=a-x,g⁢(x)=b-x, and f⁢(x)=x,b⁢(x)=b-x are solutions.
