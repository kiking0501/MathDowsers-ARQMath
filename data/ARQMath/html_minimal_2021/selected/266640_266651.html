(1) It is not clear how “n2≤2n” implies “bn≥nk”. Note that the latter is true only for sufficiently large n (for every b>1 there exists N s.t. for n≥N we have bn≥nk). (2) The fact that bn≥nk doesn't imply that bn/nk→∞ as n→∞.

limn→∞⁡bnnk=∞

You can use the root test, too: limn→∞⁡bnnkn=b>1

Therefore, the limit diverges.

The root test takes the lim of the n-th root of the term: limn→∞⁡|an|n=α.

If α<1 the sum/limit converges.

If α>1 the sum/limit diverges.

If α=1, the root test is inconclusive.
