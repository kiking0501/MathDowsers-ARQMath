There is a unique Möbius transformation mapping a given set of 3 points in the extended complex plane ℂ∪{∞} to another set of 3 points.

You can use that fact to show that your function maps the unit circle to itself. It might map the unit disk to the set of points outside the unit circle, but you can show that it doesn't by looking at the image of any point in the disk, say 0.

By b), fa has an inverse, so it must be bijective. The result follows.

Möbius transformations always have an inverse, so they are bijective. Since we already know the one in question maps the unit circle to itself, it must map the open unit disk either to itself or to z:|z|>1. Any other options can be ruled out because Möbius transformations are holomorphic and, in particular, continuous (this is handwavy but can be made rigorous). By checking what it maps, for example, 0 to, we see that it does indeed map the unit disk to itself.
