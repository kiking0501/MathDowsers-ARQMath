@eyy321 a⁢x≡b(modm) hasasolution if and only if gcd⁡(a,m)∣b. So a⁢x≡gcd⁡(a,m)⁢k(modm) has a solution for any integers a,m,k, even if gcd⁡(a,m)>1.

A few methods to solve a⁢x≡b(modm):

1) Checking x≡{0,1,2,…,m-1}(modm), i.e. using brute force.

2) Doing something similar to this: 3⁢x≡2≡-3(mod5)⇔3x≡-1≡4(mod5)

3) Using inverses / Extended Euclidean Algorithm.

Inverses can always solve a⁢x≡b(modm) when gcd⁡(a,m). The solution is x≡b⁢a-1(modm).

You can find a-1modm by either using Extended Euclidean Algorithm or solving a⁢x≡1(modm).

Using Extended Euclidean Algorithm:

14=14⁢(1)+9⁢(0)9=14⁢(0)+9⁢(1)5=14⁢(1)+9⁢(-1)4=14⁢(-1)+9⁢(2)1=14⁢(2)+9⁢(-3)

Therefore 9-1≡-3≡11(mod14).
