How should I solve this exercise? Find the values of the real parameters a and b for which the following function is continuous on ℝ:

\begin{cases} e^x+a\cos(x) \text{ if } x\le0 \\ x^2+b\sin(x) \text{ if } x>0 \\ \end{cases} Thank you!!!

The necessary and sufficient condition for which a function f⁢(x) is continuous in a point c is that limx→c-⁡f⁢(x)=limx→c+⁡f⁢(x) Note that f⁢(x) is continuous in ]-∞;0[ and ]0;+∞[, because ex, cos⁡(x) and sin⁡(x) are continuous ∀x∈ℝ. The only "critical" point is x=0. By the definition of continuous function you get limx→c-⁡f⁢(x)=limx→c+⁡f⁢(x) e0+a⁢cos⁡(0)=02+b⁢sin⁡(0) 1+a=0⇒a=-1 As regards b, there are insufficient conditions to determine it, so b can be any real number.
