When you wrote "e-s proof" did you mean ε-δ proof? Typeset by: `ε-δ`.

Take ϵ>0. Find k>1 large enough that 1/x<ϵ/4 whenever x≥k. Now, since f⁢(x) is continuous and [0,k+1] is compact, then f restricted to [0,k+1] is uniformly continuous. Let δ1>0 be small enough that if x,y∈[0,k+1] then |x-y|<δ1 implies |f⁢(x)-f⁢(y)|<ϵ. Now, suppose that x∈(k+1,∞) and |x-y|<1. In particular this implies that k<x and k<y. Then |f⁢(x)-f⁢(y)|=|(y+1)⁢sin⁡(x3)-(x+1)⁢sin⁡(y3)(x+1)⁢(y+1)|≤|y⁢sin⁡(x+1)-x⁢sin⁡(y+1)(x+1)⁢(y+1)|+|sin⁡(x+1)-sin⁡(y+1)(x+1)⁢(y+1)|≤2⁢max⁡(x,y)(x+1)⁢(y+1)+2(x+1)⁢(y+1)⁢&⁢l⁢t;2min⁡(x,y)+2(x+1)⁢(y+1)<2⁢1k+2⁢1k<4⁢(ϵ/4)=ϵ. The very last inequality came with our choice of k. So, taking δ=min⁡(δ1,1) will do the trick!

Graphically, you might think of uniform continuity as saying that f can't move vertically too steeply at any points (otherwise any small changes in the domain [controlled by δ] can still result in large changes in the range [controlled by ϵ]). For example, f⁢(x)=1/x is not uniformly continuous on (0,1) since near 0 it grows nearly vertically. Similarly, for any n>1, f⁢(x)=±xn is not uniformly continuous on (0,∞) because as x approaches ∞ the function f moves nearly vertically (up if +xn down if -xn).

No problem! To be sure, the graphically illustration is just for some intuition. It is possible for a function to head vertically "briefly" but still be uniformly continuous. For example f⁢(x)=x on [0,∞) is vertical at x=0, but doesn't travel vertically "long enough" for it to prevent f⁢(x) from uniform continuity. It does, however, prevent f⁢(x) from being Lipshitz.
