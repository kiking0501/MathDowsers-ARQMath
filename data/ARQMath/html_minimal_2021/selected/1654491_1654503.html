In the set of real numbers ℝ, the BW theorem is often written : "Any bounded infinite set has a cluster point". So according to the equivalence, we can say that in ℝ, "any bouded set verifies (BL)", which is not true because we know that only bounded and closed sets do...

EDIT : to be more precise, here is the wrong reasoning:

general-topology compactness

A sequence of points in a bounded but not closed subset will have a cluster point in ℝ, but the cluster point may fail to be a member of the subset -- so it is not necessarily sequentially compact.

OK, so: verify=satisfy=fulfil. Hence, for any infinite unbounded set E of ℝ, there exists at least one real r which is a cluster point of E (which is true for instance for ]0,1]). But we know that for any set of a metric space, the existence of a cluster point is equivalent that this set fulfils (BL) (any covering by open sets can be reduced to a finite covering...). So E fulfils (BL), and so is compact (which is false for ]0,1]).
