(a) X∈𝒯 true, all subsets of X are in the discrete topology on X (and X is of course in any topology on X by the first axiom of topology as well).

(b) {X}∈𝒯: {X} is not a subset of X, as X∉X, so this cannot be true. (assuming the axiom of foundation)

(c) {∅} is also not a subset of X (because otherwise ∅∈X which is false, assuming, as I suppose we must assume here, that all of a to f are different from ∅ (a priori, ∅ can be an element of a set, but as X is enumerated as {a,…,f}, we assume this is not the case here (the question is somewhat badly formulated, IMHO).

(d) ∅ is a member of any topology on X, so true.

(e) ∅∈X we assume is false (see (c)).

(f) {∅}∈X we likewise assume to be false (see (c), we assume a to f are "unrelated" elements, not the empty set, or this set, e.g.)

(g) {a}∈𝒯 is true, because {a}⊂X (because a∈X, by enumeration) and all subsets of X are in 𝒯.

(h) a∈𝒯 is false, as a is not a subset of X (we assume; a priori a={b} could hold, and then a would be a subset of X!; this is part of the bad formulation criticism).

(i) ∅⊆𝒯 is true: all elements of ∅ are in 𝒯, this is an empty statement as there are no elements. Or equivalently, ∅⊆A is true whatever A is.

(j) {a}∈X is false (we assume, see before, as b could equal {a} for all we know... ).

(k) {∅}⊆𝒯 is true, as ∅∈𝒯 (inclusion: every element left is an element right..)

(l) a∈X, true by inspection.

(m) X⊆𝒯, false, as a∈X but not a∈𝒯, see (h).

(n) {a}⊆𝒯 is false, as a∈{a} but not a∈𝒯, see (h) again.

(o) {X}⊆𝒯 is true, as the only element left, X, is indeed an element of 𝒯, see (a).

(p) a⊆𝒯 I call false (it's not known what the elements of a are, if a={b} happened to be true, then it would hold, so really we cannot say based on what's given).

Under the given assumptions, indeed 6 are true.
