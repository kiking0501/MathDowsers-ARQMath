Maximum product a⁢b for a+b=k - proof based on AM-GM inequality

If a,b are two positive real numbers such that their sum is a+b=k. Then the product a⁢b is maximum if and only if a=b=k2.

I proved the above theorem above using (a+b)2 but I want to know how can it be proved using A.M. G.M. inequality.

algebra-precalculus inequality a.m.-g.m.-inequality

AM-GM inequality: a+b2≥a⁢b, with equality if and only if a=b.

Substituting a+b=k and taking squares gives us k24≥a⁢b. So the product a⁢b is bounded from above by a constant, and this upper bound can be achieved only by taking a=b=k2.
