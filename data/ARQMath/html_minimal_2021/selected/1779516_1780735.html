If f is any function then the symbol ∫aaf⁢(x)⁢𝑑x is defined to be 0. That answers the question in your title. But it appears that your question is in the context of substitution during evaluation of an integral. What follows is a detailed answer in that context.

I think there is some typo in the question. If you substitute t=sin⁡x then the integrand x⁢cos⁡x1+sin2⁡x should change to arcsin⁡t1+t2 Let's assume that this typo is corrected. Then we should ideally have (at least that is what OP is expecting) ∫02⁢πx⁢cos⁡x1+sin2⁡x⁢𝑑x=∫00arcsin⁡t1+t2⁢𝑑t The change of variables is based on the following theorem:

If g′ is continuous on [a,b] and f is continuous on g⁢([a,b]) then ∫g⁢(a)g⁢(b)f⁢(t)⁢𝑑t=∫abf⁢(g⁢(x))⁢g′⁢(x)⁢𝑑x

Note that in the above theorem we don't need g to be strictly monotone. For the current question, f⁢(t)=arcsin⁡t1+t2 and g⁢(x)=sin⁡x Clearly g′⁢(x)=cos⁡x is continuous on [0,2⁢π]. However range of g i.e. g⁢([0,2⁢π]) is not just consisting of 0 but is the interval [-1,1]. Moreover f⁢(t) is clearly continuous on this range [-1,1] and hence the substitution is justified and the integral should be 0 by equation (1) (as both the bounds of integration are equal).

Then what is the problem?? Let's see if we have f⁢(g⁢(x))=x1+sin2⁡x for all x∈[0,2⁢π]? Clearly if we put x=π so that g⁢(x)=0 and then f⁢(g⁢(x))=f⁢(0)=0 but RHS of (2) is clearly equal to π when x=π. And therefore we get a problem. The reason is simply that we can not write the part x1+sin2⁡x into the form f⁢(g⁢(x)) for all x∈[0,2⁢π] (the part cos⁡x⁢d⁢x can be safely written in the form g′⁢(x)⁢d⁢x). The substitution t=sin⁡x is thus OK but we are not getting a suitable function f so that the integrand can be expressed in the form f⁢(g⁢(x))⁢g′⁢(x) for all x∈[0,2⁢π].

Let's also see why it is not possible to express the integrand in the form f⁢(g⁢(x))⁢g′⁢(x) in this particular case. As noted above the issue comes in representing x/(1+sin2⁡x) in form f⁢(sin⁡x). The reason is that sometimes we have the integrand not in form of f⁢(g⁢(x))⁢g′⁢(x) but rather in the form of f⁢(x,g⁢(x))⁢g′⁢(x). Thus there is some part of f which is made of g⁢(x) but some part is not made of g⁢(x) but rather made of x. Now the only way to get x from g⁢(x) is to go for the inverse g-1 and this exists only when g⁢(x) is one-one (for continuous functions this implies strict monotone nature). Therefore in cases when the integrand is not of the form f⁢(g⁢(x))⁢g′⁢(x) but rather of the form f⁢(x,g⁢(x))⁢g′⁢(x) then we need to split the interval of integration into multiple sub-intervals such that on each sub-interval there is an inverse of g⁢(x) (which implies g⁢(x) is strictly monotone).

Note that if the integral was ∫02⁢πsin⁡x⁢cos⁡x1+sin2⁡x⁢𝑑x then we don't have the problem mentioned above and t=sin⁡x reduces the above integral to ∫00t1+t2⁢𝑑t=0 The substitution works fine even if we integrate in [0,π] and result is 0.

How to salvage the situation for our original integral (1)?? We split the interval [0,2⁢π] into 4 sub-intervals of equal length. For the interval [0,π/2] the function f is given by f⁢(t)=arcsin⁡t1+t2 For [π/2,π] we have f⁢(t)=π-arcsin⁡t1+t2 For interval [π,3⁢π/2] we have f⁢(t)=π-arcsin⁡t1+t2 and finally for [3⁢π/2,2⁢π] we have f⁢(t)=2⁢π+arcsin⁡t1+t2 (the bounds for integrating with respect to t are easily calculated for these four intervals by using t=sin⁡x). Note that the above choices for f⁢(t) guarantee that the equation f⁢(g⁢(x))=x1+sin2⁡x is satisfied for all x∈[0,2⁢π].

We thus have \begin{align} I &= \int_{0}^{2\pi}\frac{x\cos x}{1 + \sin^{2}x}\,dx\notag\\ &= \int_{0}^{\pi/2}\frac{x\cos x}{1 + \sin^{2}x}\,dx + \int_{\pi/2}^{\pi}\frac{x\cos x}{1 + \sin^{2}x}\,dx\notag\\ &\,\,\,\,\,\,\,\, + \int_{\pi}^{3\pi/2}\frac{x\cos x}{1 + \sin^{2}x}\,dx + \int_{3\pi/2}^{2\pi}\frac{x\cos x}{1 + \sin^{2}x}\,dx\notag\\ &= \int_{0}^{1}\frac{\arcsin t}{1 + t^{2}}\,dt + \int_{1}^{0}\frac{\pi - \arcsin t}{1 + t^{2}}\,dt\notag\\ &\,\,\,\,\,\,\,\, + \int_{0}^{-1}\frac{\pi - \arcsin t}{1 + t^{2}}\,dt + \int_{-1}^{0}\frac{2\pi + \arcsin t}{1 + t^{2}}\,dt\notag\\ &= 2\int_{0}^{1}\frac{\arcsin t}{1 + t^{2}}\,dt + 2\int_{-1}^{0}\frac{\arcsin t}{1 + t^{2}}\,dt\notag\\ &\,\,\,\,\,\,\,\,+ 2\pi\int_{-1}^{0}\frac{dt}{1 + t^{2}} - \pi\int_{-1}^{1}\frac{dt}{1 + t^{2}}\notag\\ &= 2\int_{-1}^{1}\frac{\arcsin t}{1 + t^{2}}\notag\\ &= 0\notag \end{align} So the end result is still 0 as expected by equation (1) but the change of variables in equation (1) is not justified because we don't have suitable functions to meet the criteria for change of variables during integration.
