I think that you have made a little mistake; we need some precise definitions.

Consider first-order logic and a "standard" proof system for it; i.e.logical axioms + inference rules (a so-called Hilbert-style proof system; see Elliott Mendelson, Introduction to mathematical logic (4ed - 1997) or Herbert Enderton, A Mathematical Introduction to Logic (2nd - 2001); for natural deduction, thinghs are a little bit different [more rules, no logical axioms], but the basic concepts are the same; see Dirk van Dalen, Logic and Structure (5th ed - 2013) or Ian Chiswell & Wilfrid Hodges, Mathematical Logic (2007)).

Note. Mendelson uses two inference rules : modus ponens and geenralization, while Enderton uses only modus ponens.

We have to start from the definition of deduction form assumptions [see Enderton, page 111] :

A deduction of φ from Γ is a finite sequence (α0,…⁢αn) of formulas such that αn is φ and for each k≤n, either

(a1) αk is in Γ, or

(a2) αk is in the set Λ of logical axioms, or

(b) αk is obtained by modus ponens from two earlier formulas in the sequence; that is, for some i and j less than k, αj is αi→αk.

If such a deduction exists, we say that φ is deducible from Γ, or that φ is a theorem of Γ, and we write Γ⊢φ.

Consider now, your example : "a proof of the theorem represented by A→C is to be able to deduce C from A".

In this way, we obtain a deduction of C from the assumption A :

A⊢C.

Of course, in the above deduction, we have used also the set Λ of (logical) axioms.

Then we apply Deduction Theorem to get :

that if C was deduced from A, then A→C is provable, i.e. ⊢A→C.

By soundness, we conclude that A→C is a (first-order) valid.

See this post for more details.

Consider now a finite set of sentences Γ={γ1,…,γn}. Then, as you said :

we can join all assumptions into one single formula G=γ1∧γ2∧…∧γn, and simply consider a theorem as a conditional statement G→φ.

For example, consider Group theory [see Mendelson, page 71].

G is the conjunction of the (finite) list of first-order axiom of group theory.

Now, let φ be a theorem whatever of group theory; from the consideartions above, we have :

G⊢φ,

from which we conclude :

⊢G→φ.

The last one is a valid formula of first-order logic; that is it is true in every interpretation. This is consistent with mathematical practice: if we consider a structure that is a group, this will obviously satisfy group axioms, i.e. G.

Being φ a tehorem of group theory, by soundness it will be a logical consequence of G, i.e. it will be true in every interpretation that satisfy G. In conclusion, G→φ will be true in every interpreattion, i.e.valid.

But this does not imply that φ is valid, because it is not a theorem of first-order logic.

In conclusion,

G→φ is a logical theorem (of f-o logic) and thus it is valid.

Instead :

φ is a theorem of group theory but not necessarily a logical one; thus, it is not valid.

Consider Enderton's system (I use it for simplicity, because it has modus ponens as single rule of inference).

The axiom system for f-o logic [see page 112] is :

The logical axioms are then all generalizations of wffs of the following forms, where x and у are variables and α and β are wffs:

  1. Tautologies;

  2. ∀x⁢α→α⁢(t/x), where t is substitutable for x in α;

  3. ∀x(α→β)→(∀xα→∀xβ);

  4. α→∀x⁢α, where x does not occur free in α.

In addition, the equality axioms.

Consider the f-o language for number theory [see Enderton, page 70] : an individual constant : 0, the successor (unary) function : S, the two (binary) functions "sum" : * and "product" : ×.

With this language we build the following instance of axiom 2 :

∀x⁢¬⁡(S⁢(x)=0)→¬⁡(1=0);

where we have used as α the formula : ¬⁡(S⁢(x)=0) and as t the term : S⁢(0), i.e.1.

Being an instance of a first-order axiom, the formula is provable :

⊢∀x⁢¬⁡(S⁢(x)=0)→¬⁡(1=0),

i.e.it is a theorem of f-o logic, because we have proved it using (only) f-o logical axioms.

Thus, by soundness is valid (i.e.true in every interpretation).

Consider now the f-o axiom system for number theory [see Enderton, page 203] ; the first one is :

∀x⁢¬⁡(S⁢(x)=0).

Thus, from the above instance of f-o axiom 2, we have :

∀x⁢¬⁡(S⁢(x)=0)⊢¬⁡(1=0).

In conclusion, ¬⁡(1=0) is a theorem of number theory, because we have proved it (in f-o logic) from the (mathematical) axioms of f-o number theory.

This theorem is not valid; it is a logical consequence of the mathematical axioms of number theory; thus, it will be true in every structure which satisfy f-o number-theory axioms.
