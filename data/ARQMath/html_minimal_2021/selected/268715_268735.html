@nice question ali. +1

Suppose by contradiction that such a function exists. Then it is non-constant.

Let a<b be so that f⁢(a)≠f⁢(b). Then by the IVT f⁢([a,b]) is a non-trivial interval. Let call this interval [c,d].

Thus

f⁢([a,b]∩ℚ)=[c,d]∩(ℝ\ℚ). This implies that f takes a countable set onto an uncountable set, contradiction.
