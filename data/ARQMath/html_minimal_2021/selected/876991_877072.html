From comments above, it appears that I was mistaken in treating the null set as an element, not a possible subset. It appears to be logically inconsistent to have the null set excluded as a subset for all but the "empty" universe. Thanks again to all the commenters.

Consider a new relation ∈′ among sets, defined by x∈′y iff x∈y OR (y is empty AND x=y). Note that the original relation ∈ can be recovered from this ∈′ by: x∈y iff x∈′y AND y∉′y (this uses the axiom of Foundation, which guarantees in particular that y∉y).

If we interpret this relation ∈′ as a belonging relation among sets, then there is no set which is empty-in-the-∈′-sense: it has been replaced by an atom o which satisfies o∈′o (and it is the unique set with this property).

Now we can axiomatize set theory for the ∈′ relation: indeed, take the axiomatization of ZFC and replace every occurrence of "x∈y" by "x∈′y AND y∉′y", and add two further axioms which say that (1) for all x,y, if y∈′y and x∈′y then x=y, and (2) if y is such that there is no z≠y with z∈′y, then in fact y∈′y (equivalently, these axioms can be formulated by saying that "x∈′y iff x∈y OR ((there is no z such that z∈y) AND x=y)" where ∈ has been replaced in the right hand side of the "iff" in the same manner as in the axioms of ZFC; this is logically equivalent to (1) and (2)). It is then easy to see that the original ZFC set theory can be recovered from this ZFC′ by defining x∈y iff x∈′y AND y∉′y. So both theories are equiconsistent (they are co-interpretable).

So yes, this provides an "alternative set theory" in which there is no such thing as the ∈′-empty set: instead, the axioms guarantee the existence of o such that o is the unique ∈′-element of o. (Furthermore, even if we interpret ∈′ as ∈, I think the axioms of ZFC′ minus Foundation′ and the axiom (2) are consistent with ZFC minus Foundation, or something like that. So it's not even so wildly "alternative".)

Of course, this set theory is profoundly uninteresting. It just stupidly replaces the empty set by a self-containing atom by changing the ∈ relation. But it can be done.
