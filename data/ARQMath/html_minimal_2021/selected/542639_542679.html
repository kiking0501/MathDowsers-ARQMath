Your argument that ℬ covers ℤ is correct but unnecessarily complicated, since ℤ∈ℬ. The next bit, however, does not make sense: {k⁢n} is a set containing one integer, k⁢n, and is not in ℬ. If you really meant {k⁢n:k∈ℤ}, which is in ℬ, then N1 and N2 are the same set. Moreover, there are members of ℬ containing n that are not of this form. You need to start with completely arbitrary N1,N2∈ℬ with n∈N1∩N2. Let N1=a1+b1⁢ℤ and N2=a2+b2⁢ℤ; then there are k1,k2∈ℤ such that n=a1+b1⁢k1=a2+b2⁢k2. Now you need to find a,b∈ℤ with b≠0 such that n∈a+b⁢ℤ⊆((a1+b1⁢ℤ)∩(a2+b2⁢ℤ)).

This actually takes a little work. Perhaps the most straightforward approach is to begin by showing that if n∈r+s⁢ℤ, then r+s⁢ℤ=n+s⁢ℤ. Thus, if n∈N1∩N2, then (a1+b1⁢ℤ)∩(a2+b2⁢ℤ)=(n+b1⁢ℤ)∩(n+b2⁢ℤ),

and you need only show that (n+b1⁢ℤ)∩(n+b2⁢ℤ)=n+m⁢ℤ, where m=lcm⁢(b1,b2).

ℤ with this topology is not compact: the set of primes is an infinite, closed, discrete subset. (If p is prime, find an open nbhd of p that contains only multiples of p; if n is composite, find an open nbhd of n that contains only composite numbers.)

Added: To show that the space is Hausdorff, let m,n∈ℤ with m≠n. Let p be any positive integer that does not divide n-m, e.g., a prime larger than |n-m|; then m+p⁢ℤ and n+p⁢ℤ are disjoint open nbhds of m and n, respectively. They are clearly open nbhds of m and n. If k∈(m+p⁢ℤ)∩(n+p⁢ℤ), then there are integers r and s such that k=m+p⁢r=n+p⁢s. But then n-m=p⁢r-p⁢s=p⁢(r-s), where r-s is an integer, so p divides n-m, contradicting the choice of p.

Let P be the set of positive primes. If p∈P, then p⁢ℤ=0+p⁢ℤ is an open nbhd of p that does not contain any other prime: if q is a prime different from p, then p is not a divisor of q, so q∉p⁢ℤ. Thus, (p⁢ℤ)∩P={p}, each p∈P is an isolated point of P, and P is therefore discrete. Now suppose that n∈ℤ∖P. If n is composite, then every multiple of n is either composite or 0, so n⁢ℤ is an open nbhd of n disjoint from P. If n=0, then 4⁢ℤ is an open nbhd of n disjoint from P, since every element of 4⁢ℤ is 0 or a multiple of 4 and therefore not prime. The only remaining possibility is that n=-p for some p∈P. In that case -p+3⁢p⁢ℤ is an open nbhd of n disjoint from P: every element of -p+3⁢p⁢ℤ is a multiple of p, so p is the only prime that could possibly belong to -p+3⁢p⁢ℤ, and it doesn’t, since 23 isn’t an integer.
