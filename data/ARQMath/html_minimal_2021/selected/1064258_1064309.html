Understanding the definition of continuity from real analysis

I've stared at and worked with the definition of continuity of a real valued function at a point for many (like 3) years, but there are some things that have always bothered me about it.

First, here is the definition I'm talking about:

Definition. If f:ℝ→ℝ is a function, and x∈ℝ, we say f is continuous at x if ∀ϵ>0, ∃δ>0 such that |x-y|<δ⟹|f⁢(x)-f⁢(y)|<ϵ.

I understand that this means: given any interval, or ball, around our output f⁢(x), we can always find an interval, or ball, around x that is being mapped into the ball around f⁢(x).

I also understand the following two points, which I think are really important, even though they may seem obvious: 1. Given an ϵ>0, when we find some δ that satisfies this condition, any positive number smaller than δ will also work, so in effect we find infinitely many δ. 2. Given an ϵ>0, if we can find a δ that satisfies this condition, then this δ works for every larger ϵ, too.

So here are my questions:

  1. We often say that intuitively, a function is continuous at a point if "small changes in the input lead to small changes in the output". But what does "small" mean? It can't mean the same thing for input and output because the changes in the input are with respect to δ and the changes in the output are with respect to ϵ... who is to say these two numbers are the same type of "small"?

  2. What is the purpose of starting the definition by looking at changes in the output (i.e., saying ∀ϵ>0 first)? If we are saying small changes in the input lead to small changes in the output, shouldn't we start with a small change in the input and check that it leads to a small change in the output?

  3. Why do many people say, "as we shrink ϵ, δ will shrink"? I don't really see how that follows from the definition. And, for example, a constant function is continuous but does not satisfy this shrinking property.

Here's a point I didn't see mentioned in Ian's answer:

The only counterexamples to (3) are constant. To be more specific:

Claim: Let f⁢(x) be a function continuous at x0, and let δ⁢(ε) be an assignment of an appropriate δ⁢(ε) to each ε>0, a δ⁢(ε) such that |x-x0|<δ⁢(ε)⟹|f⁢(x)-f⁢(x0)|<ε (this is possible by continuity at x0. Then, if f is not constant in a neighborhood of x0, then limε→0⁡δ⁢(ε)=0.

Proof: We prove the contrapositive; suppose limε→0⁡δ⁢(ε)≠0. Then there is some sequence εk→0 and some positive real c such that δ⁢(εk)>c for all k. (Check this - it just comes out of the definition of convergence.) Then for |x-x0|<c, |f⁢(x)-f⁢(x0)|<εk for all k; as εk→0, and c does not depend on k, we see that for |x-x0|<c⟹|f⁢(x)-f⁢(x0)|=0; that is, if |x-x0|<c, then f⁢(x)=f⁢(x0). So f was constant in a neighborhood of x0.
