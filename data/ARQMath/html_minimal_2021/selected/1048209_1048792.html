Not sure why this is not mentioned: a function f⁢(n) from the positive integers to, for example, the positive integers, is called "multiplicative" in the number theory sense if gcd⁡(a,b)=1⟹f⁢(a⁢b)=f⁢(a)⁢f⁢(b). This definition also works if the values of f are allowed to be fractions, real numbers, complex numbers, whatever.

Proposition: if f⁢(n) is multiplicative, then g⁢(n)=∑d|nf⁢(d) is also multiplicative

Proposition: a multiplicative function is determined completely by its values on primes and prime powers

Corollary: two multiplicative functions that agree at primes and prime powers agree for all numbers.

Proposition: Euler's totient function ϕ⁢(n) is multiplicative.
