The familiar "exponent rules" http://mathworld.wolfram.com/ExponentLaws.html, are only valid if all quantities involved are real numbers. This subtlety is not taught when the rules are first introduced because, well, these rules are typically introduced long before students see complex numbers. Then when we move to the field ℂ, suddenly these "iron-clad" rules conflict with complex arithmetic!

Indeed, we have to think of 1/i as the number to which multiplication by i produces 1. In other words, we never want to use the identification i=-1 in ℂ because a is no longer unambiguous, as @mixedmath stated above.
