I think it is basically correct... of course it is just a very specific case of Lagrange's theorem: you take a subgroup H of order 3 (or 4), observe that it must have a coset which is disjoint, and obtain a contradiction because then there would already be too many elements. Also, if you insist on being 'elementary', in my opinion at least, you should not leave the verification that all these elements are different to the reader since it is basically the essence of the proof.
Also I think it should be structured differently: for instance "for every element g∈G we obtain a k∈ℕ such that gk=e. Then we distinguish the following cases (1) k=5 for some g∈G then blah; (2) k=4 for some g∈G then blah; (3) k=3 for some g∈G; (4) k=2 for all g∈G. "That way it is more clear that these cases exhaust all possibilities.
