At every point other than x=1, the function f⁢(x)=x2-1x-1 (depicted above) looks exactly like the function g⁢(x)=x+1. The limit limx→1⁡f⁢(x) doesn't mean "evaluate f⁢(1)," it means "find the value that f gets really close to as x gets really close to 1." As you can see in the picture above, as x approaches 1, f⁢(x) approaches 2.
