So we can get all n such that 3⁢\nmid⁢n or 9∣n.

Note that (3⁢p+k)3≡k3≡k (mod 9) for k∈{-1,0,1}. We can write a=3⁢p+k, b=3⁢q+l and c=3⁢r+m, where k,l,m∈{-1,0,1}. Then a3+b3+c3-3⁢a⁢b⁢c≡k+l+m-3⁢(3⁢p+k)⁢(3⁢q+l)⁢(3⁢r+m)≡k+l+m-3⁢k⁢l⁢m (mod ⁢9) Additionally k+l+m-3⁢k⁢l⁢m≡k+l+m (mod 3) Suppose 3∣n. Then 3∣k+l+m.

If k=0, l=0 or m=0 the other two must be opposite, so k+l+m-3⁢k⁢l⁢m=0 and 9∣n. Else we can only have k,l,m=-1 or k,l,m=1. But in both of these cases we get 9∣n too.

We can conclude the only possible values are such n∈ℤ that 3⁢\nmid⁢n or 9∣n.
