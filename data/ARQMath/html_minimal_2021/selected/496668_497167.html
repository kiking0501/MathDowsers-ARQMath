Alternative argument - set theory problem

I tried to solve the following problem:

Let ℱ be a nonempty family of sets with the following properties:

(a) If X∈ℱ, then there are some Y∈ℱ and Z∈ℱ such that Y∩Z=∅ and Y∪Z=X.

(b) If X∈ℱ, and Y∪Z=X,Y∩Z=∅, then either Y∈ℱ or Z∈ℱ.

Show that there is a decreasing sequence X0⊇X1⊇X2⊇… of sets Xn∈ℱ such that ⋂n=0∞Xn=∅.

(some old Miklos Schweitzer problem)

And I came up with the following proof

Proof: Suppose that for every decreasing sequence (Xn) we have ∩Xn≠∅. Pick now an arbitrary decreasing sequence (Yn) and denote Y=∩Yn. We would like to prove that Y∈ℱ. Suppose that Y∉ℱ. Then we construct the sequence An=Yn∖Y and note that since Yn=An∪Y, by rule (b) either Y or An is in ℱ. Suppose that An∈ℱ for infinitely many n. Then we reach a contradiction with the assumption made, since ∩An=∩(Yn∖Y)=(∩Yn)∖Y=∅. Since this is false, we have Y∈ℱ.

Now we are done, because we have just proven that every chain in ℱ ordered by X≤Y⇔X⊇Y has an upper bound, so by Zorn's lemma, ℱ has maximal elements.

Pick now a maximal element M. If it is not the empty set, then by rule (a) it can be decomposed into two parts which are both in ℱ and one is strictly greater than it (in the proposed ordering). Therefore ∅ is the maximal element of ℱ and we are done since we can choose the sequence Xn=∅.

Is it possible to come up with an ending argument which does not use the constant sequence Xn=∅?

Towards a contradiction assume for every descending sequence {Xn:n∈ω} in ℱ, ⋂nXn is non empty.

Claim: For every X∈ℱ, there are disjoint sets Y,Z∈ℱ contained in X such that (∀W)(Y⊆W⊆Y⋃Z⟹W∈ℱ).

Proof of claim: Fix X∈ℱ. First obtain a disjoint collection {Xn:n∈ω} with Xn∈ℱ, X′=⋃n∈ωXn∈ℱ and X′⊆X. Next inductively try to construct a sequence {An:n∈ω}⊆ℱ such that

(1) A0=X′

(2) An+1⊆An

(3) An+1⁢⋂(⋃i<n+1Xi)=ϕ

(4) An+1⁢⋂Xi∈ℱ for infinitely many i's

Notice that the construction must stop at some finite stage otherwise {An:n∈ω} is a descending sequence in ℱ with empty intersection. Suppose An+1 doesn't exist (and An does). Let m be larger than n such that An⁢⋂Xm∈ℱ. Put Y=An⁢⋂Xn and Z=An⁢⋂Xm. Then Z∈ℱ. If possible, suppose Y⊆W⊆Y⁢⋃Z and W∉ℱ. Then An+1=An\W∈ℱ satisfies clauses (2), (3) and (4) which is impossible. This finishes the proof of claim.

Now fix X∈ℱ. Let Y0,Z0 be like the Y,Z of claim. Let Y1,Z1 be similar sets for Z0 and so on. Then Xn=⋃i≥nYi is a descending sequence in ℱ with empty intersection - Contradiction.
