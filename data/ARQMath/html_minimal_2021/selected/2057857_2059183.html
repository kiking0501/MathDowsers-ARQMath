If I understand you correctly, you have a relation ∼ defined on the set A={0,2,4,6} by a∼b if and only if a+b>2, and the question is whether ∼ is an equivalence relation.

Start with the definition: in order to be an equivalence relation, the relation ∼ must be reflexive, symmetric, and transitive. If you can’t already define those three properties, you should make a point of learning them as soon as possible, because you’ll need to know them.

  • Reflexivity: The relation ∼ is reflexive if a∼a for each a∈A. Go back to the definition of ∼: a∼a if and only if a+a>2, so ∼ is reflexive if and only if a+a>2 for each a∈A. But 0+0=0 is not greater than 2, so ∼ is not reflexive. The fact that 2∼2 (since 2+2=4>2), 4∼4 (since 4+4=8>2), and 6∼6 (since 6+6=12>2) is irrelevant: all it takes is the one failure to show that ∼ is not reflexive.

That already shows that ∼ cannot be an equivalence relation, but let’s check the other two properties of an equivalence relation just for the practice.

  • Symmetry: The relation ∼ is symmetric if whenever a∼b holds, then b∼a also holds. If a∼b holds, then by definition a+b>2; but b+a=a+b, so b+a>2 as well, and therefore by definition b∼a holds. Thus, this relation is symmetric.

  • Transitivity: The relation ∼ is transitive if whenver a∼b and b∼c hold (where a,b, and c are not necessarily distinct elements of A), then a∼c also holds. Since 0+4=4>2, we know that 0∼4. We also just saw that ∼ is symmetric, so 4∼0. Taking a=0,b=4, and c=0, we see that we have an example here in which a∼b and b∼c, but a≁c: 0+0=0⁢&̸⁢g⁢t;2, so 0≁0. Thus, the relation is not transitive.
