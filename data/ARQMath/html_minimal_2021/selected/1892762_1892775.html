Hint: say x is rational, then f⁢(x)=1. In order for f to be continuous, for all ϵ>0, there would have to be a ball Bδ⁢(x) so that |f⁢(x)-f⁢(y)|=|1-f⁢(y)|<ϵ on Bδ⁢(x), i.e. 1-ϵ<f⁢(y)<1+ϵ. However, since ℝ-ℚ is dense in ℝ, there will be y values arbitrarily close to x so that y∉ℚ, and so f⁢(y)=0. So, you just need to find a value for ϵ for which the definition of continuity doesn't work. The process will be similar for irrational x.
