Can an ordinary differential equation contain more than one ordinary derivative to more than one independent variable?

If we have this equation:d⁢fd⁢x+d⁢gd⁢y=f+gIs this an ODE? If yes, can we consider it linear?

ordinary-differential-equations
An ODE can only contain derivatives with respect to one independent variable.

Suppose that I is an intervall in ℝ, f,g:I→ℝ are differentiable and that f′⁢(x)+g′⁢(y)=f⁢(x)+g⁢(y) for all x,y∈I.

Then we have f′⁢(x)-f⁢(x)=g⁢(y)-g′⁢(y) for all x,y∈I.

Thus there is a constant c such that

f′⁢(x)-f⁢(x)=c for all x∈I and g⁢(y)-g′⁢(y)=c for all y∈I.

Can you proceed ?
