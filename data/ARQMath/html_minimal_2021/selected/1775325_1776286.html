There is a simpler way to calculate the limit provided we know the following two facts:

  • ex≥1+x for all x.
  • ex+y=ex⋅ey for all x,y.

For the derivation here we need the first inequality ex≥1+x to hold for all x∈(-1,∞) and not on whole of ℝ. Also we need the second property only for the specific case when y=-x so that ex⁢e-x=1.

Both these facts can be established with some effort (see second half of the answer) via definition ex=limn→∞⁡(1+xn)n I now prove that (ex-1)/x→1 as x→0. First we handle the case when x→0+. The inequality ex≥1+x shows that ex-1x≥1 Further let 0<x<1 and then we have the inequality e-x≥1-x We can take reciprocals to get ex≤11-x Here we make use of the fact that ex+y=ex⁢ey. And this inequality gives us ex-1x≤11-x From equations (2) and (3) we get 1≤ex-1x≤11-x for 0<x<1 and letting x→0+ and using Squeeze theorem we get limx→0+⁡ex-1x=1 Note that the above limit implies that ex→1 as x→0+.

Next we handle the case when x→0-. We put x=-y to get limx→0-⁡ex-1x=limy→0+⁡e-y-1-y=limy→0+⁡ey-1y⋅1ey=1⋅1=1 and using equations (4) and (5) we have limx→0⁡ex-1x=1

For those who are interested in the proof of the properties of ex mentioned at the start of my answer, I provide an outline (with some details to be filled by the reader). Before we can proceed to establish the properties of ex it is necessary to show that the definition of ex via limit in (1) makes sense.

It is easy to prove (and this is available in many textbooks and also on MSE) that the following limit limn→∞⁡(1+1n)n exists and is normally denoted by e. Moreover 2<e<3. Using the existence of limit (6) and a little amount of algebraic manipulation it is easy to prove that if x is a positive integer then limn→∞⁡(1+xn)n=ex For a real number x we need some more work. For x>0 it is easy to see via binomial theorem that sequence (1+(x/n))n is increasing and further if r is a positive integer with r>x then (1+(x/n))n<(1+(r/n))n. Now (1+(r/n))n is also increasing and tends to er hence it follows that (1+(r/n))n≤er for all positive integers n. Hence the sequence (1+(x/n))n is also bounded above by er and thus tends to a limit. It follows that the limit F⁢(x)=limn→∞⁡(1+xn)n exists for all x≥0 and moreover since the sequence is increasing the limit is not less than the value of sequence at n=1. Thus we see that F⁢(x)=limn→∞⁡(1+xn)n≥1+x for all x≥0. Thus F⁢(x)>0 for all x≥0.

For x<0 let us put y=-x so that y>0. The limit F⁢(y)=limn→∞⁡(1+yn)n exists and is positive. Further we know via Bernoulli's Inequality that if n>|x| then 1-x2n≤(1-x2n2)n≤1 and hence on taking limit when n→∞ we get via Squeeze Theorem limn→∞⁡(1-x2n2)n=1 Dividing (8⁢b) by (8⁢a) (and noting that y=-x) we see that the limit limn→∞⁡(1+xn)n exists and is positive. Thus we have proved that the limit F⁢(x)=limn→∞⁡(1+xn)n exists for all x and is positive and the definition (1) is now justified. Moreover equation (8⁢b) shows that F⁢(x)⁢F⁢(-x)=1 for all x. Therefore we have F⁢(-x)=1/F⁢(x)=limx→∞⁡(1+xn)-n and using binomial theorem for general index and assuming 0<x<1 we can prove that F⁢(-x)≥1-x. Combined with equation (7) it follows that F⁢(x)≥1+x for all x∈(-1,∞) and this establishes the first property of F⁢(x). Also note that we have proved the specific case of second property namely F⁢(-x)⁢F⁢(x)=1 which was required in our solution, but it is easy to prove in general that Ai↓̸∅. For this we use the following lemma:

Lemma 1: If |1-z|≥δ is a sequence of real or complex terms such that n⁢(xn-1)→0 as n→∞ then xnn→1 as n→∞.

This lemma is proved in this answer. Now let us define the sequence |1-z|≥δ by xn=(1+x+yn)(1+xn)⁢(1+yn) and it is easy to check that n⁢(xn-1)→0 and therefore xnn→1 so that Ai↓̸∅ for all x,y.
