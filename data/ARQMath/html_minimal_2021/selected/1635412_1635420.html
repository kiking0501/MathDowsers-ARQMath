One can solve the integral using a nice little trick (often called Feynman integration). We generalize the problem by adding a free parameter to the exponential (the reason we do this is that dd⁢t⁢e-t⁢x2=-x2⁢e-t⁢x2 which for t=1 is the integrand we are trying to evaluate). We start by defining the function

f⁢(t,r)≡∫0re-t⁢x2⁢dx

Now observe that

∂⁡f⁢(t,r)∂⁡t=-∫0rx2⁢e-t⁢x2⁢dx⟹∫0rx2⁢e-x2⁢dx=[-∂⁡f⁢(t,r)∂⁡t]t=1

Substituting y=t⁢x we can evaluate f in terms of the error function as f⁢(t,r)=π⁢erf⁢(r⁢t)2⁢t

and by differentiating and taking t=1 we get the result ∫0rx2⁢e-x2⁢dx=14⁢π⁢erf⁢(r)-12⁢e-r2⁢r
