There is a problem with your definition of ∞.

You can't say that ∞ is the number larger than any finite number, because ∞ is not a number in the first place. You can't say that some set of numbers (ℕ or ℝ, for example) contains ∞ if you want usual rules to hold; for example from ∞+a=∞ Subtract ∞ to get

a=0 for every a. This is clearly absurd.

So ∞ is not a number, and 1∞ does not mean anything. You can say that

limx→∞⁡1x=0 but this has a very precise meaning and you should work with that.

This is defined as

∀ϵ&gt;0∃N:x&gt;N⟹|1x|&lt;ϵ

It basically means: for very number greater than zero, if you take numbers that are big enough, eventually 1x will become smaller than the number you've chosen at the beginning.

You can see that this responds very well to what we think of infinity (numbers as big as you want!) but the actual object of ∞ is nowhere to be seen.
