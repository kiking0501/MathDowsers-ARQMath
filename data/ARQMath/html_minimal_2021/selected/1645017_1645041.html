Hint: If you want to use the Squeeze Theorem, you can proceed geometrically and observe that for x≠0 \begin{align}\frac12|\sin x| \le \frac12|x|\le \frac12|\tan x| &\implies 1\le \frac{|x|}{|\sin x|}\le \frac{1}{|\cos x|}\\ &\implies |\cos x|\le \frac{|\sin x|}{|x|}\le 1\\&\implies\cos^2x\le \frac{\sin^2 x}{x^2}\le1\end{align}

This geometric interpretation pre-supposes that you have chosen a scale for the angle that already satisfies this inequality resp. has the limit as 1. Remember, geometrically the angle is an equivalence class of ordered pairs of intersecting lines.
