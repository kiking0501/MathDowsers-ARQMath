Fourier transform of an aperiodic function

Given a square-integrable function f⁢(t)∈ℝ defined on t∈ℝ which is both

  1. Aperiodic in the sense that f⁢(t)=f⁢(t+T) if and only if T=0
  2. Discretely valued in the sense that f⁢(t)∈{f1,f2⁢…}

where the fi are either (a) a finite number or (b) a countably infinite number of distinct real values.

My expectation is that, in both cases (a) and (b), the Fourier transform f^⁢(ω) will be decaying as f^⁢(ω)∼O⁢(ω-1) for large |ω|, arguing simply that f⁢(t) must have at least one discontinuity, and the Fourier transform of the step function decays this way.

Is this intuitive picture sufficient to argue this? Or are there cases where it is incorrect.

Periodic functions are not L2 but their Fourier transform still makes sense in the sense of distribution theory. This regularization by taking "small" decay (which is actually quite large decay from a mathematician's or statistician's perspective) converges only in the sense of distribution theory.

(b) If countably many values are allowed, the tail of the Fourier transform can be heavier than |ω|-1. Indeed, if f⁢(x)=|x|-α with 0<α<1/2 near zero (and is truncated at infinity), the Fourier transform will behave like |ω|α-1 at infinity. (Why? The decay of Fourier transform is controlled by ∥τ1/ω⁢f-f∥1 where τ is the translation operator. Translating |x|-α by 1/ω and subtracting cancels off some of it, but it leaves a neighborhood of size 1/|ω| around 0 without much cancellation, and this neighborhood contributes |1/ω|-α+1 to the L1 norm.)

A countably-valued function can have this type of singularity at 0 too, just discretize |x|-α at the scales 2-k<|x|<21-k.

(a) If a function takes on finitely many values on interval, that is, f⁢(x)=∑ak⁢χIk where each Ik is an interval, then the explicit form of the Fourier transform can be found, and it indeed decays like |ω|-1.

But if you allow general measurable sets, i.e, f⁢(x)=∑ak⁢χEk for some disjoint measurable sets Ek, then again there's a problem because the decay of ∥τ1/ω⁢χEk-χEk∥1 can be slower than 1/|ω|; I think it can actually be arbitrarily slow.
