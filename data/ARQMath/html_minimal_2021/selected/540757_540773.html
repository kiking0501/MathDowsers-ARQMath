The set of interior points.

int⁡E is the set of all interior points in E⊆ℝ. Recall that a point p is an interior point of E if Bϵ⁢p⊂E. (where Bϵ⁢p denotes the open ball with radius ϵ, centered at p.) Also, note that an open set E is where all the points are interior.
