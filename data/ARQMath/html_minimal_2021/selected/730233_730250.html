Well, the 'digest' would be that there is no name for the processes, but if you want to describe them you can with the up arrows. Is also fairly easy to see that 2↑n2=2↑n-12=…=4 for any n. (Here, ↑n means n consequtive up arrows).
There is a name for this operation sequence. The sequence is called the [hyperoperation sequence](http://en.wikipedia.org/wiki/Hyperoperation), and it can be represented in a [variety of forms](http://en.wikipedia.org/wiki/Hyperoperation#Notations).
