Number of subgroups of order p in Cp×Cp

I'm looking for the number of subgroups of order p in Cp×Cp, using the double counting technique. I've tried doing it with Lagrange's theorem but don't make any progress. How should I do it? And how many normal subgroups are there of order p?

abstract-algebra

Your group G is abelian, so all subgroups are normal.

All elements (except the identity e) have order p. So G contains p2-1 elements of order p.

Thus a subgroup of order p contains e, and p-1 elements of order p.

By Lagrange's Theorem, two distinct subgroup of order p will have trivial intersection.

Therefore, if k is the number of subgroups of order p, you will have k⁢(p-1)=p2-1.
