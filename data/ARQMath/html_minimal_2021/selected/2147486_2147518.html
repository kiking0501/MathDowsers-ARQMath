Explicit components of yx=xy

When graphing the implicit equation yx=xy it seems like it consists of two possibly explicit curves. On the one hand there is the linear solution y=x. But as it appears there is a second curve monotonously decreasing as x increases, intersecting with the curve y=x at coordinate point (2,2). Is there any way to find out the explicit form of this other curve?
