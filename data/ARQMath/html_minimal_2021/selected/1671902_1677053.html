I assume you actually want to know what the determinant is. For a 2 by 2 we have, (1) |a0a1a0x|=a0⋅(x-a1) For a 3 by 3, we have, |a0a1a2a0xa2a0a1x| Since the upper right 2 by 2 is already known, we can evaluate along the bottom row. If we also note that the expansion along the bottom a1 is zero, we get, (2) |a0a1a2a0xa2a0a1x|=a0⋅(x-a1)⋅(x-a2)

In fact, we now know that the determinant Dn for a n by n matrix of this form obeys,

(3) Dn+1=Dn⋅(x-an-1)

Which implies using D1=a0, that

(4) Dn=a0⋅∏k=1n-1(x-ak)

I'm more physically oriented, so I'd just take this result and calculate the corresponding path integral. However, if you wish, I leave it as an exercise to rigorously prove this using induction.
