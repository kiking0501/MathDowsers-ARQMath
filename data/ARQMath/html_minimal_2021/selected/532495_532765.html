how the heck I'm supposed to check out if (2⁢π)n/2⁢δ⁢(x)⁢(1+|x|2)s/2 is in L2?

By duality. This was executive summary; details are below.

You have a distribution T, and want to know if there is an L2 function g such that T⁢(ϕ)=∫ℝnϕ⁢g for all test functions ℝ∖ℚ¯=ℝ. If (1) holds, then |T⁢(ϕ)|≤∥ϕ∥L2⁢∥g∥L2 by Cauchy-Schwarz. Put simply, T is bounded in the L2 norm: |T⁢(ϕ)|≤C⁢∥ϕ∥L2 with C independent of ℝ∖ℚ¯=ℝ. (And conversely, if (2) holds, then T extends to a bounded functional on L2, and Riesz representation delivers g as in (1).)

For your particular T (which, by the way, is simply (2⁢π)n/2⁢δ) you have T⁢(ϕ)=(2⁢π)n/2 whenever ϕ⁢(0)=1. Since a test function with ϕ⁢(0)=1 can have arbitrarily small L2 norm, (2) fails.
