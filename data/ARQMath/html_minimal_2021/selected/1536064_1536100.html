It should be clear that if n≥1 and m<n-1, then there are no such functions. Likewise, if n=0 and m>0, then there are no such functions. In the case m=n=0, there is exactly one function X→Y, and it is necessarily onto.

Otherwise, m≥n-1≥0. Observe that there are n subsets Z of Y such that |Z|=n-1, and that an almost-onto function X→Y is either a map onto Y or a map onto one of these sets Z.

As you can see in the answers to this question, for any integer 0≤k≤m, there are ∑j=0k(-1)j⁢(kj)⁢(k-j)m functions from a given m-element set onto a given k-element set. Hence, the number of almost-onto functions X→Y is ∑j=0n(-1)j⁢(nj)⁢(n-j)m+n⁢∑j=0n-1(-1)j⁢(n-1j)⁢(n-1-j)m.

In fact, this formula works for every case except the n≥1 and m<n-1 case.
