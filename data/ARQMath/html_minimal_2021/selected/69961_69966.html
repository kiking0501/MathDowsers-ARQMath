By inspection, you can see how to represent 0, 3, 5, and 8. Now, given any integer n≥9, classify n into one of the following cases.

Case n≡0 (mod 3): In this case, n=3⁢k for some k, so there is nothing to show (it is already a certain multiple of 3).

Case n≡1 (mod 3): In this case, n=3⁢k+1 for some k≥3 (since n≥9). We can write n=3⁢(k-3)+5⋅2.

Case n≡2 (mod 3): In this case, n=3⁢k+2, for some k≥3. We can write n=3⁢(k-1)+5.

So, not only do all those numbers belong to the set, but there is an easy way to figure out how they are represented.
