Theres is metric space (X,d) and a set A⊆X, hence for any x,y∈X and a∈A

d⁢(x,a)≤d⁢(x,y)+d⁢(y,a)⁢∀a∈A .... this is true since d() is a valid metric.

Let x,y be fixed members of the metric space and suppose A is a subset of the metric space. Let D={d⁢(x,a)∈ℝ:a∈A} and let E={d⁢(x,y)+d⁢(y,a)∈ℝ:a∈A}.

Let arbitrary e∈E be given. Then e=d⁢(x,y)+d⁢(y,a) for some a∈A. Thus, there is some d∈D (namely, d⁢(x,a)) such that d⁢(x,a)≤e. By definition of inf, inf⁡D≤d and therefore inf⁡D≤e. Since e was arbitrary, this is true ∀e∈E. This shows that inf⁡D is a lower bound for E. Again, by definition of inf, this must mean that inf⁡D≤inf⁡E, as required. Yes, your assertion is ok.

@GautamPai: The proposition (as you state it) is *not* true. For example, consider A=[0,1] and B={-1,1}. Clearly, for all a∈A, there exists a b∈B such that a≤b. However, inf⁡B<inf⁡A. On the other hand, the result mentioned by Shai isn't quite correct, either (consider A empty and B any non-empty set of reals). However, if A,B⊆ℝ with A≠∅ such that for all a∈A,b∈B we have a≤b, then it does follow that inf⁡A≤inf⁡B.
Or, if A,B⊆ℝ such that for all b∈B there is some a∈A such that a≤b, then inf⁡A≤inf⁡B.
