This problem is solvable with the next step conditioning method. Let μk denote the mean number of tosses until 5 consecutive heads occurs, given that k consecutive heads just occured. Obviously μ5=0. Conditioning on the outcome of the next coin throw: μk=1+12⁢μk+1+12⁢μ0 Solving the resulting linear system:

In[28]:= Solve[Table[mu[k] == 1 + 1/2 mu[k + 1] + mu[0]/2, {k, 0, 4}],    Table[mu[k], {k, 0, 4}]] /. mu[5] -> 0  Out[28]= {{mu[0] -> 62, mu[1] -> 60, mu[2] -> 56, mu[3] -> 48,    mu[4] -> 32}}

Hence the expected number of coin flips μ0 equals 62.
