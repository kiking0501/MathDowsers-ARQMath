@coffeemath See my comment to Michael Hardy's comment for a proof (I think).
As we can we read in the text [A Representation of Rational Numbers](http://www.mathteacherctk.com/blog/2012/04/a-representation-of-rational-numbers/), *the identity aa=r, with a rational r is possible in two cases only: when a is an integer or irrational. Put differently, every rational number in the interval (e-1/e,∞), except for those in the form nn, with integer n, are in the form aa with a irrational.*

If the problem is finding a real number x such that xx=n, for a given n∈ℕ, then notice that the function x↦xx is continuous and goes from 1 down to a positive number less than 1 and then up to ∞ as x goes from 0 to ∞, so the intermediate value theorem tells us a solution exists.

Finding the solution probably has to be done numerically rather than algebraically.
