How many ways to distribute 19 identical objects to 4 distinct people so everyone gets at most 5?

I know to distribute r identical objects to n groups it's n^r... So I have 4^19

But I don't know how to deal with the fact that everyone gets at most 5.

combinatorics

Assuming that all 19 objects are distributed, let each one get n1,n2,n3,n4, we have ∑ni=19, with each ni≤5

Consider variables yi=5-ni, then the constraints become ∑yi=1.

So given that yi can only take values 0 and 1, we will have only \color⁢b⁢l⁢u⁢e⁢4 ways.
