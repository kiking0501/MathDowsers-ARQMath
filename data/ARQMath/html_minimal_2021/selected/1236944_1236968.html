Your statement needs to be more precise.

The implication ⇒ is obvious, if n=p2 then it follows immediately from the definition or formula pf ϕ⁢(n) that n-ϕ⁢(n)=p.

For the converse, you need to be more precise. The way in which is stated, is correct but trivial: you are saying n=p2 if and only if n=p2 and n=(n-ϕ⁢(n))2.

If you mean n=p2⇔n-ϕ⁢(n) is prime, this is not true. n=15 fails the converse.

Added

If p=n-ϕ⁢(n)=n it follows that n=p2.

Now, it is easy to see that for all integers m we have ϕ⁢(m2)=m⁢ϕ⁢(m). Therefore

p=p2-p⁢ϕ⁢(p)⇒ϕ⁢(p)=p-1

This implies p∈P.
