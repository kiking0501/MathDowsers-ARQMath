Show unique fixed point using continuity

Let f:[a,b]→[a,b] be a real-valued function such that,

|f⁢(x)-f⁢(y)|<|x-y|,∀x,y∈[a,b],x≠y.

I need to show that f has a fixed point.

My attempt:

I defined g⁢(x)=|f⁢(x)-x| over [a,b] showed that it is continuous, how can I use it to show that f has unique fixed point. Any hints will be appreciated...

If you let y tend to x, that is you take y=x+h, and take lim (h→0) in the given inequality, then you can show that the function f⁢(x) is continuous. This fact is enough to show the existence of a fixed point.

Define the function g⁢(x)=f⁢(x)-x, g:[a,b]→[a,b].

Check that f⁢(a)≥a,f⁢(b)≤b according to the codomain of the function f.

If f⁢(a)=a, then we are done because a is a fixed point, else assume f⁢(a)>a.

Now see that f⁢(a)>a,f⁢(b)<b⇒g⁢(a)>0,g⁢(b)<0, and g⁢(x) being the difference of two continuous functions over [a,b], f⁢(x) and x, g is itself is continuous. So g must follow the Intermediate Value Property, so that g⁢(a)>0,g⁢(b)<0 ensures the existence of some c∈[a,b] such that g⁢(c)=0⇒f⁢(c)=c, so that c is our fixed point.

If there are more than two fixed points, let them be c,d∈[a,b], and c≠d.

According to the assumption, f⁢(c)-c=f⁢(d)-d=0⇒f⁢(c)-f⁢(d)=c-d⇒∣f⁢(c)-f⁢(d)∣=∣c-d∣ which violates the given condition.
