Since -π⁢x2+2⁢π⁢i⁢x⁢ξ=-π⁢(x-i⁢ξ)2-π⁢ξ2, we write

∫-∞∞e-π⁢x2⁢e2⁢π⁢i⁢x⁢ξ⁢𝑑x=∫-∞∞e-π⁢(x-i⁢ξ)2⁢e-π⁢ξ2⁢𝑑x=e-π⁢ξ2⁢∫-∞∞e-π⁢(x-i⁢ξ)2⁢𝑑x

Let ξ=a+b⁢i. Using the u-substition u=x+b, we obtain

∫-∞∞e-π⁢(x-i⁢ξ)2⁢𝑑x=∫-∞∞e-π⁢(u-a⁢i)2⁢𝑑u

The last integral can be computed using the rectangular contour Γ⁢(R) with vertices -R,R,R-a⁢i, and -R-a⁢i. Let f⁢(z)=e-π⁢z2. By Cauchy's theorem, ∫Γ⁢(R)f⁢(z)⁢𝑑z=0. The integrals along the vertical edges of Γ⁢(R) are bounded by a constant times e-π⁢R2, which tend to 0 as R→∞. Hence,

∫-∞∞e-π⁢(u-a⁢i)2⁢𝑑u=∫-∞∞e-π⁢u2⁢𝑑u=1

and the value of the original integral is e-π⁢ξ2.
