If y=4⁢arcsin⁡x3 then x3=sin⁡y4

Differentiate this with respect to x using the chain rule to obtain 3⁢x2=y′4⁢cos⁡y4

Now cos⁡y4=1-sin2⁡y4=1-x6 so that y′=12⁢x21-x6

This implicit differentiation is a method you can use if you forget the formula and don't have tables.
