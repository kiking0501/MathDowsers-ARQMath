**Thoughts:** First, find f~:[-π/2,π/2]→ℝ such that f~⁢(f~⁢(x))=sin⁡(x). Intuitively this should be possible, since on this interval sin⁡(x) is a strictly increasing function. From there, it suffices to define f⁢(x)=f~⁢([x]π) where [x]π denotes the unique value in [-π/2,π/2] such that x=2⁢π⁢k+[x]π **or** x=2⁢π⁢k+π-[x]π for some k∈ℤ.
