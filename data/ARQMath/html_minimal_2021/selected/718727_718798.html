We can define exponentiation to every real number as follows:

Let α∈ℝ and x∈ℝ+ be a non negative real number. We define xα as xα:=exp⁢(α⁢ln⁡(x)) with this definition one can prove all the 'exponent laws', for example xα⁢xβ=xα+β

Another approach is to start with exponentiation of natural numbers, defined recursively as follows:

For n∈ℕ, and x∈ℝ+; xn=xn-1⋅x x1=x Then we can extend this definition, allowing the exponent to be an integer number putting x-1=1x

To allow rational exponents, we define the n-th root as the (unique positive) solution to yn=x and we denote it as x1n=y

The final step needs the supreme axiom;

Let α∈ℝ and x∈ℝ+, xα:=sup⁢{xr∣r∈ℚ⁢ and ⁢r≤α}

One can prove that the definition given at the beginning of this post and this last one are equivalent.
