In general, suppose n is an integer that is not a perfect square. I will show that n is irrational. Note: This proof is not original, although I did come up with it independently.

Let m=⌊n⌋. Since n is not a perfect square (this is where we use that assumption), m<n.

Suppose n is rational, so n=ab for some positive integers a and b.

At this point, we can go in two different directions. We can either assume that b is the smallest positive integer such that b⁢n is an integer, and deduce a contradiction by finding a smaller positive integer d such that d⁢n is an integer, or we can show that there is a smaller positive integer d such that d⁢n is an integer and thus create a contradiction using infinite descent.

I will use the first method.

Suppose b is the smallest positive integer such that b⁢n is an integer. Let a=b⁢n As above, let m=⌊n⌋, so that m<n<m+1.

so that (a-m⁢b)⁢n=n⁢b-m⁢a is also an integer.

Since m<n<m+1, m<a/b<m+1 or m⁢b<a<m⁢b+b, or 0<a-m⁢b<b.

Therefore a-m⁢b is a positive integer smaller than b and (a-m⁢b)⁢n is an integer. This contradicts the definition of b as the smallest such integer.

You can put n=17 or n=2 (which is where I first saw this proof and generalized it to what you see here), or any non-perfect square and show that n is irrational this way.
