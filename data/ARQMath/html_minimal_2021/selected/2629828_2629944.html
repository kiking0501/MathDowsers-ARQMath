Let f:[0,π]→ℝ is continuous and f⁢(0)=0, then which of the following are true?

  1. If ∫0πf⁢(t)⁢cos⁡n⁢t⁢d⁢t=0 for n∈{0}∪ℕ, then f≡0
  2. If ∫0πf⁢(t)⁢sin⁡n⁢t⁢d⁢t=0 for n∈ℕ, then f≡0
  3. If ∫0πtn⁢f⁢(t)⁢𝑑t=0 for {0}∪ℕ, then f≡0
real-analysis integration
By Weierstrass there is a sequence of polynomials pn converging uniformly to f on [0,π]. Therefore, if we assume (3), 0=∫0πpn⁢(x)⁢f⁢(x)⁢𝑑x→∫0πf2⁢(x)⁢𝑑x. Therefore, f is zero.
