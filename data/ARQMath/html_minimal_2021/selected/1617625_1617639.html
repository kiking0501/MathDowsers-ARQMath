@Workaholic This is a u-substitution question according to author, if I substitute $ u=x^2+1 $, $ x = \sqrt{u-1} $ and $ \frac{du}{dx}=2x $ in $ \int \frac{1}{(x^2+1)^{3/2}} \frac{2x}{2x} dx $ I get $$ \int \frac{1}{u^{3/2}} \frac1{2x} du = \frac1{2} \int \frac{1}{u^{3/2}\sqrt{u-1}} du = \frac1{2} \int \frac{1}{\sqrt{u^3(u-1)}} du $$ Using $ s=u^4-u $ and $ \frac{ds}{du}=4u^3-1 $ doesn't seem to get anywhere, so I've been trying something else.
@CloudIcarus There is an even better substitution you can make. See below.

Substitute x=tan⁡u⟹d⁢x=sec2⁡u⁢d⁢u.

∴∫1(x2+1)3/2⁢𝑑x=∫sec2⁡u(tan2⁡u+1)3/2⁢𝑑u=∫sec2⁡usec3⁡u⁢𝑑u=∫cos⁡u⁢d⁢u=sin⁡u+C Now, note that u=tan-1⁡x. Therefore, ∫1(x2+1)3/2⁢𝑑x=sin⁡(tan-1⁡x)+C

As @martycohen mentioned below, one can simplify to sin⁡(tan-1⁡x)+C=x1+x2+C

Note that sin⁡(arctan⁡(x))=x/1+x2.
