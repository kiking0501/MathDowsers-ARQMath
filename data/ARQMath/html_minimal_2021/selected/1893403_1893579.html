Upper bounding using negative exponentials

We know that for a fixed x>0, the convergence of (1+xn)n→ex is from below and so we have the following bound for any n∈ℕ:

(1+xn)n≤ex.

However, for a fixed y<0, y=-x the convergence of (1+yn)n=(1-xn)n→e-x is from above and so we certainly can't write (1-xn)n≤e-x.

@Yuriy S I was mistaken.

There is a misstatement in the OP. The sequence en=(1-xn)n is actually an increasing sequence. To see this, we write for x<n

en+1en

where in going from (1) to (2) we exploited Bernoulli's Inequality.

Therefore, en increases monotonically for x<n to e-x and

e-x≥(1-xn)n
