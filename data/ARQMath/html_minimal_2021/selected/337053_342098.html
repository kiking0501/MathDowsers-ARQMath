This is what I thought about, it should be some comment, but this exceeds the character limit for comment.

If you take Wilson Theorem that

(p-1)!≡-1modp, then

(p-2)!≡1modp.

If we take inverse of a in Fp as 1a, we know that 1a+1b=a+ba⁢b in Fp, it can taken as usual addition.

Then

(p-3)!≡-12modp

(p-4)!≡-12⋅(-13)modp

⋯

1!≡(-1)p-3⁢1(p-2)!modp

Thus

1!+2!+⋯⁢(p-3)!≡-∑k=0p-2(-1)k⁢1k!modp

And we state without proof.

Lemma:

Consider the fraction pq with q=m!, then the closest number to exp⁡(-1) with form pq should satisfy

pq=∑j=0m(-1)j⁢1j!

Then

-∑k=0p-2(-1)k⁢1k!≡-floor⁢[(p-2)!exp⁡(1)]modp

However, this cannot be checked for large p, since the constant exp⁡(1) is not accurate enough. I simply checked for smaller primes.

However, I felt that there should be other solutions than 3 and 11, otherwise, this can be a very interesting property for the constant.

Unfortunately, this doesn't really provide much help because it's very hard to translate back from the 'analytical' description to anything useful about the residues; even as simple a question as 'is 1nmodp less than or greater than p2?' doesn't really (AFAIK) have a clear large-scale 'analytical' structure to it.
