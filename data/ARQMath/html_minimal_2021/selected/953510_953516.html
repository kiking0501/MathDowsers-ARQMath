Let x=1/h, so that now you're interested in limx→∞⁡e-x1/x=limx→∞⁡xex=limx→∞⁡1ex=0, using l'Hôpital's in the second equality.

You can prove that en≥n2 for large enough integers n by induction, and this implies that ex+1≥x2 for large enough x and so x/ex≤1/(e⁢x) for large enough x. If you allow yourself to use the Taylor expansion of ex, then we get ex>x2/2 directly for all x≥0.
The transition from integers to real numbers looks like it might take a bit of work. I would prefer to prove that ex/2≥x for x≥2 by showing that f⁢(x)=ex/2-x is increasing on that interval. Squaring both sides, you get ex≥x2, so x/ex≤1/x.
