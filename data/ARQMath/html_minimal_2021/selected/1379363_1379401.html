Another way to determine continuity is to check for the following condition: $$ \forall A\in\mathscr{P}(X)\,\,\,,\,\,\,f(\text{Cl}(A))\subseteq \text{Cl}f(A) $$ That is to say, a function is continuous if and only if it does not seperate points from sets.

Consider f:A→B. Changes in the domain "control" what happens to the image, the value of x∈A is "causal" for the value f⁢(x)∈B. Therefore you may think that going from domain to image is the natural way to define continuity. However, continuity is about how well we can predict the image f⁢(x) if we slightly modify x; that is, if only we can keep x "under control" well enough (i.e., keep changes, influences, error, or whatever sufficiently small) then we also keep f⁢(x) under control (i.e., keep its change or error as mall as necessary/desired). In this formulation it still sounds like something going from A to B. But if we formulate that "desire" (for a metric space) as |f⁢(x)-f⁢(x0)|<ε and the control we impose on the domain side as |x0-x|<δ, then we arrive exactly at the epsilon-delta-definition: For any ϵ>0 (as desired) there exists (i.e., we can control) δ>0 such that |x-x0|<δ implies |f⁢(x)-f⁢(x0)|<ε. This is where the change of sides occurs: ε is picked first and then depending on it we pick δ.

So can we formulate this with images of open sets? No, because while the definition implies something about the image of an open ball, namely f⁢(Bδ⁢(x0))⊆Bε⁢(f⁢(x0)) (with B denoting open ball), the problem is that ϵ is given before we pick our δ. That is, we can not cast this into "For every open ball Bδ⁢(x0), the image f⁢(Bδ⁢(x0)) has some property". Rewriting the inclusion as Bδ⁢(x0)⊆f-1⁢(Bε⁢(x0)) we obtain the right formulation, i.e., the pre-image of an open ball (around a point f⁢(x0) in the image) contains an open ball (around that point x0); and in the generalization: The pre-image of an open set is open.
