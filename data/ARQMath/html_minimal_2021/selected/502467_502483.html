Determinants of Block Matricies

I read on wikipedia that

D⁢e⁢t⁢(ABBA)

is equal to D⁢e⁢t⁢(A+B)⁢D⁢e⁢t⁢(A-B) if A and B commute. Does this hold true even if A and B are not invertible? I spent a short amount of time trying to come up with a counter example, but it worked for the noninvertible matricies I picked. I'm curious that if this is in fact true, if anyone has a proof of it?

Yes, it works for any A and B. See Denis Serre's posting on http://mathoverflow.net/questions/48936/iterated-calculation-of-determinants for something vastly more generla.

We need no assumptions about A,B except that they have same dimension. Then apply basic operation of subtracting a set of rows from set of the other rows and also summing a set of columns with set of other columns, which does not change the determinant. You will simply get:

det⁡(ABBA)=det⁡(ABB-AA-B)=det⁡(A+BB0A-B)=det⁡(A-B)⁢det⁡(A+B)
