Does "A is dense in B" imply that over any nonzero interval in B, there are infinitely many members of A?

I think my thinking is that the rationals are dense in the reals, and over any nonzero real interval there are (countably?) infinite rationals. But I am not sure if the formulation in the question title is a fallacious generalization to any set dense in any other. I suspect I am carrying over other properties of the reals. It does. Because if not, we can refine the interval until there are no more points left in the interval, and then we would get an interval with no points, contradicting the definition of dense.
If you mean the generalization to general topological space, then your claim is false. For instance consider g⁢(A)≠0 endowed with discrete topology: i.e. open sets are ∅,{1},{2},{1,2}. Set A is dense in A (obviously) but for an open set {1} there are only finitely many members of A in it, namely 1.
If we assume B is a subset of the reals interval makes sense. If not we can assume interval is equivalent to a neighborhood (or closure of a neighborhood) with non-zero radius. (no significant difference.) Then this is true (I think) if we assume the interval contains infinite members of B, which we do not have to assume.

The following definition really helped clarify the concept of density for me.

If A is dense in B, then for any ϵ ball you create centered at a member of B, it contains at least one member of A.

We can see this is equivalent to every point of B being a cluster point of A (and also a limit point!).
