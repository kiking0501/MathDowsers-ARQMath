convex functions and geometic mean

every convex function f preserves arithmetic mean: f⁢(x+y2)≤f⁢(x)+f⁢(y)2, this is a particular case of the convexity condition.

Does convex function f preserve geometic mean: f⁢(x⁢y)≤f⁢(x)⁢f⁢(y) ? (of course here f:(0,∞)→(0,∞))

the answer is no, but there is no increasing function among counterexampes I know.

so the question is: does every increasing convex function f:(0,∞)→(0,∞) preserve geometic mean?

real-analysis

Counterexample: Take any convex increasing function f:(0,1)→(0,1) that satisfies:

\begin{align} &f(1) = 0.1\\ &f(2) = 1 \\ &f(4) = 3 \end{align}

Then f⁢(1⁢4)>f⁢(1)⁢f⁢(4) because 1>0.3. You can get such a function by assuming f⁢(0)=0 and filling in peicewise linearly. Note that the slopes over the intervals (0,1], [1,2], [2,∞) are 0.1,0.9,1, respectively, and so the function f⁢(x) is indeed convex increasing.
