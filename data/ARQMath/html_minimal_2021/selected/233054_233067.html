Intermediate value property problem and continuous function

How to prove that a strictly increasing function f:[a,b]→ℝ which has the intermediate value property is continuous on [a,b].

real-analysis

Hint: Since f is increasing, for every c∈[a,b], f⁢(c±):=limx→c±⁡f⁢(x) exists, and it suffices to show that f⁢(c)=f⁢(c±), where for c=a(resp. b), only consider c+(resp. c-). If it is not the case, you may use the intermediate value property to derive a contradiction.
