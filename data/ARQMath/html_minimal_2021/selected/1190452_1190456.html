symmetric means if x is related to y, then y is related to x. If (x,y) is in there, then (y,x) is in there. Well, (2,4) is in there but (4,2) is not, so it is not symmetric.

Symmetry requires that if ⟨a,b⟩∈R, then ⟨b,a⟩∈R as well. In this case there are two violations of symmetry: ⟨2,4⟩ is in R while ⟨4,2⟩ is not, and ⟨3,4⟩ is in R while ⟨4,3⟩ is not. These two violations are unrelated. It’s not that ⟨2,4⟩ doesn’t ‘have symmetry with’ ⟨3,4⟩: R would still not be symmetric even if we removed ⟨3,4⟩ from it, because it would still contain ⟨2,4⟩ but not ⟨4,2⟩.

To show that a relation does not have one of these properties you need only find one failure of the property. Here, for instance, the moment you find that ⟨2,4⟩∈R and ⟨4,2⟩∉R, you’ve shown that R is not symmetric. You also know that R is not reflexive: the ordered pair ⟨3,4⟩ shows that 4 is in the field of the relation, but the pair ⟨4,4⟩ is not in R.

@jc707270: No, it’s not antisymmetric, because it contains both ⟨3,2⟩ and ⟨2,3⟩. An antisymmetric relation is one that never has both ⟨a,b⟩ and ⟨b,a⟩ unless a and b are actually the same element.
