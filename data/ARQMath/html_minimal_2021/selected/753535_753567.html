How to expand 𝐫T⁢𝐀-1⁢𝐫=(𝐲-𝐀𝐱)T⁢𝐀-1⁢(𝐲-𝐀𝐱)

I need to expand:

𝐫T⁢𝐀-1⁢𝐫=(𝐲-𝐀𝐱)T⁢𝐀-1⁢(𝐲-𝐀𝐱)

I believe that 𝐀𝐁≠𝐁𝐀, 𝐀𝐀-1=1, and that (𝐲-𝐀𝐱)T=(𝐲T-𝐀T⁢𝐱T)

but I am not really sure how to go about the expansion, in particular I worry about the fact that 𝐀𝐁≠𝐁𝐀. Thanks.

First note that (as (𝐀𝐁)T=𝐁T⁢𝐀T) we have (𝐲-𝐀𝐱)T=(𝐲T-𝐱T⁢𝐀T)

Thus, to expand 𝐫T⁢𝐀-1⁢𝐫=(𝐲-𝐀𝐱)T⁢𝐀-1⁢(𝐲-𝐀𝐱), the following operations need to be carried out:- (𝐲-𝐀𝐱)T⁢𝐀-1⁢(𝐲-𝐀𝐱)=(𝐲T-𝐱T⁢𝐀T)⁢𝐀-1⁢(𝐲-𝐀𝐱)=𝐲T⁢𝐀-1⁢𝐲-𝐲T⁢𝐀-1⁢𝐀𝐱-𝐱T⁢𝐀T⁢𝐀-1⁢𝐲+𝐱T⁢𝐀T⁢𝐀-1⁢𝐀𝐱=𝐲T⁢𝐀-1⁢𝐲-𝐲T⁢𝐱-𝐱T⁢𝐀T⁢𝐀-1⁢𝐲+𝐱T⁢𝐀T⁢𝐱=𝐲T⁢𝐀-1⁢𝐲-𝐱T⁢𝐲-𝐱T⁢𝐀T⁢𝐀-1⁢𝐲+𝐱T⁢𝐀T⁢𝐱=𝐲T⁢𝐀-1⁢𝐲-𝐱T⁢(𝐈+𝐀T⁢𝐀-1)⁢𝐲+𝐱T⁢𝐀T⁢𝐱 where we assume that 𝐲T⁢𝐱 is a scalar, so that it will equal 𝐱T⁢𝐲.
