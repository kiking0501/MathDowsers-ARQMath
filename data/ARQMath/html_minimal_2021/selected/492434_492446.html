How to get all solutions to equations with square roots

I would like to find all solutions to

b-a⁢1+a2+b2=a2⁢(a⁢b-1+a2+b2) a-b⁢1+a2+b2=b2⁢(a⁢b-1+a2+b2)

I found some solutions. For example, a=1,b=±i and b=1,a=±i. How can I find all solutions?

algebra-precalculus

One technique with this kind of problem is to isolate the square root, and then square the equation (which brings in possible extra roots from the negative value of the square root).

[Now corrected for error pointed out by Anush in copying over the equations]

Beginning that process here we find that the first equation becomes: b⁢(1-a3)=(a-a2)⁢1+a2+b2 and notice that the factor a-1 will cancel, provided that a≠1 so thatb⁢(1+a+a2)=a⁢1+a2+b2

Similarly, from the second equation, if b≠1 we have a⁢(1+b+b2)=b⁢1+a2+b2

Now we can eliminate the square root by subrtacting a× the second equation from the b× first to obtain (b2-a2)+a⁢b⁢(b-a)=(b-a)⁢(a+b+a⁢b)=0

So that either a⁢b+a+b=0 or a=b

This is substantial progress.

If a=b we note the solution a=b=0, and assume that a,b≠0,1. The two equations are the same and we need to solve (1+a+a2)=1+2⁢a2

If we square this we obtain 1+2⁢a+3⁢a2+2⁢a3+a4=1+2⁢a2 which becomes 2⁢a+a2+2⁢a3+a4=0 or a⁢(2+a)⁢(a2+1)=0

a=0 we know about. a=-2,b=-2 is another solution. a=b=±i works in the original equation for a=b=i but not for a=b=-i.

If now a+b+a⁢b=0, the term b⁢(1+a+a2)=b+a⁢b+a2⁢b=-a+a2⁢b=a⁢(a⁢b-1) so we have

a⁢(a⁢b-1)=a⁢a2+b2+1 and if a≠0 we can cancel and square to obtain (a⁢b-1)2=a2+b2+1 Now we can write a⁢b=-(a+b) so that the equation becomes a2+b2+1+(2⁢a⁢b+2⁢a+2⁢b)=a2+b2+1 The term in brackets is zero, so the equation is an identity and gives no additional constraint.

Note that the condition a⁢b+a+b=0 is equivalent to (a+1)⁢(b+1)=1, and if b≠-1 this is also a=-bb+1

Some checking is required to identify which solutions get the right sign for the square root - assuming the solutions are real numbers (and complex solutions have been canvassed). We know we have a2+b2+1=(a⁢b-1)2 so that a2+b2+1=|a⁢b-1|, so for a solution we need a⁢b⁢\gt⁢1. This in turn means -b2b+1⁢\gt⁢1.

If b+1⁢\gt⁢0 this reduces to -b2⁢\gt⁢b+1⁢\gt⁢0 which is impossible. So b+1⁢\lt⁢0 (ignoring b=-1) and -b2⁢\lt⁢b+1 or b2+b+1⁢\gt⁢0 which is equivalent to (2⁢b+1)2+3⁢\gt⁢0, which is always true.

So we have the condition b⁢\lt-1 and from the form (a+1)⁢(b+1)=1 we see that this also implies a⁢\lt-1
