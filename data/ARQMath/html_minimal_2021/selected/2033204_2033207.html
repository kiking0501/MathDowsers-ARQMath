Simple integration problem from real analysis

If f is continuous and nonnegative on the interval [a,b] and ∫abf⁢(x)⁢𝑑x=0 show that f is identically equal to zero there.

This is intuitively obvious but I think we're supposed to prove it using the monotone property.

Monotone Property: Let f and g be continuous on [a,b]. Then, if f⁢(x)≤g⁢(x)⁢∀a≤x≤b, ∫abf⁢(x)⁢𝑑x≤∫abg⁢(x)⁢𝑑x

Any hints would be much appreciated.

real-analysis integration
Indeed, given a continuous nonnegative function f which is not identically zero, try to find a function g with g≤f such that g can be exactly integrated and ∫abg⁢(x)⁢𝑑x>0. (You can make g continuous but it might be easier to make it piecewise constant instead.) Then applying the monotone property gives the desired result through contraposition.

You must assume a<b: the claim is false if a=b.

If it's not identically 0, then there is some ϵ>0 and interval (c,d)⊆(a,b) on which f⁢(x)>ϵ>0. Then ∫abf⁢(x)⁢𝑑x≥(d-c)⁢ϵ.
