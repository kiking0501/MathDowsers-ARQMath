Let R be the event it rains, A and B be the events that A and B predict it rains. We want P⁢(R|A∩Bc), which you can use Baye's rule to find: P(R|A∩Bc)=P⁢(A∩Bc|R)⋅P⁢(R)P⁢(A∩Bc)              =P⁢(A∩Bc|R)⋅P⁢(R)P⁢(A∩Bc|R)⋅P⁢(R)+P⁢(A∩Bc|Rc)⋅P⁢(Rc) You can now find each of the terms in the last expression.
