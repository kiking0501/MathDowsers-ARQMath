There's many way to view it but basically x⁢(1-x)≤14⇔-x2+x-14≤0. -x2+x-14 is a parabola that "opens downward" and has a maximum.

Let x=12+e and then 1-x=12-e. Then x⁢(1-x)=(12+e)⁢(12-e)=14-e2. As e2≥0 it is true that x⁢(1-x)≤14 but it is not true that x⁢(1-x)<14.

If x=12 then x⁢(1-x)=14. But if x≠12 and e≠0 then x⁢(1-x)=14-e2<14.
