\begin{equation*} \displaystyle \sum\limits_{n=0}^{\infty} 0.7^n=\frac{1}{1-0.7} = 10/3 ? \end{equation*}

∑n=0∞xn=11-x ?

What about if we take only a finite number of terms? Is there a simpler formula?

∑n=0Nxn

By definition, a "series" (an "infinite sum") ∑n=k∞an is defined to be a limit, namely ∑n=k∞an=limN→∞⁡∑n=kNan. That is, the "infinite sum" is the limit of the "partial sums", if this limit exists. If the limit exists, equal to some number S, we say the series "converges" to the limit, and we write ∑n=k∞an=S. If the limit does not exist, we say the series diverges and is not equal to any number.

So writing that ∑n=0∞0.7n=11-0.7 means that we are asserting that limN→∞⁡∑n=0N0.7n=11-0.7.

So what your question is really asking is: why is this limit equal to 11-0.7? (Or rather, that is the only way to make sense of the question).

In order to figure out the limit, it is useful (but not strictly necessary) to have a formula for the partial sums, sN=∑n=0N0.7n. This is where the formulas others have given come in. If you take the Nth partial sum and multiply by 0.7, you get sN=1+(0.7)+(0.7)2+⋯+(0.7)N(0.7)⁢sN=(0.7)+(0.7)2+⋯+(0.7)N+(0.7)N+1 so that (1-0.7)⁢sN=sN-(0.7)⁢sN=1-(0.7)N+1. Solving for sN gives sN=1-(0.7)N+11-0.7. What is the limit as N→∞? The only part of the expression that depends on N is (0.7)N+1. Since |0.7|⁢\lt⁢1, then limN→∞⁡(0.7)N+1=0. So, limN→∞⁡sN=limN→∞⁡(1-(0.7)N+11-0.7)=limN→∞⁡1-limN→∞⁡(0.7)N+1limN→∞⁡1-limN→∞⁡0.7=1-01-0.7=11-0.7. Since the limit exists, then we write ∑n=0∞(0.7)n=11-0.7.

More generally, a sum of the form a+a⁢r+a⁢r2+a⁢r3+⋯+a⁢rk with a and r constant is said to be a "geometric series" with initial term a and common ratio r. If a=0, then the sum is equal to 0. If r=1, then the sum is equal to (k+1)⁢a. If r≠1, then we can proceed as above. Letting S=a+a⁢r+⋯+a⁢rk we have that S-r⁢S=(a+a⁢r+⋯+a⁢rk)-(a⁢r+a⁢r2+⋯+ak+1)=a-a⁢rk+1 so that (1-r)⁢S=a⁢(1-rk+1). Dividing through by 1-r (which is not zero since r≠1), we get S=a⁢(1-rk+1)1-r.

A series of the form ∑n=0∞a⁢rn with a and r constants is called an infinite geometric series. If r=1, then limN→∞⁡∑n=0Na⁢rn=limN→∞⁡∑n=0Na=limN→∞⁡(N+1)⁢a=∞, so the series diverges. If r≠1, then using the formula above we have: ∑n=0∞a⁢rn=limN→∞⁡∑n=0Na⁢rN=limN→∞⁡a⁢(1-rN+1)1-r. The limit exists if and only if limN→∞⁡rN+1 exists. Since |r|⁢\lt⁢1 it follows that: a=0

Your particular example has a=1 and r=0.7.

Since this recently came up (09/29/2011), let's provide a formal proof that |r|⁢\lt⁢1

If r⁢\gt⁢1, then write r=1+k, with k⁢\gt⁢0. By the binomial theorem, rn=(1+k)n⁢\gt⁢1+n⁢k, so it suffices to show that for every real number M there exists n∈ℕ such that n⁢k⁢\gt⁢M. This is equivalent to asking for a natural number n such that n⁢\gt⁢Mk, and this holds by the Archimedean property; hence if r⁢\gt⁢1, then limn→∞⁡rn does not exist. From this it follows that if r⁢\lt-1 then the limit also does not exist: given any M, there exists n such that r2⁢n⁢\gt⁢M and r2⁢n+1⁢\lt⁢M, so limn→∞⁡rn does not exist if r⁢\lt-1.

If r=-1, then for every real number L either |L-1|⁢\gt⁢12 or |L+1|⁢\gt⁢12. Thus, for every L and for every M there exists n⁢\gt⁢M such that |L-rn|⁢\gt⁢12 proving the limit cannot equal L; thus, the limit does not exist. If r=1, then rn=1 for all n, so for every ϵ⁢\gt⁢0 we can take N=1, and for all n≥N we have |rn-1|⁢\lt⁢ϵ, hence limN→∞⁡1n=1. Similarly, if r=0, then limn→∞⁡rn=0 by taking N=1 for any ϵ⁢\gt⁢0.

Next, assume that 0⁢\lt⁢r⁢\lt⁢1. Then the sequence {rn}n=1∞ is strictly decreasing and bounded below by 0: we have 0⁢\lt⁢r⁢\lt⁢1, so multiplying by r⁢\gt⁢0 we get 0⁢\lt⁢r2⁢\lt⁢r. Assuming 0⁢\lt⁢rk+1⁢\lt⁢rk, multiplying through by r we get 0⁢\lt⁢rk+2⁢\lt⁢rk+1, so by induction we have that 0⁢\lt⁢rn+1⁢\lt⁢rn for every n.

Since the sequence is bounded below, let ρ≥0 be the infimum of {rn}n=1∞. Then limn→∞⁡rn=ρ: indeed, let ϵ⁢\gt⁢0. By the definition of infimum, there exists N such that ρ≤rN⁢\lt⁢ρ+ϵ; hence for all n≥N, |ρ-rn|=rn-ρ≤rN-ρ⁢\lt⁢ϵ. Hence limn→∞⁡rn=ρ.

In particular, limn→∞⁡r2⁢n=ρ, since {r2⁢n}n=1∞ is a subsequence of the converging sequence {rn}n=1∞. On the other hand, I claim that limn→∞⁡r2⁢n=ρ2: indeed, let ϵ⁢\gt⁢0. Then there exists N such that for all n≥N, rn-ρ⁢\lt⁢ϵ. Moreover, we can assume that ϵ is small enough so that ρ+ϵ⁢\lt⁢1. Then |r2⁢n-ρ2|=|rn-ρ|⁢|rn+ρ|=(rn-ρ)⁢(rn+ρ)⁢\lt⁢(rn-ρ)⁢(ρ+ϵ)⁢\lt⁢rn-ρ⁢\lt⁢ϵ. Thus, limn→∞⁡r2⁢n=ρ2. Since a sequence can have only one limit, and the sequence of r2⁢n converges to both ρ and ρ2, then ρ=ρ2. Hence ρ=0 or ρ=1. But ρ=inf⁢{rn∣n∈ℕ}≤r⁢\lt⁢1. Hence ρ=0.

Thus, if 0⁢\lt⁢r⁢\lt⁢1, then limn→∞⁡rn=0.

Finally, if -1⁢\lt⁢r⁢\lt⁢0, then 0⁢\lt⁢|r|⁢\lt⁢1. Let ϵ⁢\gt⁢0. Then there exists N such that for all n≥N we have |rn|=||r|n|⁢\lt⁢ϵ, since limn→∞⁡|r|n=0. Thus, for all ϵ⁢\gt⁢0 there exists N such that for all n≥N, |rn-0|⁢\lt⁢ϵ. This proves that limn→∞⁡rn=0, as desired.

In summary, |r|⁢\lt⁢1

The argument suggested by Srivatsan Narayanan in the comments to deal with the case 0⁢\lt⁢|r|⁢\lt⁢1 is less clumsy than mine above: there exists a⁢\gt⁢0 such that |r|=11+a. Then we can use the binomial theorem as above to get that |rn|=|r|n=1(1+a)n≤11+n⁢a⁢\lt⁢1n⁢a. By the Archimedean Property, for every ϵ⁢\gt⁢0 there exists N∈ℕ such that N⁢a⁢\gt⁢1ϵ, and hence for all n≥N, 1n⁢a≤1N⁢a⁢\lt⁢ϵ. This proves that limn→∞⁡|r|n=0 when 0⁢\lt⁢|r|⁢\lt⁢1, without having to invoke the infimum property explicitly.
