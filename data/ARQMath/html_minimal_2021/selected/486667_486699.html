Proving that if R¯ transitive (where R equivalence relation), |A/R|=1

Let A≠∅ a set and R⊆A×A equivalence relation s.t the complementary relation R¯=(A×A)∖R is transitive. Prove that |A/R|=1 (cardinality of quotient set is 1).

There has to be a contradiction if the cardinality is bigger then 1 but I haven't found it yet. I'll be glad for any hint.

If I am not mistaken, the quotient set is the set of equivalence classes under R. Thus the assumption |A/R|=1 means there is exactly one such class, i.e. any two elements a,b∈A satisfy (a,b)∈R. So that's what we need to prove. Now, I will write a⁢R⁢b for (a,b)∈R from here on out. Suppose then that there are more than one class. Then there exists c∈A such that a⁢R⁢c is false. Thus (a,c)∈R¯, or a⁢R¯⁢c. I claim that we then have c⁢R¯⁢a as well. For if not, then c⁢R⁢a, and by the symmetry of R, a⁢R⁢c, a contradiction. So now the transitivity of R¯ implies both a⁢R¯⁢a and c⁢R¯⁢c, contradicting the reflexitivity of R, I.e. that x⁢R⁢x for all x∈A. Thus there is but one equivalence class under R. QED.
