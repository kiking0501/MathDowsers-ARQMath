Write f⁢(x)=1-xx2+2⁢x+c As x→±∞ this goes to 1. If the denominator has a real root it will go off to ±∞ so c⁢\gt⁢1. The denominator is then always positive, so the cases where f⁢(x)⁢\gt⁢1 are where x⁢\lt⁢0 and the cases where f⁢(x)⁢\lt⁢1 are where x⁢\gt⁢0. Take the derivative, set to zero, and find the x value of the local maximum as a function of c. Plug that x value into f⁢(x) and set it equal to 32. You will get an equation for c.
