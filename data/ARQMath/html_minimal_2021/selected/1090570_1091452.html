What you did is correct. But there is an easier way. Remember that for polynomial p⁢(x), if p⁢(a)=0 then (x-a) is a factor of p⁢(x).

Denote the determinant by Δ. It is obviously a polynomial in x,y and z. Now, note that:

  • x=0⟹Δ=0, so x is a factor of Δ. Same for y=0 and z=0.
  • x=y⟹Δ=0, so (x-y) is a factor of Δ. Similarly for y=z and z=x

Finally note that Δ is degree 6 polynomial. So it cannot have more than 6 linear factors, and we have listed all of them above. Clearly Δ=C⁢x⁢y⁢z⁢(x-y)⁢(y-z)⁢(z-x) where C is some constant. Taking some values (eg. x=1,y=2,z=3), we get C=1.
