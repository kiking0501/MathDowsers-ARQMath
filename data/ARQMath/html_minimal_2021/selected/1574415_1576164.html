Meaning of the Corollary 31.29 of the Chinese Remainder Theorem

I was reading CLRS and it had the following Corollary:

If n1,n2,…,nk are pairwise relatively prime and n=n1⁢n2⁢…⁢nk then for all integers x and a,

x≡a(modni)

for i=1,2,…,k if and only if

x≡a(modn)

There seems to be some words missing in the theorem/corollary. Should it read:

For all integers x and a that satisfy the equation:

x≡a(modni)

we have that it implies that:

x≡a(modn)

is also true. Furthermore, the converse is true too.

In other words, if: x≡a(modni) for some integer a and x, then it implies that x≡a(modn) and visa versa. i.e. If x≡a(modn) is true then x≡a(modni).

Is that correct?

Yes, the meaning is exactly that you mentioned. "If and only if" means that the implication is true in both directions.

It means something like x≡2(mod5) and x≡2(mod7) if and only if x≡2(mod35)
