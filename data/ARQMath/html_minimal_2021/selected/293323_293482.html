The following argument is admittedly on the intuitive side, using concavity mixed with "flatter than" comparisons.

If x<0 then since x2>0 we have 5x-4x<0<6x2-5x2. If 0<x<1 then in fact 0<x2<x<1, and the functions ux and ux2 are both concave down, with ux2 being flatter between 5 and 6 than is ux there. So 6x2-5x2<6x-5x<5x-4x. Finally if x>1 then in fact 1<x<x2 and the functions ux and ux2 are both concave up, with ux being flatter between 5 and 6 than is ux2 there. So 6x2-5x2>6x-5x>5x-4x.

If the above (admittedly vague) arguments are right, we have ruled out all possibilities other than x=0,1 for the equation 5x-4x=6x2-5x2.
