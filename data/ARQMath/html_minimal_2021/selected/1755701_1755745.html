Note that if limx→a⁡(f⁢(x)+1|f⁢(x)|)=0, then for x close enough to a, f⁢(x)+1|f⁢(x)|<1. A quick look at the graph of x+1|x| shows that this means f⁢(x) is negative, so you actually have that limx→a⁡(f⁢(x)-1f⁢(x))=0.

Some algebra shows that limx→a⁡(f⁢(x)-1f⁢(x))2+4=limx→a⁡(f⁢(x)+1f⁢(x))2=4, and thus that limx→a⁡(f⁢(x)+1f⁢(x))=±2. Noting that f is negative as x→a, we can take the negative root, thus seeing that:

limx→a⁡(f⁢(x)-1f⁢(x))=0

limx→a⁡(f⁢(x)+1f⁢(x))=-2

and thus, finally,that f⁢(x)→-1 as x→a.

Note: One can use the fact that g⁢(x)=x+1|x| has a continuous inverse around 0 directly, though this arguably presupposes extra knowledge about inverse functions. In any case, if you're comfortable using it, it provides a slightly quicker route to the answer.
