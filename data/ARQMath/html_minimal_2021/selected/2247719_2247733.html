What's max⁡(N1,N2,N3,…)?
You have confused maximum element with maximal element. A maximum element is unique but a maximal element is not. Hence the notation max⁡{N⁢1,N⁢2,…} is **not** justified.

In a general partially ordered set (S,≤), an element is called x maximal if there does not exist y such that x<y. For instance, if S={x1,x2,x3,x4,x5,x6,x7,x8,x9} and the order is the divisor order (xi≤xj if and only if i divides j), then the elements x5,x6,x7,x8,x9 are all maximal. This is much weaker than saying that for every y we have y≤x. Such an x might be called maximum or greatest. In finite sets this is the same as saying x is the unique maximal element. For example if we add the element x5⋅7⋅8⋅9 to our set then this is the greatest element.

In group theory, the relation of inclusion M⊆N is far more natural and useful than the weaker order relation |M|≤|N|, and a subgroup is called maximal if it is maximal with respect to inclusion. As you say this is the same as not being contained in a larger subgroup.
