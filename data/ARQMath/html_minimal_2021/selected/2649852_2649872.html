How to calculate the last digits of a big number using modular arithemtic?

I'm trying to calculate the last 2 digits of 9999 using modular arithmetic somehow. Please help.

See patterns in the last two digits of powers of 9. They will repeat after some time. To do this, you need not take powers of 9, but instead take the last two digits each time. For example, 92=81, 93=729, 29×9=261, etc. While doing this, you will see that this pattern recurs. Find the period of recurrence and the answer is 999 modulo the period.

I start with the long way that works every time. Skip to the last paragraph for a very short solution in this case.

Your goal is to compute 9999(mod100).

One way to compute this is to use Euler's totient function φ. Since the totient function is multiplicative, φ⁢(100)=φ⁢(22)⁢φ⁢(52)=(22-21)⁢(52-51)=2⋅20=40.

Euler's theorem states that aφ⁢(100)≡1(mod100) for a relatively prime to 100. Therefore, it is enough to compute 9999(modφ⁢(100))(mod100).

Therefore, our problem is reduced to 999(mod40). By following the same pattern, we see that φ⁢(40)=φ⁢(23)⁢φ⁢(5)=(23-22)⁢(51-50)=4⋅4=16. Therefore, we need to compute 999(modφ⁢(40))(mod40). Therefore, our problem reduces to computing 99(mod16). Following in our pattern, it is enough to compute 99(modφ⁢(16)). Since φ⁢(16)=24-23=8, we know that 99≡91=9(mod16). Therefore, 999≡99(mod40). Moreover, since 99=9⋅9⋅9⋅9⋅9⋅9⋅9⋅9⋅9, and 9⋅9=81≡1(mod40), 99≡9(mod40).

Putting this together, 9999≡99(mod100). Now, this is fairly straight-forward to compute since 99=(93)3. Since 93=729≡29(mod100), we have that 9999≡99≡(29)3(mod100). Since 293=24389≡89(mod100), we have that 9999≡99≡(29)3≡89(mod100). Therefore, the last two digits are 89.

It is faster, however, to notice that 910≡1(mod100), so we only need to compute 999(mod10) because multiples of 10 in the exponent correspond to multiplying by 1(mod100). Therefore, all we need is the remainder modulo 10 in the exponent, i.e., 9999≡9999(mod10)(mod100). Since 9≡-1(mod10), and 99 is odd, we know that 999≡-1≡9(mod10). Therefore, the original expression is equivaent to 99(mod100), which can be calculated as above (or on a calculator).
