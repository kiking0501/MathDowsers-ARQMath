Note that n⁢(n+1)⁢(2⁢n+1)=2⁢n⁢(n+1)⁢(n+1/2). Letting m=n+1/2, we get:

2780⋅3=(m-1/2)⁢(m+1/2)⁢m=m3-m/4 So we want m to be just over 3⋅27803. That should give you a candidate for n. You should check that this n works.

Note: The m trick is a bit of magic, but we know that n⁢(n+1)⁢(2⁢n+1) grows "like" 2⁢n3, so even if you don't find the m trick, you can guess it will be near 3⋅27803. The m trick makes it clear that if such n exists, there has to be only a single value.
