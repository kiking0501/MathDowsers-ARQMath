Continuity of a function containing infinite sum of floor function

How to find the points of discontinuity of the following function f⁢(x)=limn→∞⁡∑r=1n⌊2⁢r⁢x⌋n2

limits continuity limits-without-lhopital floor-function

Let x>0. Notice that 2⁢r⁢x-1n≤⌊2⁢r⁢x⌋n≤2⁢r⁢xn Hence -1n+1n⁢∑r=1n2⁢r⁢xn≤1n⁢∑r=1n⌊2⁢r⁢x⌋n≤1n⁢∑r=1n2⁢r⁢xn You can see the summation on the LHS (and RHS) as a Riemann sum: limn→∞⁡1n⁢∑r=1n2⁢r⁢xn=x⁢∫012⁢t⁢𝑑t=x The squeeze theorem finishes the proof that limn→∞⁡1n⁢∑r=1n⌊2⁢r⁢x⌋n=x A similar argument could be applied for x<0 yielding the same result. For x=0 the limit is zero. So the function in question is continuous.
