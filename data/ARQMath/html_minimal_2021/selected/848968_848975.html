We can proceed more simply in showing that E is bounded. Since {xn}n=1∞ is bounded, then there is some M>0 such that |xn|≤M for all n. Now, suppose that L∈E, and show that |L|<M+1, using the definitions. (Actually, you can show that |L|<M+ϵ for all ϵ>0, so that |L|≤M, but that's more than we really need.) Since L was an arbitrary element of E, then E is bounded.

Now, let's put α=sup⁡E. Since α-1 isn't an upper bound of E, then there is some L1∈E such that α-1<L1, and so there is some least n (say n1) such that xn>α-1. (Why?) Suppose we have found nk for some positive integer k. Since α-1k+1 isn't an upper bound of E, then there is some Lk+1∈E such that α-1k+1<Lk+1, and so there is some least n (say nk+1) such that xn>α-1k+1 and n>nk. (Why?) In this fashion, we construct a subsequence {xnk}k=1∞ of {xn}n=1∞ such that xnk→α. You'll need to justify that this actually is a (well-defined) subsequence, and that it converges to α. Your argument in the case that S is finite doesn't quite work. Rather, you can say that if L∈E, then L must be a sequence term that gets repeated infinitely-many times, so that L∈S. That shows you that E⊆S and so is bounded.
As for the infinite case, you are correct that any accumulation point of E will be a subsequential limit, but you haven't fully justified it here. (You *can* use a similar approach to the one I outline in showing that the supremum of E lies in E.)
