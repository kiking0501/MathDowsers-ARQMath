Since A is denumerable, there exists f:ℕ→A such that f is a bijection.

Define g:ℕ→A, g⁢(n)={x,a⁢m⁢p;if n=1,f⁢(n),a⁢m⁢p; if n≠1

Claim: g is also bijective.

Let n1,n2∈ℕ such that g⁢(n1)=g⁢(n2).

If g⁢(n1)=x, then n=1. If g⁢(n1)≠g⁢(n2), then g⁢(n1)=f⁢(n1)=g⁢(n2)=f⁢(n2). We have f⁢(n1)=f⁢(n2). Since f is a bijection, then F0,F1,F2,F3,F4. Therefore g is injective.

Let b∈A∪{x}. There exists n=1 such that g⁢(n)=x=b. If n≠1, then there exists m∈ℕ such that g⁢(m)=f⁢(m)=b. Since f is bijective, there exists m∈ℕ such that f⁢(m)=b=g⁢(m). So g is surjective.

Therefore g is bijective.

Therefore A∪{x} is denumerable.
