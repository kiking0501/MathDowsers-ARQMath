This is the number of ways to split m distinct balls into n boxes so that no box is empty (or the number of surjections f:{1,2,…⁢m}→{1,2⁢…⁢n}).

The proof is a simple inclusion/exclusion proof.

Let Ax be the set of functions that don't contain x in the range.

We wish to find nm-|A1∪A2⁢⋯∪An| (in other words, we want to count how many functions are not surjective).

Because of inclusion exclusion |A1∪A2⁢⋯∪An|=∑k=1n(-1)k+1⁢∑i1<i2⁢…<ik|Ai1∩Ai2⁢⋯∩Aik|

Clearly |Ai1∩Ai2⁢⋯∩Aik|=(n-k)m independently of the actual values of i1,i2⁢…⁢ik.

So our sum is just ∑k=1n(-1)k+1⁢(nk)⁢(n-k)m

Just to finish as smoothly as possible:

nm-∑k=1n(-1)k+1⁢(nk)⁢(n-k)m=∑k=0n(-1)k⁢(nk)⁢(n-k)m
