Can you help me with the limit of a function up and down bounded?

If 2⁢x-1≤f⁢(x)≤x2 for 0<x<3. Find limx→1⁡f⁢(x).
How can I solve this?

The squeeze theorem seems appropriate for this question.
https://en.wikipedia.org/wiki/Squeeze_theorem
2⁢x-1≤f⁢(x)≤x2 ; limx→1⁡2⁢x-1≤limx→1⁡f⁢(x)≤limx→1⁡x2 ; l⁢i⁢mx→1⁢x2=l⁢i⁢mx→1⁢2⁢x-1=1 ; so, l⁢i⁢mx→1⁢f⁢(x)=1?
Yes it's right..

Since f⁢(x)≤x2, then:

limx→1⁡f⁢(x)≤limx→1⁡x2=1.

On the other hand, since f⁢(x)≥2⁢x-1, then:

limx→1⁡f⁢(x)≥limx→1⁡2⁢x-1=1.

As a consequence, we get that:

1≤limx→1⁡f⁢(x)≤1.

Then:

limx→1⁡f⁢(x)=1.

This is a good example for the squeeze theorem. Indeed, if a⁢(x)≤f⁢(x)≤b⁢(x), then:

limx→k⁡a⁢(x)≤limx→k⁡f⁢(x)≤limx→k⁡b⁢(x).

Then, if limx→k⁡a⁢(x)=limx→k⁡b⁢(x)=q, we can conclude that:

limx→k⁡f⁢(x)=q.
