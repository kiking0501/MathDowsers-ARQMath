First question

If lim(x,y)→(0,0)⁡f⁢(x,y)=L then for any ϵ>0 there exist δ>0 such that ∥(x,y)∥<δ⟹|f⁢(x,y)-L|<ϵ. But ∥(x,y)∥<δ⇔r<δ. So, r<δ⟹|g⁢(r,θ)-L|<ϵ,∀θ∈[0,2⁢π).

The converse can be obtained in a similar way. If for any ϵ>0 there exist δ>0 such that r<δ⟹|g⁢(r,θ)-L|<ϵ, then (using again that ∥(x,y)∥<δ⇔r<δ) it is

∥(x,y)∥<δ⟹|f⁢(x,y)-L|<ϵ. That is, lim(x,y)→(0,0)⁡f⁢(x,y)=L

Second question

We have seen that if lim(x,y)→(0,0)⁡f⁢(x,y)=L then for any ϵ>0 there exist δ>0 such that r<δ⟹|g⁢(r,θ)-L|<ϵ,∀θ∈[0,2⁢π). Thus, limr→0+⁡g⁢(r,θ)=L. Indeed, θ doesn't need to have a given value. It follows from above that for any curve θ⁢(r) it is limr→0+⁡g⁢(r,θ⁢(r))=L.

Next we will give an example to show that the converse is false. Consider f⁢(x,y)=x2⁢yx4+y2. Then g⁢(r,θ)=r3⁢cos⁡θ⁢sin2⁡θr2⁢(r2⁢cos2⁡θ+sin2⁡θ)=r⁢cos⁡θ⁢sin2⁡θr2⁢cos2⁡θ+sin2⁡θ. Thus, g⁢(r,θ)→0 as r→0, ∀θ. But, through the path y=x it is limx→0⁡x3x4+x2=limx→0⁡xx2+1=0 and through the path y=x2 it is limx→0⁡x4x4+x4=12. Thus the limit doesn't exist.

The reason because it fails is that cos⁡θ⁢sin2⁡θr2⁢cos2⁡θ+sin2⁡θ is not uniformly bounded in θ. If previous expression were bounded then the converse would hold.
