Why (ℤ/p⁢ℤ)∗ is cyclic of order p⁢−⁢1?

It should be clear that p is prime. First of all note that ℤ/p⁢ℤ is a field, so every nonzero element is inverible. Let k<p-1 be the maximal order of the elements of ℤ/p⁢ℤ*. Then ∀a∈ℤ/p⁢ℤ*:ak=1. But this means that the polynomial xk-1 of degree k has p-1>k distinct roots, a contradiction, so k=p-1. So there is an element a of order p-1. Such an element is called a primitive root.

How we can prove the fact: ℤ/p⁢ℤ is cyclic ⟹ n=2,4,pk,2⁢pk?

This question is rather ambiguous. I suppose that you ask for a proof that ℤ/n⁢ℤ is cyclic for n=2,4,pk,2⁢pk for an odd prime p. The cases 2 and 4 are easily proved by hand. In the case n=pk we have the following : If a∈ℤn* (a simpler way of writing ℤ/n⁢ℤ*) is a primitive root of ℤp then either a or a+p is a primitive root of ℤp2, and if a is a primitive root of ℤp2 then a is a primitive root of ℤpk for k>2. You can find these proofs here.
