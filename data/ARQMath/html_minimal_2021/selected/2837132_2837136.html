Equivalence of Dedekind infinity and infinity, and Axiom of Choice

Lemma

Let X be an infinite set. Then:

  1. X contains a countable subset N such that NC=X⁢\smallsetminus⁢N is infinite;
  2. For any such N⊆X, X is in bijection with NC.

Proof.

  1. Pick x1∈X. X⁢\smallsetminus⁢{x1}≠\varnothing, so there exists x1≠x2∈X. Call F1:={x1},F2:={x2}. Now, X⁢\smallsetminus⁢F2≠\varnothing, as X is infinite and F2 is finite. So there exists x3∈X⁢\smallsetminus⁢F2. Call F3:=F2∪{x3}. Rinse and repeat, and with enough repetitions, for any n, there is Fn with n elements such that F1⊆F2⊆…⊆Fn⊆…. Now set N:=⋃nFn. Set f1:{1}→F1 by f1⁢(1)=x1. Set f2:{1,2}→F2 by f2⁢(1)=x1,f2⁢(2)=x2. Now f1 can be viewed as f2|{1}. Go on like this, and for any n you have fn:{1,…,n}→Fn, and fn|{1,…,n-1}=fn-1 (modulo viewing fn-1 as a Fn-valued function). Set f⁢(i)=fi⁢(i). This is a function defined on ℕ and taking values in F, and satisfies f|{1,…,i}=fi (modulo viewing fi as F-valued). Clearly it is defined on all ℕ, since for any i there is an fi. Suppose f⁢(n1)=f⁢(n2). Then, if m:=max⁡{n1,n2}, fm⁢(n1)=fm⁢(n2), but all fi are injective, so F0,F1,F2,F3,F4, making f injective too. If f is surjective, item 1 is proved. But any x∈F must belong to an Fi, so there is fi which has it as a value, that is we have k≤i:fi⁢(k)=x, but then f⁢(k)=x, proving f⁢(ℕ)=F. If FC is not infinite, just pick N:=f⁢(2⁢ℕ) as your new countable set, and its complement will be FC∪f⁢(2⁢ℕ+1), which is a superset of the countable f⁢(2⁢ℕ+1) and therefore must be infinite.
  2. Let N={x1,…,xn,…}. By 1, N contains a countable subset M={y1,…,yn,…}. Let us construct f:NC→X which is bijective. If x∈NC⁢\smallsetminus⁢M, set f⁢(x)=x. If x∈M, it will be a yn, so we can set f⁢(yn)=yn2 if n is even, and f⁢(yn)=xn+12 if n is odd. f is clearly injective, and it is also surjective. Indeed, let x∈X. If x∈NC⁢\smallsetminus⁢M, then f⁢(x)=x, so x is its own preimage. If x=yk, its preimage is y2⁢k. If x=xk, then its preimage is y2⁢k-1.

Corollary

X is infinite iff there exists A⁢\subsetneq⁢X such that A,X are in bijection.

Proof.

  • If X is finite, it is in bijection with {1,…,n} for some n. We now show those are not in bijection with any of their proper subsets by induction on n. If n=1, the only proper subset is the empty set, which is clearly not in bijection with {1}. Suppose we have proved our claim for n-1. Assume by way of contradiction there exists f:{1,…,n}→{1,…,n}⁢\smallsetminus⁢{x} for some x≤n, with f bijective. Pick y≤n such that y≠x. f|{1,…,n}⁢\smallsetminus⁢{y} will be a bijection from its domain to h:={1,…,n}⁢\smallsetminus⁢{x,y}. Then again, if we set g⁢(k)=k for k<y and g⁢(k)=k-1 for k>y, g is bijective from {1,…,n}⁢\smallsetminus⁢{y} to {1,…,n-1}, and g~:=g|{1,…,n}⁢\smallsetminus⁢{x,y} is bijective from its domain to {1,…,n-1}⁢\smallsetminus⁢{g⁢(x)}, so that g~-1∘h∘g is a bijection from {1,…,n-1} to {1,…,n-1}⁢\smallsetminus⁢{g⁢(x)}⁢\subsetneq⁢{1,…,n-1}, which contradicts the induction hypothesis.
  • As for an infinite set, we have an N such as is given by the above lemma, and the same lemma gives us NC⁢\subsetneq⁢X is in bijection with X, thus completing the proof.

This "repetition" is not valid in first-order logic. For any particular finite n, you can repeat the argument n times to prove the existence of elements x1,…,xn with the properties you want. But you can't repeat it infinitely many times; that would make your proof infinitely long. (Not to mention that even if you could have a next step in the proof after these infinitely many steps, there is no axiom of ZFC that would allow you to cobble these infinitely many elements xn into a single set, since you have no single formula that can refer to all of them.)

To formalize an infinite iterative process like this in ZF(C), you must use recursion. Specifically, you can prove a theorem that says if you have some function G defined on finite tuples, then there exists a unique infinite sequence (xn) such that xn=G⁢(x1,…,xn-1) for each n. But to use this theorem, you need to already have this function G, and it needs to be a function: it needs to uniquely tell you how to choose xn given x1,…,xn-1. The procedure you describe is just, "pick xn to be some element of X that is not equal to any of x1,…,xn-1", but this does not define a function since this xn is not uniquely determined.

Using the axiom of choice, you can fix this. In particular, at the start of your proof you can invoke the axiom of choice to let C be a choice function on all the nonempty subsets of X, and then you can define G⁢(x1,…,xn-1)=C⁢(X∖{x1,…,xn-1}).

So, in brief, the axiom of choice is needed to simultaneously choose all of the infinitely many elements xn you will use in your construction.
