𝐀𝐭𝐭𝐞𝐦𝐩𝐭: It is true that a set is countable if there exists an injective function f:S⁢→⁢N from S to the natural numbers ℕ. Since ℕ∪{a} has cardinality |ℕ|+1, ℕ∪{a} is not countable. However, there is a surjective mapping from ℕ∪{a}→ℕ. A set X is uncountably infinite if X is nonempty and there is no surjective function from the natural numbers to X. ℕ∪{a} is nonempty, and there is a surjective mapping from ℕ∪{a}→ℕ. But we had said that it is not countably infinite. This seems to be a contradiction, so ℕ∪{a} is countably infinite.
