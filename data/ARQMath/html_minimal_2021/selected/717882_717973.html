The second form is deceptive: the function $ \ x^{2/2} \ $ is _not_ formally the same as the function $ \ x \ . $ There is a domain issue that must be considered for the implied square-root in $ \ x^{2/2} \ , $ just as there would be for $ \ x^{3/2} \ . $
2/2=1. So x22 and x1 are the same thing.

The rule xp⁢q=(xp)q is not always valid if p or q are not integers and x<0. As you have noticed -1=(-1)1=(-1)22≠((-1)2)12=112=1.

So, unfortunately, x22≠x2 for x<0. For the last question: x22=x1=x is defined for all x.
