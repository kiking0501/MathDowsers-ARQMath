Prove by induction that (binomial theorem) (ss)+(s+1s)+⋯+(ns)=(n+1s+1) for all s and all n>s.

Your base case should be n=s+1 (for any s≥0). The induction step is an increment on n.
Note that s is fixed. It is n that is changing. The equation has the form ∑k=sn(ks)=n+1s+1. The RHS counts the number of (s+1)-element subsets of the (n+1)-element set {0,1,2,…,n}. The LHS counts the same thing since the term (ks), s≤k≤n, is the number of (s+1)-element subsets with largest element k.
