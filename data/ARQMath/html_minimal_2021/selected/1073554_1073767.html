Let's suppose that we are aware of the integral ∫Ck∏i=1kxiαi-1⁢d⁢xi=∏i=1kΓ⁢(αi)Γ⁢(∑i=1kαi) (𝟏)

(See here for a proof).

It turns out that only thing we need is to rewrite properly the inner one of the two products:

P⁢(k)=∏j=ik(xi⁢xj)αi⁢j-1⁢ =? Let's try the easy example k=3, (assuming the αi⁢j are symmetric)we get (x1⁢x1)α11-1⁢(x1⁢x2)α12-1⁢(x1⁢x3)α13-1⁢(x2⁢x2)α22-1⁢(x2⁢x3)α23-1⁢(x3⁢x3)α33-1 which equals x12⁢α11+α12+α13-4⁢x22⁢α22+α12+α23-4⁢x32⁢α33+α13+α23-4 or P⁢(3)=∏i3xi2⁢αi⁢i+∑j≠i3αi⁢j-4 It is obvious (or can be shown by induction) that one can generalize this to arbitrary k P⁢(k)=∏i=1kxiαi⁢i-1+∑j=1k(αi⁢j-1) Plugging this expression into our integral (using ∏ibi⁢∏iai=∏iai⁢bi) we see that it is given by ∫Ck∏i=1kd⁢xi⁢∏j=ik(xi⁢xj)αi⁢j-1=∫Ck∏i=1kd⁢xi⁢xiαi⁢i-1+∑j=1k(αi⁢j-1)

which allows us to just use (𝟏), replacing αi by αi⁢i+∑j=1k(αi⁢j-1)

∫Ck∏i=1kd⁢xi⁢∏j=ik(xi⁢xj)αi⁢j-1=∏i=1kΓ⁢(αi⁢i+∑j=1k(αi⁢j-1))Γ⁢(∑i=1k[αi⁢i+∑j=1k(αi⁢j-1)])

Edit: It appears that in my original answer, i assumed implicitly that the ai⁢j are symmetric αi⁢j=αj⁢i. If this is not the case, one has to modify the answer as follows: Replace ∑j≠ikαi⁢j by ∑j≠ik(Θ⁢(i-j)⁢αi⁢j+Θ⁢(j-i)⁢αj⁢i) where Θ⁢(x) is Heaviside's step function.
