For different proofs of Wilson's theorem see [here](https://en.wikipedia.org/wiki/Wilson%27s_theorem#Proofs) (they use Lagrange's theorem, though, not Cauchy's theorem).

There is a proof of Wilson's theorem using the Cauchy–Frobenius theorem on counting orbits: there exist (p-1)! cyclic permutations of p points. The cyclic group Cp acts on them by conjugation, which we can view geometrically as rotation. There are p-1 fixed cycles, hence Cauchy–Frobenius implies that (p-1)!≡p-1modp (for details see here).
