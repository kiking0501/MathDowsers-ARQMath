You want to solve

∇×𝐕=𝐔

with the condition ∇⋅𝐔=∂⁡A∂⁡x+∂⁡B∂⁡y+∂⁡C∂⁡z=0

Note that since the curl of any gradient field is 0, the solution will not be unique, i.e. if dim⁡ker⁡T+dim⁡Im⁡T=n is a solution, then another solution is 𝐕¯=𝐕+∇⁡ϕ, where ϕ⁢(x,y,z) is some scalar function.

It's enough to find one particular solution, then add on an arbitrary gradient field at the end. A solution can be found by setting one of the components of dim⁡ker⁡T+dim⁡Im⁡T=n to 0. Let's say H=0 so the system is reduced to

\begin{align} - \frac{\partial G}{\partial z} &= A \\ \frac{\partial F}{\partial z} &= B \\ \frac{\partial G}{\partial x} - \frac{\partial F}{\partial y} &= C \end{align}

Then integrate the first two equations

\begin{align} G &= - \int A\ dz + g(x,y) \\ F &= \int B dz + f(x,y) \end{align}

At this point, the two arbitrary functions leftover from integration should be set so that the third equation is satisfied

∂⁡G∂⁡x-∂⁡F∂⁡y=-∫(∂⁡A∂⁡x+∂⁡B∂⁡y)⁢𝑑z+∂⁡g∂⁡x-∂⁡f∂⁡y=C

Suppose that

-∫(∂⁡A∂⁡x+∂⁡B∂⁡y)⁢𝑑z=C+h⁢(x,y)

where h⁢(x,y) is a known function, then we must have

∂⁡g∂⁡x-∂⁡f∂⁡y+h=0

To make it easy again, we can arbitrarily set g=0 and f=∫h⁢𝑑y

You just need to find *one* valid solution, then add on an arbritrary gradient field at the end. Any solution will be fine, so you can make an educated guess that makes the problem easier to solve.
