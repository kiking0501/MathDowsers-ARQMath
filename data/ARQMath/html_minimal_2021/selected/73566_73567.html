The problem with your argument is you are assuming that SΩ is at least as big as each Sn, but at some point you need to directly analyze SΩ. So you might as well forget Sn and directly think about SΩ. Then you just need to find infinitely many permutations of Ω and you're done. You can pick easy ones...
You should first realize all the finite groups Sn as subgroups of SΩ. This is easy: extend an element σ of Sn to an element of SΩ by defining σ⁢(k)=k for all k>n. Then you can proceed according to your plan, and derive a contradiction from the contrapositive assumption that |SΩ| would be finite. Too bad that you didn't choose Ω={0,1,2,…}. Then you could follow Jim's suggestion e.g. by bitwise XORing with an arbitrary integer, and produce at least |Ω| distinct permutations :-)
Let σ∈Sn (where Sn is the permutation group on n letters) and *define* σ¯∈SΩ by the rule σ¯⁢(i)=σ⁢(i) if 1≤i≤n and σ¯⁢(i)=i if i>n. If fn:Sn→SΩ is *defined* by fn⁢(σ)=σ¯, then prove that fn:Sn→SΩ is injective. In particular, |SΩ|≥n! for all positive integers n and thus SΩ is an infinite set.

Note that there are two common interpretations for SΩ when Ω is infinite: it can be the group of all bijections Ω→Ω, or it can be the group of all permutations "of finite support" (that is, all bijections σ:Ω→Ω such that {x∈Ω∣σ⁢(x)≠x} is finite). Doesn't affect the answer here, because the latter is already infinite, but is a good idea to specify which one you mean (just like one should always say if D2⁢k means the dihedral group of *order* 2⁢k, or the dihedral group of *degree* 2⁢k).

Nope. Unfortunately, what you have written is not rigorous enough. Here is an argument. (In general, whenever you want to prove that some set is infinite, assume it is finite and using the elements construct another element which is not in the set. This is in the spirit of Euclid's argument for infinitude of primes.)

Let SΩ={f:Ω→Ω|f⁢ is bijective}. We shall first prove that SΩ is an infinite set. Assume that SΩ has only finitely many elements, say N of them. Let them be f1,f2,…,fN. We shall denote each fk as follows. fk={ak⁢1,ak⁢2,…,ak⁢n,…} where ak⁢l=fk⁢(l). We then have the following.

f1={a11,a12,a13,…,a1⁢N,…}f2={a21,a22,a23,…,a2⁢N,…}f3={a31,a32,a33,…,a3⁢N,…}⋮⋮⋮fN={aN⁢1,aN⁢2,aN⁢3,…,aN⁢N,…}

Let M=max⁡{a11,a22,a33,…,aN⁢N}. We shall now construct a f which is a bijection from Ω to Ω but doesn't match with any of the fk's above. f⁢(m)={M+mm∈{1,2,…,M}m-Mm∈{M+1,M+2,…,2⁢M}mm≥2⁢M+1 Clearly, f is a bijection from Ω to Ω. In-fact, the inverse of f is itself. It is also easy to check that f doesn't match with any of the fk's. If f were to match with fk for some k, then we must have fk⁢(k)=f⁢(k). However, we have that fk⁢(k)=ak⁢k≤M<M+k=f⁢(k). Hence, f doesn't match with any of the listed fk's. This is true for any N∈ℕ. Hence, the set SΩ is an infinite set.

Now we need to prove that SΩ is a group.

Consider f1,f2∈SΩ. Let f=f1∘f2. If we have f1⁢(f2⁢(a1))=f1⁢(f2⁢(a2)), then since f1 is a injective, we have f2⁢(a1)=f2⁢(a2) and since f2 is a injective, we get a1=a2. Hence, f1∘f2 is injective. Given any c∈{1,2,3,…}, since f1 is surjective, ∃b∈{1,2,3,…} such that f1⁢(b)=c. Since f2 is surjective, ∃a∈{1,2,3,…} such that f2⁢(a)=b. Hence, ∃a∈{1,2,3,…} such that f1⁢(f2⁢(a))=c. Hence, SΩ is closed under function compositions.

The associativity follows from the fact that composition of functions is associative.

The map f⁢(n)=n, ∀n∈Ω acts as the identity map since for any g∈SΩ, we have g⁢(f⁢(n))=g⁢(n) and f⁢(g⁢(n))=g⁢(n).

For any f∈SΩ, since f is a bijection, there exists an inverse map, g, which is also a bijection and hence g∈SΩ. Hence, any element in SΩ has an inverse.

Hence, SΩ is a group under function compositions.
