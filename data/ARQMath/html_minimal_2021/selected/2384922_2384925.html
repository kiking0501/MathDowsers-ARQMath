set-theory definition

As Jech says, "ZFC ... has only one type of object, namely sets". The axiom of extensionality says that all objects of the universe behave like sets: for any x and y, if z∈x iff z∈y for all z, then x=y. That is, every object is determined by which other objects are elements of it. In this sense, "everything is a set" in ZFC, and so in the context of ZFC, the term "set" just refers to any object in the universe. There are no objects other than sets which could be elements of sets.

And Foundation ensures that objects don't have weird loopiness, so everything behaves even more like sets.
