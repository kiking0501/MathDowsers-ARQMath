Euler's theorem doesn't say that aφ⁢(n)≡1(modn)⇔gcd⁡(a,n)=1 The implication is one sided only

The problem is that Euler's theorem does not give the smallest exponent for which ak≡1modn. That smallest exponent is given by the Carmichael function λ⁢(n), and λ⁢(21)=6 (as compared to φ⁢(21)=12), so we have 26≡1mod21 This yields x=11+6⁢k=5+6⁢k, as desired.
