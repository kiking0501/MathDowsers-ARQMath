I have literally never heard the \vDash relation referred to as a "tautology". A tautology is the same thing as a valid formula of propositional logic.

In \vDash⁢φ is is not \vDash that is a tautology. Rather, \vDash⁢φ asserts that φ is a semantic consequence of nothing -- which happens to be the case when φ is a tautology.

@user525966: Yes, something that is a semantic consequence of nothing is always true. However \vDash is neither "valid" nor a tautology. \vDash is the symbol for (a) the semantic consequence relation and (b) the evaluation relation in a particular interpretation. The latter of these is used to _define_ validity (and tautologies) but it makes no sense to say that the \vDash relation itself is "valid" or a "tautology". @user525966: It means that φ is "valid" and "always true" and "a semantic consequence of the empty set". If φ is a _propositional_ formula this is also expressed by the word "tautology". The word "sound" is not used about formulas.
@HenningMakholm How can φ be always true and yet not a tautology outside of propositional logic?
@user525966: For example, in first-order logic the formula ∀x(x=x) is always true, but is not a tautology. The notation Γ⊧φ means φ is true in every structure in which all statements in Γ are true.

The notation Γ⊢φ means φ can be proved by using the statements in φ.

The latter depends on some notion of proof. One wants such a notion to satisfy three desiderata:

  • Soundness, i.e. if Γ⊢φ then Γ⊧φ.
  • Completeness, i.e. if Γ⊧φ then Γ⊢φ.
  • Effectiveness, i.e. there is a proof-checking algorithm, which will correctly identify its input as either a valid proof or not.

Note that Γ is allowed to be a set of statements whereas φ is just one statement. So why not conjoin all the statements in Γ into one? The problem here is that if one is allowed to join infinitely many statements into one, then putting such an infinite conjunction in the role of φ has the result that one cannot satisfy all three desiderata simultaneously. Any proof of that takes a lot of work.

Does \vDash⁢φ (empty left-hand side) mean φ is a tautology?
@user525966 : It means φ is true in every "structure" (I haven't been explicit here about what a "structure" is). Sometimes people reserve the word "tautology" for things in propositional logic rather than predicate logic (predicate logic uses universal and existential quantifiers; propositional logic only uses Boolean connectives such as "and", "or", and "not"). Sometimes people express it by saying this means φ is "universally valid".
How would you describe "structure"? Is this the same as "interpretation" or "model"? What does it mean?
@user525966 In the standard semantics of classical first order logic, sentences are interpreted in structures, i.e. structures are the interpretations. For instance, the first order sentence ∀x⁢∃y⁢(x⁢R⁢y) can be interpreted in the structure (ℝ,<) (where we're interpreting the relation symbol R as <) to mean the reals have no maximal element. So it's true in this structure. If a set of first order sentences are all true in a structure, we say the structure is a model of the set of sentences. https://en.wikipedia.org/wiki/Structure_(mathematical_logic)
