I. Transformation

Given any QDE,

a⁢x2+b⁢x⁢y+c⁢y2+d⁢x+e⁢y+f=0

then it can be transformed to two Pell-type equations,

ui2-D⁢vi2=ki

as,

(D⁢y-2⁢a⁢e+b⁢d)2-D⁢(2⁢a⁢x+b⁢y+d)2=4⁢a⁢(a⁢e2+c⁢d2-b⁢d⁢e+D⁢f)

(D⁢x-2⁢c⁢d+b⁢e)2-D⁢(2⁢c⁢y+b⁢x+e)2=4⁢c⁢(a⁢e2+c⁢d2-b⁢d⁢e+D⁢f)

with the same discriminant D=b2-4⁢a⁢c.

But there is a problem. Once you've found integer u,v then, by undoing the transformation, it does not guarantee that x,y will be integers as well.

II. Example

4⁢x2+20⁢x⁢y+5⁢y2-24⁢x-20⁢y-73=0

By formulas (3),(4), this can be transformed to either,

p2-320⁢q2=-4×113920 r2-320⁢s2=-5×113920

Let's use the Alpertron:

  1. It gives a small solution p,q=80,38 which yields rational x,y=378,54.
  2. Fortunately, another family starts with p,q=640,54 and gives integer l⁢x+m⁢y+n=0.
  3. We also have r,s=280,45 which yields x,y=118,154.
  4. But r,s=480,50 givesl⁢x+m⁢y+n=0.

Thus, if you're not fortunate, you'll only end up with rational x,y.
