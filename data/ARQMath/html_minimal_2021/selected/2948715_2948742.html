if we show P⁢(m),m<n⟹P⁢(n) then somehow that must mean that P⁢(0) is true (or P⁢(b) is true for some set of base cases b∈B⁢a⁢s⁢e⁢C⁢a⁢s⁢e⁢s)

  1. show P(0)

or for strong induction

  1. show P(0)

  1. ∀m,m<n,P⁢(m)⟹P⁢(n)

A valid proof by complete induction includes a uniform proof for all k of the inferences listed below. As such it necessarily includes a proof ($\rm\color{#0a0}{vacuous}$) of the base case P⁢(0). See the schema below.

$$\begin{align} \color{#0a0}{\bbox[3px,border:2px solid #0a0]{\phantom{:}}}\Rightarrow\,\color{#c00}{ P(0)}\\ P(0)\Rightarrow\, P(1)\\ P(0),P(1)\Rightarrow\, P(2)\\ \vdots\qquad\ \ \ \ \\ P(0),P(1),\ldots,P(k-1)\,\Rightarrow\,P(k)\\ \end{align}\qquad\qquad\qquad\qquad\ \ $$

While a valid inductive proof necessarily implies a proof of P⁢(0), this may not occur explicitly. Rather, it may be a special case of a much more general implication derived in the proof. For example, in many such proofs the natural base case(s) is not a single number but rather a much larger set. Let's examine a simple induction where the base cases are all odd naturals.

If n≥1 is an integer then n=2i⁢j for some odd j and some integer i≥0. For if n is odd then n=20⁢n, else n=2⁢k for  1≤k<n so induction ⇒k=2i⁢j, so n=2⁢k=2i+1⁢j.  QED

Here the base case P⁢(1) is not explicitly proved. Instead it is a special case of the more general inference that n odd ⇒n=20⁢n. In such factorization (decomposition) problems the natural base cases are all irreducibles (and units) - not only the least natural in the statement, e.g. in the proof of existence of prime factorizations of integers n>1, with base cases being all primes.

Remark Ditto for its contrapositive form infinite descent: if given a counterexample ¬⁢P⁢(n) we can prove there exists a smaller counterexample ¬⁢P⁢(k),k<n, then no counterexample exists, else iterating the proof would yield an infinite descending chain of counterexamples, contra ℕ is well-ordered. Or, reformulated, if there is a counterexample then, by well order, we can choose a minimal one ("minimal criminal"), contra the proof yields a smaller one.
