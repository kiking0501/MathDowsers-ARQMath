If f is continuous and f⁢[a,b] contains only finitely many real numbers, then f is constant on [a,b]

Suppose that f:[a,b]→ℝ is continuous and that f⁢[a,b] contains only finitely many real numbers. Prove that f is constant on [a,b].

calculus real-analysis
An idea: use the Intermediate Value Theorem.
f⁢([a,b]) is also connected since [a,b] is connected . But if you have only finitely value then contradiction

The continuous image of an interval ([a,b] is an interval) is connected, that is, you cannot separate the image as a union of disjoint non-empty open subsets. Since there are only finitely many points, there must only be one point, since ℝ has the property that any two distinct points can be separated by disjoint non-empty open sets. Conclude f is constant.
