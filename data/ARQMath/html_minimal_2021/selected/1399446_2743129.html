X¯ is the UMVUE for θ. In fact, you haven't expressed the minimal sufficient statistic correctly.

The following result will make this clear:

Theorem: Given simple random sample from a population X with density of the form

fθ⁢(x)=c⁢(θ)⁢h⁢(x)⁢e∑j=1pqj⁢(θ)⁢Tj⁢(x)

where the functions qj⁢(θ) and Tj⁢(x) are such that there are no linear combinations of them so that it could be possible to simplify the exponent, then the statistic

(∑i=1nT1⁢(Xi),∑i=1nT2⁢(Xi),…,∑i=1nTp⁢(Xi))

Is a minimal sufficient statistic for the given population

If in this case we have X∼N⁢(θ,θ), the density of the population is

fθ(x)=1θ⁢2⁢πe-(x-θ)22⁢θ2 ,x∈ℝ,θ>0

So we can give the following factorization

fθ(x)=1θ⁢2⁢πe-12e-x22⁢θ2+xθ ,x∈ℝ,θ>0

making the follwing identifications:

c⁢(θ)=1θ⁢2⁢π⁢e-12,h⁢(x)=1,q1⁢(θ)=-12⁢θ2,q2⁢(θ)=1θ,T1⁢(x)=x2,T2⁢(x)=x

And applying the previous theorem, gives us that the minimal sufficient statistic for this distribution is

T=(∑i=1nXi2,∑i=1nXi)

Of course, we can't simplify the exponent further, as the vectors from T are linearly independent.

We would like to check as well that this is a complete statistic, but there is no need to verify that is satisfies the definition, as the following result states:

Theorem: In the same conditions of the previous theorem, the statistic

(∑i=1nT1⁢(Xi),∑i=1nT2⁢(Xi),…,∑i=1nTp⁢(Xi))

is complete if the image of the function (q1⁢(θ),q2⁢(θ),…,qn⁢(θ)) contains an open subset of ℝp

(see E.L. Lehmann "Testing Statistical Hypothesis", Springer Texts in Statistics, 1986)

It is clear that the function q⁢(θ)=(q1⁢(θ),q2⁢(θ))=(-12⁢θ2,1θ) contains some open subset of ℝ2 for θ>0, so we conclude from the previous theorem that the statistic T is minimal, sufficient and complete.

To conclude that X¯ is the UMVUE for θ, note that

T⋆=(X¯,s2)

Is another minimal sufficient and complete statistic, and also

Eθ[X¯]=θ  ,θ>0

So we conclude from the Lehmann-Scheffé theorem that X¯ is the UMVUE for θ as it is the only unbiased estimator for θ that is function of T⋆

To see a proof - using the definition - that in a normal population the statistic (X¯,s2) is complete, see https://math.stackexchange.com/a/2741439/351735
