It may be laborious when c is huge, but b being huge should be no problem.

The Chinese remainder theorem can reduce the computation needed. For example, we can factor 21=3⋅7, and have

1⋅7-2⋅3=1

(in general, we can use the extended Euclidean algorithm to produce this formula)

Consequently, if

x≡a(mod3)  x≡b(mod7)

then

x≡a⋅(1⋅7)+b⋅(-2⋅3)(mod21)

Thus, we can compute 5844325mod21 by using our favorite means to compute:

5844325≡2(mod3)  5844325≡3(mod7)

and thus

5844325≡2⋅7+3⋅(-6)≡-4≡17(mod21)

Fermat's Little Theorem: 24(54⁢n+1)+11≡1mod19, where n∈ℤ+.
