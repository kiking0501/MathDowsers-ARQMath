Is it true that the linear concept has different meaning when applied to equations and to functions?

Everyone calls the "y=a⁢x+b" a linear equation even though the function f:ℝ→ℝ f⁢(x)=y=a⁢x+b is not a linear function.

Should we think that the term "Linear" when applied to "equations" means that the solutions (x,y) to the equation represents points (x,y) that form a straight-line whereas the term "Linear" when applied to functions means that the mapping holds the superposition property ( http://en.wikipedia.org/wiki/Linearity ) ?

You are right. There are two common concepts that share the name linear. There are linear transformations, from linear algebra, which are functions between vector spaces that satisfy the conditions in the Wikipedia article you linked to. Then there are polynomials of the form a⁢x+b, which are usually called linear polynomials, although when viewed as a function (on a field, say) they are not linear transformations.

As you said, graphing the function x↦a⁢x+b (as a function ℝ→ℝ), we get a straight line, so it makes sense to call it linear. Typically, though, a function from a vector space to another one that is of the form x↦A⁢x+b, where A is some linear transformation and b is a vector, is called an affine transformation.
