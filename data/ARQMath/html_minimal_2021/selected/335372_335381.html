Determining Stirling number

In the first part of the question I was asked to find the exponential generating function for sn,r, the number of ways to distribute r distinct objects into n (a fixed constant) distinct boxes with no empty box.

I determined the generating function to be (ex-1)n. The problem then asks to determine sn,r. I'm not sure how this is any different from the first part of the question, but I thought maybe it was asking for a coefficient of xr/r! How do I do this?

combinatorics generating-functions integer-partitions stirling-numbers

Employ the following two facts:

(ex-1)n=∑k=0n(nk)⁢ek⁢x⁢(-1)n-k,ek⁢x=∑r≥0krr!⁢xr.

The first is from the binomial theorem and second is from the Taylor expansion of exp.
