Is the theorem sharp? Suppose f is not monotonic, simply C1 on [a,b], and still f⁢([a,b])=[c,d] , [c,d]⊂[a,b]. Is there still a fixed point?

First, consider g⁢(c)=f⁢(c)-c. g⁢(c)≥0 is obviously true because f⁢([c,d])→[c,d].

If g⁢(c)=0 then we have our x and we're done.

So, now let's consider g⁢(c)>0 and the value of g⁢(d).

Through a similar argument as used for g⁢(c)≥0, we have g⁢(d)≤0.

Again, if g⁢(d)=0 we're done.

So we have the three cases g⁢(c)=0, g⁢(d)=0, or g⁢(c)>0 and g⁢(d)<0.

In the last case, if g⁢(x) is continuous (which it is if f⁢(x) is), we use the Intermediate Value Theorem to prove existance of i⁢d1,0:(X,τ1)→(X,τ0) such that g⁢(α)=0.

So, as long as f⁢(x) is continuous and f⁢([c,d])→[c,d], we have shown ∃α∈[c,d] such that f⁢(α)=α.
