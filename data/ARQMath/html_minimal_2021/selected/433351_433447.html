I've added link to the [revision](http://www.proofwiki.org/w/index.php?title=Axiom_of_Choice_Implies_Zorn%27s_Lemma&oldid=106803) of the article at the time of posting the question. (It is quite probable that it will be further edited.)

For your first question, remember the hypothesis of Zorn’s lemma: every chain in ⟨X,⪯⟩ has an upper bound in X. Thus, if C∈𝕏, there is an x∈X such that y⪯x for each y∈C. But then C⊆s¯⁢(x)∈𝕊 by the definition of s¯.

For your second question, suppose that C∈𝕏: C is a chain in ⟨X,⪯⟩. In particular, C is a subset of X. Thus, 𝕏⊆℘⁢(X), and it makes sense to talk about the partial order ⟨𝕏,⊆⟩. If 𝒞 is a chain in this partial order, and C1,C2∈𝒞, then either C1⊆C2 or C2⊆C1: that’s what it means to say that 𝒞 is a chain in ⟨𝕏,⊆⟩. This does not guarantee, however, that ⋃𝒞 belongs to the chain 𝒞.

In ⟨℘⁢(ℝ),⊆⟩, for instance, {[1n,1]:n∈ℤ+} is a chain of sets, but the union of this chain is the set (0,1], which is not in the chain.

All that we can say for sure is that ⋃𝒞∈𝕏, i.e., that ⋃𝒞 is a chain in ⟨X,⪯⟩. (To see this, suppose that x,y∈⋃𝒞. Then there are C1,C2∈𝒞 such that x∈C1 and y∈C2. Since 𝒞 is a chain, either C1⊆C2, or C2⊆C1. Without loss of generality suppose that C1⊆C2. Then x,y∈C2, and since C2 is a chain in ⟨X,⪯⟩, either x⪯y or y⪯x. Thus, x and y in ⋃𝒞 are comparable, and by definition ⋃𝒞 is a chain in ⟨X,⪯⟩.)

The same is true for any 𝒯⊆𝕏: if 𝒞 is a chain in ⟨𝒯,⊆⟩, then ⋃𝒞∈𝕏, but we have no guarantee that ⋃𝒞 actually belongs to 𝒯. That’s why the definition of tower includes clause (3): we only want to consider those subsets 𝒯 of 𝕏 that do contain the unions of their chains.

For your third question, let 𝕋 be the set of all towers in 𝕏, and let 𝒯0=⋂𝕏. We have to show two things: that 𝒯0 is a tower in 𝕏, and that if 𝒯 is any tower in 𝕏, then 𝒯0⊆𝒯. To see that 𝒯0 is a tower in 𝕏, we must simply verify that it satisfies the three properties defining towers.

  1. ∅∈𝒯 for each 𝒯∈𝕋, so ∅∈⋂𝕋=𝒯0.
  2. If A∈𝒯0, then A∈𝒯 for each 𝒯∈𝕋, and therefore g⁢(A)∈𝒯 for each 𝒯∈𝕋. But then g⁢(A)∈⋂𝕋=𝒯0, as desired.
  3. If 𝒞 is a chain in 𝒯0, then 𝒞 is a chain in 𝒯 for each 𝒯∈𝕋, and therefore ⋃𝒞∈𝒯 for each 𝒯∈𝕋. But then ⋃𝒞∈⋂𝕋=𝒯0, as desired.

Now we know that 𝒯0 is a tower in 𝕏. If 𝒯 is any tower in 𝕏, then 𝒯0=⋂𝕋⊆𝒯, so 𝒯0⊆𝒯. That is, 𝒯0 is a subset of every tower in 𝕏; and since 𝒯0 is itself a tower in 𝕏, it is therefore the smallest tower in 𝕏. (Here smallest means smallest in the sense of ⊆; it is not a statement about cardinality.)

For your final question, recall that at that point in the proof the goal is to show that 𝒯0 is a chain in ⟨𝕏,⊆⟩. We’ve defined an element C∈𝒯0 to be comparable if it’s comparable (with respect to ⊆) with each element of 𝒯0, i.e., if for each A∈𝒯0, either A⊆C or C⊆A. We’ve proved that if C∈𝒯0 is comparable, it actually satisfies a stronger statement: for each A∈𝒯0, either A⊆C, or g⁢(C)⊆A. (This is stronger because A⊆g⁢(A) for each A∈𝕏, so if g⁢(C)⊆A, then certainly C⊆A.) But then for each A∈𝒯0, either A⊆C⊆g⁢(C), or g⁢(C)⊆A, and by definition g⁢(C) is comparable. If 𝒯C={C∈𝒯0:C⁢ is comparable}, this shows that g⁢(C)∈𝒯C whenever C∈𝒯C. And since ∅ is certainly comparable, 𝒯C satisfies the first two requirements to be a tower in 𝕏.

Suppose that we can show that 𝒯C also satisfies the third requirement and is therefore a tower in 𝕏. Then on the one hand we’ll have 𝒯C⊆𝒯0, but on the other hand we know that 𝒯0 is a subset of every tower in 𝕏, so we’ll have 𝒯0⊆𝒯C. Thus, 𝒯C=𝒯0. But that means that every element of 𝒯0 is comparable: for any C∈𝒯0, C is comparable, so for any A∈𝒯0, either C⊆A or A⊆C. And that means iprecisely that ⟨𝒯0,⊆⟩ is a chain.

All that’s missing, then is to verify that 𝒯C satisfies the third condition on towers: if 𝒞 is a chain in 𝒯C, then ⋃𝒞∈𝒯C. Suppose, then that 𝒞 is a chain in 𝒯C, and let C=⋃𝒞. 𝒯C⊆𝒯0, so 𝒞 is a chain in 𝒯0. 𝒯0 is a tower, so C∈𝒯0. To complete the proof, we need to show that C is comparable, so that C∈𝒯C.

Let A∈𝒯0 be arbitrary; we want to show that either A⊆C or C⊆A. Suppose that there is a C1∈𝒞 such that A⊆C1; then A⊆C1⊆⋃C=C. If there is no such C1, then C1⊆A for each C1∈𝒞. (Remember that every C1∈𝒞 is comparable, so if A⊈C1, then C1⊆A.) But then C=⋃𝒞=⋃C1∈𝒞C1⊆A. Thus, either A⊆C, or C⊆A, and C is indeed comparable.
