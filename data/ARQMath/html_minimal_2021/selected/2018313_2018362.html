In Differential Geometry, why does it make sense for a smooth vector field to act on a smooth function?

For instance consider the smooth vector field ϕ:ℳ⟶T⁢ℳ that maps every point on a smooth manifold ℳ to an element in the tangent space of ℳ.

A lot of books, after defining the idea of a smooth vector field, move on to talk about the space of all such vector fields Γ⁢(T⁢ℳ). It's further expressed that this space forms a C∞⁢(ℳ) - module but in order to show this, they mysteriously talk about the tangent vector fields as functions from C∞⁢(ℳ) to C∞⁢(ℳ). Am I missing a step here? I thought the vector fields are functions on the manifold, i,e, the argument of the function is a point on the manifold. How can a vector field of this sort act on a smooth function? It doesn't sense to me. Can someone please clarify this for me. An elaborate answer would be much appreciated as I'm just a beginner in this subject.

In differential geometry, a tangent vector v→=|v|⁢v^ at a point p is essentially an equivalent class of curves crossing that point p along direction v^ with speed |v|. It act on any smooth functions defined over some neighbor of p by taking directional derivatives along any member curve of corresponding equivalent class. In short, tangent vector v is really directional derivatives in disguise. So it is natural to talk about action of a tangent vector field on a smooth function.

You are really asking two different questions here.

Γ⁢(T⁢M) is a C∞-module simply because for every f∈C∞⁢(M) and ϕ∈Γ⁢(T⁢M), one can define a new vector field f⁢ϕ defined as follows: for x∈M, (f⁢ϕ)x=f⁢(x)⋅ϕx. Here the real number f⁢(x) is being multiplied to the tangent vector ϕx. This defines an action of C∞⁢(M) on Γ⁢(T⁢M).

On the other hand, the tangent space Tx⁢M can be defined as the set of all derivations on the germs of all smooth real-valued functions in a neighborhood of x. So, given ϕ∈Γ⁢(T⁢M) and f∈C∞⁢(M), we can define a smooth real-valued function ϕ⁢(f) by ϕ⁢(f)⁢(x)=ϕx⁢(f). (Note that ϕx∈Tx⁢M is a derivation on the set of germs of all smooth functions defined near x, of which f is a member) Thus we have a map ϕ:C∞⁢(M)→C∞⁢(M),f↦ϕ⁢(f).
