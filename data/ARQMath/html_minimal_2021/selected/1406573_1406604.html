Given any non compact set A⊂ℝn does there exist a continuous function f:A→ℝ which is not uniformly continuous?

It's well known that if A⊂ℝn is compact then every continuous function f:A→ℝ is uniformly continuous.So the obvious question is:

Given a non compact set A⊂ℝn does there always exist a continuous function f:A→ℝ which is not uniformly continuous?

Any ideas?

f⁢(x)=1x is continuous on (0,1) but not uniformly continuous.

Proof: The proof is by ϵ,δ method, by choosing δ=1x0 we get that the δ depend to the point of continuity so the continuity is not uniform.
