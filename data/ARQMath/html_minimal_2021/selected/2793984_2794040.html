If X is a random variable with value in S, and g:S→ℝ a real function, then you can use the formula E⁢[g⁢(X)]=∑x∈Sg⁢(x)⁢pX⁢(x)     for X discrete random variable and E⁢[g⁢(x)]=∫-∞+∞g⁢(x)⁢fX⁢(x)     for X continuous random variable

You should probably add that S is at most countable.
