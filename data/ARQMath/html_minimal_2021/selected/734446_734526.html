Let ε=|a|/4 so there is a δ>0 such that |f⁢(x)-L|<ε whenever 0<|x-a|<δ. Let 0<|x-a|<min⁡(δ,|a|/2). From the last inequality it follows that |x|>|a|/2. If x is irrational |x-L|<ε and if y is rational |L|<ε. Suppose that x is irrational. Then
