@AndréNicolas Well, not the main way, but using that method. I'll edit the post.

Let X be the random variable which counts the number of flips till we get a T (tails). Then: E⁢[X]=12⋅1+12⁢(1+E⁢[X]) ⟹E⁢[X]=2 The first term on LHS is if we get a T on a our first flip (with probability 12). The second term is if we get a H (heads) on our first flip (with probability 12), which means that we are effectively starting over with 1 extra flip.

Now let Y be the random variable which counts the number of flips till we get a HT sequence. Then: E⁢[Y]=12⁢(1+E⁢[Y])+12⁢(1+E⁢[X]) ⟹E⁢[Y]=4 The first term on LHS is if we get a T on our first flip (with probability 12), which means we are effectively starting over with 1 extra flip. The second term if we get a H on our first flip (with probability 12), then all we want is a T, which means we want to count 1 + expected number of flips to get a T.
