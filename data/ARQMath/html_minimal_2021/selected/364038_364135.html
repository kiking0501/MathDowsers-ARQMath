Let e be the expected number of tosses. It is clear that e is finite.

Start tossing. If we get a tail immediately (probability 12) then the expected number is e+1. If we get a head then a tail (probability 14), then the expected number is e+2. Continue …. If we get 4 heads then a tail, the expected number is e+5. Finally, if our first 5 tosses are heads, then the expected number is 5. Thus e=12⁢(e+1)+14⁢(e+2)+18⁢(e+3)+116⁢(e+4)+132⁢(e+5)+132⁢(5). Solve this linear equation for e. We get e=62.

If one wants, let X be the number of tosses. Then Pr⁡(X=n)≤(1/2)n-5. So E⁢(X)≤∑n⁢(1/2)n-5, a convergent series.
The same method obviously generalizes to give en, the expected number of tosses to get n consecutive heads (n≥1): en=12⁢(en+1)+14⁢(en+2)+18⁢(en+3)+116⁢(en+4)+⋯+12n⁢(en+n)+12n⁢(n), the solution of which is easily found to be en=2⁢(2n-1).
