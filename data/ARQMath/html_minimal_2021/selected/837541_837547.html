I'm wondering, is there a lower bound for (x+y)k? For example, if x,y,k>0, can we say that (x+y)k≥xk+yk? If anyone has a source/reference for this, that would be great.

Is k an integer? Then just use (x+y)k=∑n=0k(kn)⁢xk-n⁢yn and the fact that all summands are non-negative.

Assuming that k∈ℕ. (x+y)k=∑i=0k(ki)⁢xi⁢yk-i When x,y>0, all the (ki)⁢xi⁢yk-i>0, so indead (x+y)k≥xk+yk

It is interesting to note that when generalizing to k∈ℝ, it doesn't work for k≤1 (e.g. x=y=2, k=1/2, in which case we get 2 for the left hand side and 2⁢2 for the right hand side). You can generalize your answer to the reals using what you've already done with the integers, and applying the fact (with proof perhaps?) that (x+y)k is continuous and [monotonic](http://en.wikipedia.org/wiki/Monotonic_function) in k.
