The statement in this current form is false, since it states for all n. I think, there should be some restriction on n to make the question interesting at least.
Possible duplicate of [Nonzero f∈C⁢([0,1]) for which ∫01f⁢(x)⁢xn⁢𝑑x=0 for all n](http://math.stackexchange.com/questions/16831/nonzero-f-in-c0-1-for-which-int-01-fxxn-dx-0-for-all-n)

From weierstrass approximation theorem, for any ϵ>0, there exists a polynomial Pϵ⁢(x) such that ∫01|f⁢(x)-Pϵ⁢(x)|⁢𝑑x≤ϵ Now, from the condition of this question, ∫01f⁢(x)⁢Pϵ⁢(x)⁢𝑑x=0 So, (using fact f is continuous on closed interval) ∫01f2⁢(x)⁢𝑑x=∫01f⁢(x)⁢(f⁢(x)-Pϵ⁢(x))⁢𝑑x≤∫01|f⁢(x)|⁢ϵ⁢𝑑x=ϵ⁢∫01|f⁢(x)⁢d⁢x

let ϵ→0, we have ∫01(f⁢(x))2⁢𝑑x=0 since f is continuous, so f(x) = 0 for all 0≤x≤1

Therefore, there exists no such function to satisfy conditions of the question.

QED
