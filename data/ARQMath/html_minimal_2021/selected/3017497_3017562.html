Note that a3+b3+c3 is homogeneous of order 3 while (a+b+c)2 is homogeneous of order 2. So for any a,b,c>0, the statement will be true for (t⁢a,t⁢b,t⁢c) if t is sufficiently large and false if t>0 is sufficiently small. The boundary between the two regions is a certain surface.
