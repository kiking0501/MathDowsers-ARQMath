Constant complex function with real part is greater than 2

I have proved that if Re⁢(f⁢(z))>2 (f is analytic and defined in whole complex plane), then f is constant function: |f⁢(z)|≥Re⁢(f⁢(z))>2⟹|1/f⁢(z)|≤1/2<2

By Liouville's Theorem, if there exist an entire function g such that |g⁢(z)|≤C is bounded (C is a constant), then g is constant. g is the function that should be found. I defined g as follows : g⁢(z)=f⁢(z)+i, assumed that f⁢(z)=x+i⋅0
|g⁢(z)|≥Re⁢(g⁢(z))⟹|g⁢(z)|≥Re⁢(f⁢(z))⟹|g⁢(z)|≥|f⁢(z)|⟹|1/g⁢(z)|≤|1/f⁢(z)| At this point I have some doubts. |1/g⁢(z)|≤2 implies that 1/g is bounded and therefore g is constant.

My doubt is about the following: if |1/f⁢(z)|≤2 then |1/g⁢(z)|≤2 ,since |1/g⁢(z)|≤|1/f⁢(z)|. Is this right?

Yes, you are on the right track. Once we note that |f⁢(z)|≥Re⁢(f⁢(z))>2 for any z∈ℂ, we have that f⁢(z)≠0, and g:=1/f is analytic in ℂ with |g⁢(z)|<1/2 for any z∈ℂ. Then, by Liouville's Theorem, g is constant and therefore also 1/g=f is constant.
