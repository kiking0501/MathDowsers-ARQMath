A complete orthonormal system contained in a dense sub-space.

Let H be a separable complex Hilbert space. Let A be a dense sub-space of H. Is it possible to find a complete orthonormal system for H that is contained in A?

Yes.

The proof is essentially the same as finding an orthonormal basis for H itself. As a subspace of a separable metric space, A has a countable dense subset {xn}. Apply the Gram-Schmidt algorithm to {xn} to get an orthonormal set {en}. By construction, the linear span of {en} contains every xn, so it is dense in A and hence also in H, making {en} a complete orthonormal system.

Edit: I'd be interested in seeing an answer which addresses the non-separable case. I thought about using my argument with a transfinite induction, but it doesn't work. Suppose we have an orthonormal subset of A, {ei}i<α for some ordinal α. Let Eα be the closed span of {ei}i<α; if Eα≠H then we can find some vector x∈A∩Eαc. The natural choice for the next basis vector eα would be eα=x-PEα⁢x, as this will be orthogonal to all {ei}i<α; but as A is not closed, Eα need not be contained in A, so we could get eα∉A and this breaks down.

Of course the problem can happen only when α is a limit ordinal, so this seems to suggest that maybe there is something special about the separable case, and perhaps the statement is false in general. But I couldn't think of a counterexample offhand.
