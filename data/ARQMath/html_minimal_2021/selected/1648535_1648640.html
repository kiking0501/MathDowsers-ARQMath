Optimizing the area of a rectangle with one side against a wall using the am-gm inequality

Given 300 meters of fence, how can I find the dimensions of a rectangle that is built against a wall the encloses the maximum area. I found this question in a calculus book and saw a simple solution in which the derivative function of the area was set to zero. Is it possible to solve this problem using the am-gm inequality? Thanks.

calculus inequality optimization

We need to find x>0,y>0 that maximize x⁢y subject to 2⁢x+y≤300. This is straightforward by AM-GM inequality a⁢b≤a+b2 with a=2⁢x,b=y.
