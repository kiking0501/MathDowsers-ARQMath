@Nusha I believe in this context it's shorthand for 'least common multiple' -- see [the section on LCM in communtative rings, from wiki's entry on LCM](http://en.wikipedia.org/wiki/Least_common_multiple#The_LCM_in_commutative_rings)

1∈H since 1k=1.

Suppose g∈H. That is, gk=1. Then (g-1)k=(gk)-1=1. So g-1∈H.

If x,y∈H, this means xk=1 and yk=1. Then (x⁢y)k=xk⁢yk=1 since G is an abelian group. So x⁢y∈H.

H is a subgroup.
