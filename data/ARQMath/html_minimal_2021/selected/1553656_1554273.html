Is this alternative notion of continuity in metric spaces weaker than, or equivalent to the usual one?

I will try to be as clear as possible.

For simplicity I will assume that the function f for which we define continuity at some point is real function of a real variable f:ℝ→ℝ, although the same line of reasoning should be the same even if we talk about continuity of a function at a point that is defined at some metric space and that has values in some metric space. To define continuity of a function f at some point x0 from its domain we have the following standard and famous definition, the ε-δ definition of continuity, which goes like this:

Definition 1: f is continuous at the point x0 from its domain if for every ε>0 there exists δ>0 such that when |x-x0|<δ then |f⁢(x)-f⁢(x0)|<ε.

It is clear that we could write δ⁢(ε) instead of δ because there really is dependence of δ on ε.

Now, I was thinking about alternative definition of continuity that would go like this:

Definition 2: f is continuous at the point x0 from its domain if there exist two sequences εn and δn such that for every n∈ℕ we have εn>0 and δn>0 and limn→∞⁡εn=limn→∞⁡δn=0 and when |x-x0|<δn then |f⁢(x)-f⁢(x0)|<εn.

We could also write here δn⁢(εn) instead of δn because obviously there is dependence of δn on εn.

It is clear that second definition does not require that for every ε there exist δ⁢(ε) (which includes in itself an uncountable number of choices for ε) but instead requires that for every member of the sequence εn there is some δn (and this includes in itself only countable number of choices because the set {εn:n∈ℕ}) is countable).

It is clearly obvious that definition 1 implies definition 2, and the real question is is the converse true, in other words:

If the function is continuous at some point according to definition 2, is it also continuous at the same point according to definition 1?

The two definitions are equivalent in metric spaces because metric spaces are first countable: every point in a metric space has a countable neighborhood basis. (We'll explain that further below, when discussing Definition 2.)

The standard Definition 1. rephrases the definition from general topology of continuity at a point into a more concrete form that takes advantage of what ℝ has to offer. The general definition, shown in three equivalent forms, is as follows: ∀

Without loss of generality, Definition 1. restricts attention not just to open neighborhoods but to the basic open neighborhoods, the open balls centered at x and f⁢(x). In ℝ, the open balls around u are all the intervals Iu,r=(u-r,u+r),r∈ℝ+. Given x and f⁢(x), these neighborhoods are specified by their radii, so the definition becomes: ∀ε⁢∃δ⁢f⁢(Ix,δ)⊆If⁢(x),ε, which is to say, ∀ε∃δ∀y(y∈Ix,δ→f(y)∈If⁢(x),ε).

But note that v∈Iu,r exactly when |v-u|<r, so we can recast the previous form into the standard definition:

∀ε∃δ∀y(|y-x|<δ→|f(y)-f(x)|<ε).

This form is much friendlier to computation and approximation.

Definition 2. takes advantage of the fact that metric spaces are first countable: for every x in a metric space X, there is a countable collection (Ux,n)n∈ℕ of neighborhoods of x such that any neighborhood V of x contains some Ux,n. Such a collection is called a (countable) neighborhood basis at x. The sequences εn and δn give countable neighborhood bases for f⁢(x) and x respectively:

  • Ux,n=Ix,δn,
  • Uf⁢(x),n=If⁢(x),εn.

This form is even more amenable to computation. Suppose the sequence (εn) is given effectively: there is a procedure which on input n produces a rational εn. For many f we can devise another procedure which on inputs n and (εi)i≤n produces δn. (Think of the expressions which arise in analysis proofs that establish continuity of specific functions, e.g. "take δ=2⁢ε/K".) In general, computable f aren't just continuous at a point, and we may even be able to give a procedure that computes a modulus of continuity for f.

Finally, there's a third possible definition, in a sense intermediate between Definitions 1. and 2. and also equivalent to them:

f is continuous at x iff for every sequence of positive reals (εn)n∈ℕ→0, there is a sequence of positive reals (δn)n∈ℕ such that for all n and for all y, if |y-x|<δn then |f⁢(y)-f⁢(x)|<εn.

In more general topological spaces, which might not be metric spaces or even first countable, continuity can't be completely characterized by countable sequences. However, it can be characterized with the more general notion of a net. For a great self-contained examination of the limits (so to speak) of sequential convergence and the versatility of nets, see Pete Clark's notes on Convergence.
