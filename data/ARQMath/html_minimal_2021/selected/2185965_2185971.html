Meaning of these Closure Abbreviations

What do these closure abbreviations stand for in Mathematical Logic?

Semantic Closure: Con(Γ)

Syntactic Closure: Ded(Γ)

...where Γ is a set of well-formed formulas.

predicate-logic first-order-logic

Con⁢(Γ) is the set of all formulas that are logical Consequence of the set (of formulas) Γ.

Ded⁢(Γ) is the set of all formulas that are derivable from the set (of formulas) Γ, i.e. there is a Deduction of them by way of the proof system.

Okay, so **Con** comes from Logical **Con**sequence. Why is the other one **Ded** instead of **Der** from **Der**ivation?
@A_for_Abacus - **Ded** for Deduction.
So deductions of Γ is the same meaning as derivations of Γ?
@A_for_Abacus - Yes; see e.g [Natural Deduction](https://en.wikipedia.org/wiki/Natural_deduction) and [Derivation](https://en.wikipedia.org/wiki/Natural_deduction#Hypothetical_derivations).
