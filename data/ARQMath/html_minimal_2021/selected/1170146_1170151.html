Why can the hyperplanes in ℝn+1 can be represented by affine functions ℝn→ℝ?

Intuitively, it is true. But still, it is not clear for me.

An n-dimensional hyperplane in ℝn+1 defined by (b,β)∈ℝn+1 can be represented as {x∈ℝn|⟨x,b⟩=β}. I also know that the hyperplane can also be represented as an affine function ℝn∋x↦f⁢(x)=⟨b,x⟩-β. Can someone help me deduce the first expression from the second?

Thank you in advance

linear-algebra functional-analysis

We can represent a hyperplane Π∈ℝn+1 via a pair (b,β)∈(ℝn+1-{0})×ℝ. Explicitly, Π is the set {x∈ℝn+1:⟨b,x⟩=β}.

On the other hand, this is precisely the zero set of the affine function f:ℝn+1→ℝ defined by x↦⟨b,x⟩-β.

Conversely, any affine function f:ℝn+1→ℝ can be written as x↦b1⁢x1+⋯⁢bn+1⁢xn+1-β for (unique) coefficients b1,…,bn+1,β. If f is nonconstant, then b:=(b1,…⁢bn) is nonzero, and x↦⟨b,x⟩-β defines a hyperplane.
