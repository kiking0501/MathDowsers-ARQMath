First of all, I assume that you mean the logarithm to the base 2, i.e. that your problem is to calculate: log2⁡(44). Let's recall a rule for taking roots: Suppose x>0, and that n is an integer. Then it is true that xn=x1/n. I want to use this in your problem for the expression inside the logarithm. We get that 44=41/4. Next, let's recall a rule for using logarithms. If log is any logarithm, x>0, and a>0, then log⁡(xa)=a⁢log⁡(x), i.e. I can take the exponent from inside of the logarithm and pull it out. Using these two rules, I can rewrite your problem as log2⁡(44)=log2⁡(41/4)=14⁢log2⁡(4). It remains to do something about the expression log2⁡(4), but we can readily see what this is: log2⁡(4) is equal to the number x, such that 2x=4, which is just x=2. Thus log2⁡(4)=2, and so log2⁡(44)=14⁢log2⁡(4)=14⋅2=12.
