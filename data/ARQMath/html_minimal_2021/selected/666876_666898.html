It is not true that if A⁢B=0 and A≠0 then B=0. You need to make use of the fact that A is invertible.
If A is invertible and A⁢B=0, you use the fact that invertibility allows you to multiply by A-1, ie A⁢B=0⟹A-1⁢A⁢B=I⁢B=B=A-1⁢0=0
