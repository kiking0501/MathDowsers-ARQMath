Exponents in Mathematical Induction

Prove that 52⁢n-1 is a multiple of 8 for all n ∈ ℕ.

Proof:

Using the Principle of Mathematical Induction:

Let n=1. If n=1, then 52-1= 25-1= 24.

Since 24 is divisible by 8, the statement is true for n=1.

Assume the statement is true for n=k where k ∈ ℕ.

Then the statement 52⁢k-1 is a multiple of 8 is true. That is 52⁢k-1=8⁢m for some m ∈ ℕ.

We must prove that the statement is true for n=k+1. That is, we must prove that 52⁢(k+1)-1 is a multiple of 8.

I'll annotate the second solution, so hopefully it will make more sense.

52⁢(k+1)-1=52⁢k+2-1

The above step just expands 2⁢(k+1)=2⁢k+2 in the exponent.

=52⁢k+2-52⁢k+52⁢k-1

This adds the value 52⁢k and subtracts 52⁢k. Knowing to do this isn't obvious, but you should see that it is equivalent to the expression above.

=52⁢k⁢(25-1)+(52⁢k-1)

This step did several things at once. First, it splits 52⁢k+2=52⁢k×25 so that we have =52⁢k⁢(25)-52⁢k+52⁢k-1. Then we can factor out a 52⁢k from 52⁢k×25 and 52⁢k×(-1) to get 52⁢k⁢(25-1). Again, it isn't obvious that this is the correct thing to do (how did we chose which of the positive or negative 52⁢k to factor out?), but you should see again that it is equivalent to the statement above. After that, they just add parentheses around 52⁢k-1 to highlight that it is a pair (important for inductive step).

=52⁢k⁢(24)+8⁢m

Here, we simplify the (25-1)=24 (this is divisible by 8!) and we use the fact that 52⁢k-1=8⁢m from our inductive statement, for some integer m.

\therefore 52⁢(k+1)-1 ⇒ 8⁢(52⁢k⁢(3)+m)

Now, we factor out the 8. We know 52⁢k×3+m is an integer, so we are done.

By factor, I think you mean splitting up an expression into products. If so, then 52⁢k+2-1=52⁢k+2-52⁢k+52⁢k-1 was not factored, since there is no product - only sums. They simply added 52⁢k and subtracted 52⁢k. This technique appears in some proofs sometimes (adding and subtracting by the same number) and it's not called factoring. Hope that clears it up.
