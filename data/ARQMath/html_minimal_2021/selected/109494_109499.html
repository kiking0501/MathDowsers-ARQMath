Set Theory: Symmetric Relations

I was just trying to figure out this problem I came across. For a set X={1,2,3,4,5} is it possible to come up with a relation on X that is symmetric, but neither reflexive nor transitive?

Also, it is possible to find a relation that is transitive, but neither reflexive nor symmetric?

Thank you!

For the second part, there is a natural example: The "strictly larger than"-ordering on X.

For the relation R to be not transitive, let (1,2)∈R and (2,3)∈R but (1,3)∉R. If it is to be symmetric, then (2,1)∈R and (3,2)∈R. A check shows that R={(1,2),(2,3),(2,1),(3,2)} satisfies the first part.

For the relation to be not symmetric, let (1,2)∈R but ∥sm-sn∥<ε. A check shows that R={(1,2)} satisfies the second part.
