∥A†∥2-1=min∥x∥2=1⁡∥A⁢x∥2=min∥x∥2=1⁡∥[A1⁢xA2⁢x]∥2≥min∥x∥2=1⁡∥A1⁢x∥2=∥A1-1∥2-1

Start with the definition: ∥A†∥2=maxx≠0⁡∥A†⁢x∥2∥x∥2.

Any such x can be written in the form x=y+z, where y∈ℛ⁢(A) and z∈ℛ⁢(A)⟂=𝒩⁢(A*)=𝒩⁢(A†). We have that A†⁢x=A†⁢y+A†⁢z=A†⁢y and since y⟂z, ∥x∥22=∥y∥22+∥z∥22. The maximum is therefore attained when z=0. We can write y=A⁢w for some w. All this given then ∥A†∥2=maxw≠0⁡∥A†⁢A⁢w∥2∥A⁢w∥2. Since the rank of A is n, A†⁢A=I, and ∥A†∥2=maxw≠0⁡∥w∥2∥A⁢w∥2=(minw≠0⁡∥A⁢w∥2∥w∥2)-1.
