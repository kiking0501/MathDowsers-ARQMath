If F/K is an algebraic field extension, is every endomorphism of F/K an automorphism?

If F/K is a field extension of finite degree, then every endomorphism of F/K is an automorphism. Does the result also hold for algebraic extensions?

As Georges Elencwajg said, a K-endomorphism of an algebraic extension F is indeed an isomorphism. Here is another argument.

Let f:F→F be your K-endomorphism. First, note that it is into, because its kernel is an ideal of F, and since it is not F, it is {0}. We want to prove it is onto, so let a∈F. Because a is algebraic over K, it is a root of a polynomial P. Let FP be the set of roots of P in F. Note that this is a finite set (P has only a finite number of roots in F), and that f⁢(FP)⊂FP (f maps a root of P to a root of P

So f induces a injective map FP→FP and FP is finite, so it is also onto. Hence, there is b∈F such that f⁢(b)=a.
