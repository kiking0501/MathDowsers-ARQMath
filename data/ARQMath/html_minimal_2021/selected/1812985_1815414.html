@timur: I'm not sure if you got the point, but formulae are simply strings of symbols. They are utterly devoid of meaning until you interpret them. Now in set theory the intended interpretation is that every object is a set, and so formulae like "x‚àày" always make sense regardless of what x,y are. The axiom of specification is a **syntactic** rule that says that you can use any existing object S and any formula œÜ with no free variables except x to construct a new object denoted by {x:x‚ààS‚àßœÜ}. Only in our intended interpretation is it a subset of S.
@timur: Note that although functions in set theory are defined to be sets of a certain kind, not all function-like things are functions. For example the powerset operation is **not** a function because if it is then its domain would have to be the set of everything, but existence of such a set allows Russell's paradox leading to contradiction.
