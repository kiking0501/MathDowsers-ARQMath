put x=1/y so that y→∞ and then note that the given function f⁢(x)=e-1/x/x2=y2/ey. Then use the fact that ey=1+y+y22!+y33!+⋯>y36 for all y>0. Then we get 0<f⁢(x)<6/y and hence by Squeeze theorem f⁢(x)→0 as y→∞.

Don't use L'Hospital's rule. It won't work here, and when it works, it is equivalent with Taylor's polynomial at order 1, which is much less error-prone.
