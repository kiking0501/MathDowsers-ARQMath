calculus sequences-and-series matrices

Since the space E:=ℳn⁢(ℂ) of all n×n complex matrices is a finite-dimensional vector space, all norms define the same topology. So we can take a sub-multiplicative norm, that is, a norm ∥⋅∥ such that ∥A⁢B∥≤∥A∥⋅∥B∥. (For example, we can take ∥⋅∥ to be the operator norm on E.) As a finite dimensional vector space, E is complete, so it's enough to show normal convergence. We have that, for each integer n≥0, 0≤∥Ann!∥≤∥A∥nn!, and we know that, for each real number x, the series ∑n=0+∞xnn! converges (it defines the exponential function). Therefore, for any A∈E, the series ∑n=0+∞Ann! converges. (We also got the additional result that ∥eA∥≤e∥A∥ for any A∈E.)
