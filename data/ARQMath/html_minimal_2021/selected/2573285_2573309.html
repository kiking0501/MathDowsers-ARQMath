∥D∥=max⁡|Di⁢i| for any n×n diagonal matrix D

Suppose that D=(Di,j) is an n×n diagonal matrix. Is it true for any natural norm ∥⋅∥ that ∥D∥=max1≤i≤n⁡|Di⁢i|?

I know that this is true for the 2-norm, but I do not think this is true in general. Does anyone know a counterexample?

If natural norm means a matrix norm induced by a vector norm, then it is generally not true.

For any induced matrix norm and a diagonal D, we have ∥D∥=maxx≠0⁡∥D⁢x∥∥x∥≥max1≤i≤n⁡∥D⁢ei∥∥ei∥=max1≤i≤n⁡|Di⁢i| obtained by picking x to be columns of the identity matrix. So the maximum of |Di⁢i| is at least a lower bound on ∥D∥.

There are cases, where the inequality becomes an equality. This is true for any matrix p-norm with 1≤p≤∞.

In order to construct a counterexample, consider a vector norm ∥x∥M:=∥M⁢x∥∞ with a nonsingular matrix M. The induced matrix norm is related to the ∞-norm by ∥D∥M=∥M⁢D⁢M-1∥∞. For example, in ℝ2 and with D=[1a⁢m⁢p;00a⁢m⁢p;2],M=[2a⁢m⁢p;11a⁢m⁢p;2], we get ∥D∥M=3>2=max1≤i≤2⁡|Di⁢i|.

That seems to be the correct interpretation of [natural norm](http://mathworld.wolfram.com/NaturalNorm.html).
