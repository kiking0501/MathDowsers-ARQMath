Hint to a different answer than @user2373145's: Do you know how to compute the determinant of the same matrix but with an x instead of the x2 ? Now reduce your problem to this simpler one, by Laplace expanding along the last row.

You add up all the rows to the first one giving x + n-1 for all but the n-th element, which is x² + n-1.

Then you factor out the x + n-1 (with the condition that it isn't zero, prove the trivial case separately; the general solution also holds for x=-n+1), so the first row becomes all ones except for the n-th entry.

Then you subtract the first row from all the others getting an upper triangular matrix with a (1, x-1, ..., x-1, x²-(x²+n-1)/(x+n-1)) main diagonal.

Now the determinant of a triangular matrix is the product of the elements in its main diagonal.

So we have detAₙ = (x+n-1) (x-1)^(n-2) (x²-(x²+n-1)/(x+n-1)) = (x−1)^(n−1) (x²+(n−1)x+n−1).

You could treat this calculation as being over the field ℝ⁢(x), so x+n-1 is definitively non-zero :)

Sure... first of all I'm just saying there are other ways to justify the same calculation that don't require explicitly checking x=-n+1 separately. One is to interpret the elements of the matrix as being elements of the field of rational functions ℝ⁢(x), so that the element x+n-1 is a polynomial of degree 1, not the zero element. Note also that the LHS is necessarily a polynomial in x, and so is the RHS, and they agree at infinitely many values of x, so they must agree for all x.
