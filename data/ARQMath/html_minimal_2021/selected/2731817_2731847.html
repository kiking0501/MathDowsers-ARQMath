I'm not sure I follow your argument exactly but it does seem that you are not using the proper definition of countability. A set S is countable if there is an 𝑖𝑛𝑗𝑒𝑐𝑡𝑖𝑜𝑛 S→ℕ, not a 𝑏𝑖𝑗𝑒𝑐𝑡𝑖𝑜𝑛. This is why, for example, a finite set is countable. It's also unclear to me that you aimed to prove the original claim; it seems you only proved it for two sets. Furthermore, because of this you wrote all of your functions going 𝑓𝑟𝑜𝑚 ℕ rather than 𝑡𝑜 it, which seems more natural.

The way I would prove it is as follows: prove it for two sets, then use induction. Call the two sets A and B. We can assume without loss of generality that they are disjoint (if not, replace them by A and B∖A). Since they are both countable, we can write A={a1,a2,…} B={b1,b2,…} Note that if one or both of A,B is finite, the labeling above will eventually stop i.e. you can write them all down. Clearly if A,B are both finite their union is finite, hence countable. I claim that if one is countable and the other is finite, their union is countable. See if you can prove this.

The trickier case is when both are countable. You seem to have the write idea about even and odd indices. Given the labeling above, define a function f:A∪B→ℕ by f⁢(ak)=2⁢k-1 f⁢(bk)=2⁢k This is clearly injective, which proves that A∪B is countable. Note that injectivity relies on the assumption that A,B were disjoint, which is why I made that assumption above.

The inductive hypothesis is simple: if the claim holds for m-1 sets then it must hold for m sets because A1∪…∪Am=(A1∪…⁢Am-1)∪Am

In my textbook "Understanding Analysis" by Stephen Abbott, it defines countability by f:ℕ→A instead of the other way around. Also, I proved it for two sets because if A1∪A2=B is countable, then the B∪A3 is simply another union of two sets, and so on.
