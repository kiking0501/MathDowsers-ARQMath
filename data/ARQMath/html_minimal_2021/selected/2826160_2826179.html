x↦logn⁡(x) and x↦nx are each other's inverses, so if we're looking for an n where they just touch each other, we're looking for an n where one of them has the line x=y as a tangent.

Set α=ln⁡n; then one of the graphs is f⁢(x)=eα⁢x. The tangent condition is that for some x0 we have f⁢(x0)=x0  and  f′⁢(x0)=1 or, concretely eα⁢x0=x0  and  α⁢eα⁢x0=1 Combining these, we find α⁢x0=1, so x0=1/α, which we can plug back into the second equation to get α⁢e1=1 which implies α=e-1 and thus n=eα=ee-1.
