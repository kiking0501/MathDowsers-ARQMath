Start with the definition. The Lebesgue integral of a simple function s=∑j=1nαj⁢χAj is:

∫Es⁢𝑑μ=∑j=1nαi⁢μ⁢(E∩Aj)

If μ⁢(E)=0, then μ⁢(E∩Aj)=0 for all j. Thus ∫Es⁢𝑑μ=0.

The Lebesgue integral of a nonnegative function f is the supremum of integrals of all simple functions s such that 0≤s≤f. Since all of these integrals are 0, the supremum is 0 too.

Since every real function f can be written as f=f+-f- where f+ and f- are both nonnegative, we have ∫Ef⁢𝑑μ=0 too. The general result follows from the fact that every complex function f can be written as f=u+i⁢v where u and v are real.
