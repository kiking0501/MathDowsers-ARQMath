Usually the following is taken as the definition of an odd number, so I'm not sure if this qualifies as a 'hint', but here goes: an odd number is one of the form 2⁢k+1 for some integer k.

The division algorithm says that for any m and positive n in the integers, there are integers q and 0≤r⁢\lt⁢n so that m=q⁢n+r For n=2, there are two remainders (0≤r⁢\lt⁢2): 0 and 1.

m is odd if it is not divisible by 2 (the remainder when dividing by 2 is not 0). Since there is only one non-zero remainder, the remainder when dividing m by 2 must be 1.
