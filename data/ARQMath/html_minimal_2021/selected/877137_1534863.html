Strong duality: When does the optimal primal variable coincide with the primal variable giving the dual function.

I'm considering the inequality-constrained optimization problem of finding x⋆=arg⁡minx⁡f⁢(x)⁢s.t.⁢h⁢(x)≤0 which is assumed to have a unique minimizer. The objective f maps Rn→R and h maps Rn→Rm capturing multiple constraints. This problem has Lagrangian L⁢(x,λ)=f⁢(x)+λT⁢h⁢(x) and the associated dual problem is to find some λ⋆∈arg⁡maxλ⁡g⁢(λ)⁢s.t.⁢λ≥0 where the dual function is defined g⁢(λ)=minx⁡L⁢(x,λ).

Assuming that strong duality holds, the optimal value of the primal problem p⋆=f⁢(x⋆) and that of the dual problem d⋆=g⁢(λ⋆) are equal.

QUESTION: If we define x^⁢(λ)∈arg⁡minx⁡L⁢(x,λ) to be a value of the primal variable x which yields the dual function g⁢(λ)=L⁢(x^⁢(λ),λ), then do there exist functions (f,h) for which this is not necessarily equal to the solution, i.e. x^⁢(λ⋆)≠x⋆?

Strong duality does not guarantee that the optimal values are *attained*. That is: it's entirely possible that infx:h⁢(x)≤0⁡f⁢(x)=supλ≥0⁡infx⁡L⁢(x,λ)but there is no x* and/or λ* that achieves these optima. Strict primal feasibility implies dual attainment, and vice versa.
@MichaelGrant: The OP assumed that f has a unique minimiser, so the primal optimal is attained. Presumably by the min in the definition of g this means that the dual optimal value is also attained. So there is a saddle point, we have g⁢(λ*)=L⁢(x*,λ*)=f⁢(x*). However, while I suspect that it is possible that x^⁢(λ*)≠x*, it is at best a hunch.

Can I add the assumption that both the primal and dual optimal values are attained?

This is entirely possible, and happens all the time. In order to guarantee that the subproblem solution obtained at the dual optimum corresponds to the true primal optimum, you need the dual to be differentiable which is equivalent to the primal problem being strictly convex. In the context of linear programming, this is called the "problem of non-coordinability".

It has been a fairly active area of research actually, see for example http://link.springer.com/article/10.1007%2Fs10107-014-0772-2#/page-1 for a decent exposition.
