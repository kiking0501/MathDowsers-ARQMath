If you have f′⁢(xi) for a set of points {xi|i=0,1,2,⋯⁢n} and the value f⁢(x0) you can estimate the value of the function at the same points using f⁢(xi)=f⁢(xi-1)+f′⁢(xi)⁢(xi-xi-1).

The approximation is good if the intervals xi-xi-1 are small. Possibly, interpolation methods can be used as suggested here.
