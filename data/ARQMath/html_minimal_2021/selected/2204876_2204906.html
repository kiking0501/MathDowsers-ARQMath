Axiom of Foundation implies that V=V∞

Here, V denotes the class of all sets, and V∞ denotes the von Neumann universe. I've heard the claim a few times that Foundation implies that every set x can be found in Vα for some α, but I can't find a proof of this. Does anyone know of a proof?

Have you looked in set theory books or lecture notes? This is one of the important theorems of basic axiomatic set theory.

It suffices to show that every transitive set belongs to V∞, indeed if we do prove this, let x be any set, then c⁢l⁢(x)⊂Vα for some α, and then since x⊂c⁢l⁢(x), one gets x∈Vα+1 (where c⁢l⁢(x) denotes the transitive closure of x) .

So let y be a transitive set and assume y∉V∞.

Then one has that y is not a subset of V∞ indeed otherwise the application that to x∈y sends the least α such that x∈Vα would be well defined, and thus by the axiom scheme of replacement would have bounded range, and thus y∈Vα+1 where α is an upper bound of this application.

Thus the set z={x∈y∣x∉V∞} is non empty, and thus by the axiom of foundation there is x∈z such that x∩z=∅. But x∈z so x∈y so by transitivity of y, x⊂y, and so since z∩x=∅, x⊂y∖z, and thus x⊂V∞. But we've seen that this implies x∈V∞, so x∉z : a contradiction. Therefore z is empty, and y⊂V∞, and so y∈V∞.

This concludes the proof.

A bit of thought will show you take the converse is true as well, i.e. if V=V∞ then the axiom of foundation holds
