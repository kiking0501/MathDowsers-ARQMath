Working on Euler 546?
[[a/b]/c] = [(a/b - r/b)/c] = a/bc - r/bc - q/c. where a = mb + r and (a/bc - r/bc) = nc + q. [[a/c]/b] = a/bc - t/bc - u/b where a = dc + t and (a/bc - t/bc) = eb + u. Do r + q/c always equal t + u/b? (same would apply for ceiling.)

Using division with remainder write a=q⁢b⁢c+r with 0≤r<b⁢c and then write r=q′⁢b+r′ with 0≤r′<b, as well as r=q′′⁢c+r′′ with 0≤r′′<c. So a=q⁢b⁢c+q′⁢b+r′=q⁢b⁢c+q′′⁢c+r′′. Because 0≤r<b⁢c we conclude that 0≤q′<c and 0≤q′′<b. Then ⌊⌊ab⌋c⌋=⌊q⁢c+q′c⌋=q and ⌊⌊ac⌋b⌋=⌊q⁢b+q′′c⌋=q. So indeed ⌊⌊ac⌋b⌋=⌊⌊ab⌋c⌋=⌊ab⁢c⌋.
