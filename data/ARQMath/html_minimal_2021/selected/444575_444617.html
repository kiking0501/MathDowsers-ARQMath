If f:ℝn→ℝn is continuous with convex image, and locally 1-1, must it be globally 1-1?

For f:ℝ→ℝ which is continuous, being locally 1-1 implies being globally 1-1, see here. This is not true for a general mapping f:ℝn→ℝn. My intuition as to the source of this incongruity is that while continuity preserves connectedness, it does not preserve convexity.

This illustration (from Buck's Advanced Calculus) of a locally injective, not globally injective function from ℝ2 to ℝ2 somewhat supports my intuition...note the image is not convex. If it were convex, it would seem there would need to be some "critical point" (not assuming the existence of the derivative) where f was not locally 1-1.

Is there a theorem that if f:S→ℝn is continuous, for S⊂ℝn convex, with convex image, locally 1-1, then it must be 1-1 on all of S?

I think it may be more promising to consider functions that preserve convexity (or perhaps simple connectedness), so that the image of *every* convex[/simply connected] set is convex[/simply connected].

One can modify the pictured example so as to have domain, say, the set of (x,y)∈ℝ2 with 0<x<1. Imagine the function being given by taking a line segment of width 1, mapping the center of the line segment to f⁢(12,y), and then carrying around the rest of the line segment linearly. Thus we need a continuous function g:ℝ→ℝ2 so as to set f⁢(12,y)=g⁢(y). If g is smooth with nowhere vanishing derivative, f is locally injective. It is easy to choose such a g so as to make f surjective -- e.g. make sure that every once in a while g traces out the circle of radius n2 centered at the origin for each n∈ℤ+. -- and not injective -- each make sure g does what I said above at least twice for each positive integer n.

More precisely: I want the segment to be oriented so that for all y∈ℝ, it is perpendicular to the tangent vector g′⁢(y). Once I specify its orientation is at y=0 (take it to be horizontal; it doesn't matter) that determines it for all y.

Since the domain of f is diffeomorphic to ℝ2, we can pull this back to get a (smooth) counterexample f:ℝ2→ℝ2.

Added: The above construction can also be used to give an f such that for all v∈ℝ2, f-1⁢(v) is countably infinite. Countably infinite is the largest possible preimage for a locally injective function on ℝ2 (or any σ-compact T1 topological space): cover ℝ2 by countably many compact subsets. If any preimage were uncountable, its intersection with some compact subset would be uncountable, and thus the preimage of f-1⁢(v) would have an accumulation point, at which f is not locally injective.
