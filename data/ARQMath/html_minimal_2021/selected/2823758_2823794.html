The idea is that, given any nbhd V of f⁢(x), every point that is "sufficiently close" to x will map into V. It may be that some points very distant from x will also map into V but that is not relevant. We examine every nbhd V of f⁢(x) to see whether some nbhd UV of x will map into V. ...Also: For metric spaces (X,d) and (Y,e), a function f:X→Y is continuous iff limn→∞⁡e⁢(f⁢(xn),f⁢(x))=0 whenever (xn)n is a sequence in X with limn→∞d(xn.x)=0.

The "normal" definition goes like this:

It is claimed that, at fixed point, for any given ball Bϵ of radius ϵ in the image, there exists a ball Bδ, in the preimage, of radius δ such that I⁢m⁢(Bδ)⊂Bϵ. This is the implication (…)<δ⟹(…)<ϵ

Very informally, you could compare the statement, for continuous f,

For any ball Bϵ in the image, you can find a ball Bδ mapping into Bϵ

and

For any ball Bϵ in the image, its preimage contains a ball Bδ

and

The preimages of open sets are open.

In topological spaces, the last one is often taken as a definition.

Regarding your interpretation

IF U⊆Y is open THEN f−⁢1⁢(U) is open

This is perfectly valid and translates as "IF you give me an ϵ THEN I can find you a corresponding δ".

Regarding the implication, let me explain in this way, to show what happens with that implication:

Let U⊂Y be open, then for this set you can have its preimage, f-1⁢(U)⊂X, which is the set that satisfies: x∈f-1⁢(U)⟹f⁢(x)∈U So now you can freely say:

For any open U⊂Y, there is a set f-1⁢(U)⊂X.

If is just so happens, that f-1⁢(U) is open for any open U, then we call f continuous. Translating, this means that if it just so happens that for any given radius ϵ, can find a corresponding δ such that x∈Bδ⟹f⁢(x)∈Bϵ, then f is continuous.

A few more details:

You have be rather careful when you state exactly what you mean with mapping "nearby points to nearby points".

Given a metric, we can always have balls as subsets of that space. The open sets are precisely those that, for each x, have some ball around them completely contained in the open set. This is true regardless of whether the open set is a union of open intervals, the whole space, a single interval, or any other open set.

To say that f maps "nearby points to nearby points" means to say that, if you fix a point x0, and look at what happens to points nearby x0, they will all be mapped to points close to f⁢(x0). The exact meaning of this is that: for each fixed x∈f-1⁢(U), for any ball Bϵ around f⁢(x) (and one exists, and satisies Bϵ⊂U, by openness), there is a ball Bδ around the point x that maps into Bϵ. Since Bϵ⊂U, we have Bδ⊂f-1⁢(U), which by definition makes the preimage open. It's a ball around an arbitrary point completely in f-1⁢(U).

Whatever open set you have, all of the points in there will be interior, so continuity (finding matching balls Bδ and Bϵ) works at each point at a time, so to speak. And now it almost rolls off the tongue: ∀x⁢∀ϵ⁢∃δ⁢(…)

To me, it is somehow intuitively clear that if you want a statement about how some values of f⁢(x) behave, you would start with something about its target set. Maybe that's just me. You sort of start with the question "How close to f⁢(x0) do you want the outputs of f to be", which is a question about the target set.
