It's easy to prove this without using induction: (∑i=1nxi)∙(∑i=1n1xi)=n⁢(n+1)2⁢Hn≥n2⁢ln⁡(n)/2, which is greater than or equal to n2 if ln⁡n≥2, that is if n≥8. Check it for small numbers by hand and you're done.

Suppose the given inequality holds for an integer n(≥1).

Then for n+1,

(∑i=1n+1xi)⁢(∑i=1n+11xi)=(∑i=1nxi+xn+1)⁢(∑i=1n1xi+1xn+1)

Expanding this gives us =(∑i=1nxi)⁢(∑i=1n1xi)+xn+1⁢∑i=1n1xi+1xn+1⁢∑i=1nxi+1

Since the given inequality holds for the integer n, the first term is larger than n2. So,

≥n2+xn+1⁢∑i=1n1xi+1xn+1⁢∑i=1nxi+1

Now we show that xn+1⁢∑i=1n1xi+1xn+1⁢∑i=1nxi is larger than 2⁢n.

This is where we use the hint. Expanding the ∑ gives us xn+1⁢(1x1+1x2+⋯+1xn)+1xn+1⁢(x1+x2+⋯+xn)

Rearranging the terms, we get

(xn+1x1+x1xn+1)+(xn+1x2+x2xn+1)+⋯+(xn+1xn+xnxn+1)

and using the hint, this is larger than 2⁢n

Thus,

(∑i=1n+1xi)⁢(∑i=1n+11xi)≥n2+2⁢n+1=(n+1)2
