functions elementary-set-theory proof-writing

First of all, I think we need to get some definitions straight. If f:X→Y is a function (with X and Y sets), then f is said to be invertible if there exists a function g:Y→X such that g∘f=idX and f∘g=idY. If the first identity is satisfied, f is said to be left invertible, and if the other one is satisfied f is right invertible.

I have never heard of left invertible functions being referred to as invertible, and there are easy counterexamples: let X={1},Y={1,2} and define f:X→Y by f⁢(1)=1. Then the function g:Y→X defined by g⁢(1)=g⁢(2)=1 is a left inverse of f, since g⁢(f⁢(1))=1. But f⁢(g⁢(2))=1≠2, so g is not a right inverse (and it is easy to see that no such function can exist, since the range of f is a one point set, while Y is a two-point set, so any left inverse must necessarily identify two points). Also note that g is a right invertible function without a left inverse for the same reasons.

You can, however (as you appear to say), make a left invertible function invertible by restricting the range. In other words, if f:X→Y is left invertible with left inverse g, then writing Z=f⁢(X) (i.e. Z is the range or image of f),we have that f:X→Z is invertible. This is easy to show; we already have left invertibility, so all we need to show is that f∘g|Z=idZ. Consider z∈Z. Then z=f⁢(x) for some (in fact, a unique) x∈X, so we have f∘g|Z⁢(z)=(f∘g)⁢(f⁢(x))=f⁢(g∘f⁢(x))=f⁢(x)=z, since g∘f=idX (left invertibility). Hence f:X→Z is invertible.

For the other question, let f:X→Y again be an arbitrary function, and suppose there exists an 0⁢\lt⁢A⁢\lt⁢π2 such that f∘h=idY (I'm unsure if you mean codomain or range in this case, so I'll do both). Here we get an inverse by restricting f to h⁢(Y); the proof is the same as above - we may as well consider the problem of making a given left inverse of h, namely f, into a right inverse by restricting the range of h. You can do something similar if we only have f∘h=idf⁢(X), first by restricting the codomain of f to the range of f, and proceeding as above.
