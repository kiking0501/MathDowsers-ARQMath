Good job: up to now you checked that it is a basis. Now you have to check that it generates the discrete topology.

This shows that your set is a basis for some topology, not that it generates the discrete topology. However, any subset A⊆X may be written as A=⋃a∈A{a}, which shows that A is open in that topology.

Alternatively, you could have simply used Lemma 13.2 in Munkres, which states:

Let X be a topological space. Suppose 𝒳 is a collection of open sets of X such that for each open set A of X and each x in A, there is an element C of 𝒳 such that x∈C⊆A. Then 𝒳 is a basis for the topology of X.

Here, of course, each element of 𝒳 is open, and any (automatically open) subset A⊆X, and element x∈A, satisfy x∈{x}⊆A.
