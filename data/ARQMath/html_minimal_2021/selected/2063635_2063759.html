Help with limn→∞⁡nkan , a>0

limn→∞⁡nkan, a>0. This problem from Demidovich #60.

real-analysis sequences-and-series limits
a>0? or a>1?
The task is specified for a> 0

Let m:=|⌈k⌉| (the absolute value of the ceiling of k). Then by the binomial theorem we have, for each n⩾2⁢m, an so 2m+1⁢(m+1)!n⩾nman⁢(a-1)m. If a>1 then from (1) we get 2m+1⁢(m+1)!n⩾nman⁢(a-1)m⩾nkan⁢(a-1)m so 2m+1⁢(m+1)!(a-1)m⋅1n⩾nkan which implies that nkan→0.

If a=1 then we have to look at cases on k. If 0<k then nk/an=nk→∞. If k=0 the nk/an=1 and if k<0 then nk/an=nk→0. If 0<a<1 then we can use the result obtained above.
