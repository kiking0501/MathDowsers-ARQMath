limsup-and-liminf

I will show that lim infn→∞⁡𝟏Ak⁢(x)=𝟏lim infn→∞⁡An⁢(x),(∗) where 𝟏A denotes the characteristic function of the set A and lim infn→∞⁡An=⋃n≥1⋂k≥nAk denotes the limit inferior of the sets A1, A2, ... . We have x∈⋃n≥1⋂k≥nAk if and only if for some n≥1 we have x∈⋂k≥nAk, hence for some n≥1 we should have x∈Ak for all k≥n. This is to say that x is contained in all Ak from some point onwards. Thus the right hand side of (∗) is 1 if and only if x is contained in all but finitely many of the An. The left hand side is the inferior limit of a sequence of 0s and 1s, so it takes the value 1 if and only if the value 0 occurs finitely many times (otherwise 0 is a limit of a subsequence). Therefore, the left hand side is also 1 if and only if all but finitely many of the An contain x.

In the same spirit, we have lim supn→∞𝟏Ak(x)=𝟏lim supn→∞⁡An(x),(∗∗) where lim supn→∞⁡An=⋂n≥1⋃k≥nAk. Both the left hand side and the right hand side of (∗∗) are 1 if and only if x is contained in infinitely many of the An.
