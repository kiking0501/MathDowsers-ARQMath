Proof 1: Factorising.

Let n be even. Then n=2⁢k for some k∈ℕ. Hence 2n-1=22⁢k-1=(2k-1)⁢(2k+1). The edge case is n=4 (as this is the smallest even integer greater than 2), but clearly this is not a problem as 22-1=3, so 2n-1 will always be composite for the specified ns.

Proof 2: Induction to show that 22⁢k-1 is divisible by 3 for all k>1.

Base case is trivial.

Let m∈ℕ, and suppose 3 divides 22⁢m-1. Then 22⁢(m+1)-1=22⁢m+2-1=4⋅22⁢m-1=3⋅22⁢m+22⁢m-1. Hence 3 divides 22⁢(m+1)-1, as required.
