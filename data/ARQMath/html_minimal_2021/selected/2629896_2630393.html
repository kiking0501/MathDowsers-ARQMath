You can write your formula like: (2⁢n+1n+1)=∑k=0n(n+kn) Let A be set , |A|=2⁢n+1,and we can interpret left side like all subsets B⊂A, |B|=n+1. We can easily note that in every B greatest element is at least n+1.Now family of all subsets B we can divide into n+1 parts.
first part: all B subsets in which n+1 is greatest element
second part:all B subsets in which n+2 is greatest element.
....
(n+1)-th part:all B subsets in which 2⁢n+1 is greatest element Note that all parts are disjoint.
→Cardinal number of first part is (nn) because we can only use elements smaller than n+1 →Cardinal number of second part is (n+1n) because we can only use elements smaller than n+2
...
→Cardinal number of (n+1)-th part is (2⁢nn) because we can only use elements smaller than 2⁢n+1
Thus,right side is ∑k=0n(n+kn) We conclude: (2⁢n+1n+1)=∑k=0n(n+kn)
