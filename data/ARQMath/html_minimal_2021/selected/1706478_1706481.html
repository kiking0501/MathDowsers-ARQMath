Intuitively, if G′ is isomorphic to G, then the two are 'identical twins' and each one can be substituted for the other; this means that the set of isomorphisms G↦G′ is exactly the same as the set of isomorphisms G↦G. This can be formalized by composing the maps appropriately, as in carmichael's answer.
Do note that Aut⁡(G) very rarely has order |G|! for finite groups.
