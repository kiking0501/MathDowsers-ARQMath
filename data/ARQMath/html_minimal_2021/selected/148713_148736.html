This example could clarify things a little bit.

Consider (xn) where xn=1n. To prove that this is a Cauchy Sequence, we approach as follows:

Let ϵ>0 be given. We are interested in showing that there exists an integer N>0 such that m,n<N⇒|xm-xn|<ϵ. This would be equivalent to

|xn-xm|<ϵ⇔|1n-1m|<ϵ

And now since

|1n-1m|<1n+1m

If we make 1n+1m<ϵ, the result should follow. This will happen if both 1n<ϵ2, and 1m<ϵ2, i.e. when

n>2ϵ, and m>2ϵ. So, we see that if N is an integer larger than 2ϵ then m,n>N⇒|xm-xn|<ϵ
