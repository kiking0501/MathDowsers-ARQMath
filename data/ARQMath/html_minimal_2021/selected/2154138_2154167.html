That definition is either wrong (as written) or rather misleading. Here is the usual definition:

A group is a set G with a law of composition ⋅:G2→G such that

  • Identity: there is an element 1∈G which satisfies g⋅1=1⋅g=g for all g∈G
  • Inverse: for every g∈G there is an element h∈G such that g⋅h=h⋅g=1 (this element is unique and is usually written g-1)

It is sometimes convenient to write a group as a quadruple (G,⋅,--1,1).

An example of a group is the group of bijections Sym⁢(T) on a set T, with the law of composition given by function composition. The identity is exactly the identity map idT and the inverse of a bijection f:T→T is exactly the inverse map f-1:T→T.

There are certainly groups which are not of the above form. One way of seeing this is that for |T|>2 the group Sym⁢(T) is not Abelian, meaning that the law of composition is not commutative. This is because given distinct elements x,y,z∈T, we can define bijections f1,f2:T→T by w∈T∖{x,y} w∈T∖{y,z} In other words, f1 switches x and y and leaves everything else fixed, and f2 switches y and z and leaves everything else fixed. Then (f1∘f2)⁢(y)=f1⁢(f2⁢(y))=f1⁢(z)=z (f2∘f1)⁢(y)=f2⁢(f1⁢(y))=f2⁢(x)=x are unequal, so f1∘f2≠f2∘f1.

However, ℤ with addition is Abelian, so it can't be of the form Sym⁢(T) for any set T.

What is true, however, is that every group is (isomorphic) to a subgroup of Sym⁢(T) for some set T. This is known as Cayley's Theorem.

By subgroup of a group (G,⋅,--1,1) we mean a subset H⊂G for which H is closed under ⋅ and taking inverses, and contains 1.

Given a group (G,⋅,--1,1), consider Sym⁢(G). We then associated each element g∈G to the function τg:G→G defined by τg⁢(h)=g⋅h. This is invertible, since τg∘τg-1=idG. More generally, τg∘τh=τg⋅h. This allows us to treat G like a subgroup of Sym⁢(G).
