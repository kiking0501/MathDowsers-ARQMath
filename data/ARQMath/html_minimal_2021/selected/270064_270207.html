There are some details that I haven't fully vetted but here's a long sketch which I believe should show divergence.

Define cn:=n2⁢π(mod1) and let [a,b] be any interval in the torus ℝ/ℤ. The discrepancy D⁢(N) of the sequence cn is defined to be the difference between #⁢{n≤N:cn∈[a,b]} and the expected count (b-a)⁢N.

Weyl's criterion tells us that D⁢(N)=o⁢(N). More quantitatively, the Erdős-Turán inequality states that for any integer K>0,

D⁢(N)≪NK+∑k=1K1k⁢|∑n=1Nek⁢n⁢i|.

While I imagine there are better ways to control the rightmost sum on average, it's a nice fact that we can control it pointwise using a result of Mahler that there exists an absolute constant C>0 such that

|π-pq|≫1qC.

(The current best value of C is about 7.6 due to Salikhov.) Note that |∑n=1Nek⁢n⁢i|≤2⁢(1-ek⁢i)-1 by geometric series, and Mahler's theorem controls how close k can be to a multiple of 2⁢π and thereby how close ek⁢i can be to 1. This should give us |∑n=1Nek⁢n⁢i|≪kC uniformly in N, possibly with a different C.

By choosing K to be a small power of N (something like N1/C) in Erdős-Turán, we get that D⁢(N)≤N1-c for some absolute constant c>0.

For small ϵ>0, let Aϵ be the Bohr set {n∈ℕ:|sin⁡(n)|<ϵ}. By the discrepancy bound, the counting function satisfies (ignoring a tiny error term from the non-linearity of sine):

Aϵ⁢(N):={n≤N:n∈Aϵ}=ϵπ⁢N+O⁢(N1-c).

In particular, Aϵ⁢(N)≫ϵ⁢N provided that N>ϵ-C. Using this with partial summation, we can estimate the contribution to the original sum from Aϵ:

∑n∈Aϵ1n1+|sin⁡(n)|≥∑n∈Aϵn-1-ϵ=∫1∞(1+ϵ)⁢Aϵ⁢(t)⁢t-2-ϵ⁢𝑑t≥∫ϵ-C∞Aϵ⁢(t)⁢t-2-ϵ⁢𝑑t≫∫ϵ-C∞ϵ⁢t-1-ϵ⁢𝑑t,

which simplifies to ϵC⁢ϵ.

Note that as ϵ→0, ϵC⁢ϵ→1. We're very close to getting divergence. Just apply the above calculation to the set Aϵ∖Aϵ/2. This has about the same counting function and still gives ∑n∈Aϵ∖Aϵ/21n1+|sin⁡(n)|≫ϵC⁢ϵ.

Now sum this over a dyadic sequence of ϵk=2-k so that the resulting sets are disjoint, and we get a divergent sum.
