This actually isn't quite right yet. You're construction an element of B, so you want to say that (g1,g2,…,gn,…) is not in {a1,a2,a3,…}. The different notation is critical! The parentheses are for a sequence, which is an element of B, while the brackets delimit a list of elements of B. So (g1,g2,…) is like a1, not like {a1,a2,…}.
