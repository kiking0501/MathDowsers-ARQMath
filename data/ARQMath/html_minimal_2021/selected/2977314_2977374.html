Sum to n terms of the harmonic series

We know that ∑k=1∞1k=11+12+13+⋯ diverges.

But for any natural number n, ∑k=1n1k is finite.

The question is; how to compute ∑k=1n1k for sum natural number n, explicitly?

Say n=50, how to compute 11+12+13+⋯+149+150, explicitly?

There is no explicit or exact value but accurate asymptotics ∑k=1n1k=Hn and, for large values of n Hn=γ+log⁡(n)+12⁢n-112⁢n2+O⁢(1n4) For n=50, the exact value would be H50=139432375772240549607593099044504245996706400≈4.499205338 while the above approximation would give γ+log⁡(50)+29930000≈4.499205337 which is not too bad (I hope !).
