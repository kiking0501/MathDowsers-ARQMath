@TonyK, this equation corresponds to the decomposition of the event [one hits n+1] along the two possible values of X1. If X1=1, one wants to hit n+1 starting from 1, which is the same as hitting n starting from 0 and has probability Pn. If X1=2, one wants to hit n+1 starting from 2, which is the same as hitting n-1 starting from 0 and has probability Pn-1. Ergo.
