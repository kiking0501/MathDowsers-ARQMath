@MartinSleziak: Thanks; I was about to do the same. The restriction on x,y≥0 is not needed for n even, but induction fails, unless you handle n even separately. However, Jensen works for even n and all x,y∈ℝ.

Using Jensen's Inequality

This is simply Jensen's Inequality applied to the convex function xn for n≥1 and a discrete measure. In fact, this can be generalized to any non-negative sequence (αi) so that ∑i=1kαi=1 to get (∑i=1kαi⁢xi)n≤∑i=1kαi⁢xin In this particular case, k=2 and (αi)=(12,12) yields (x+y2)n≤xn+yn2 which is the same as (x+y)n≤2n-1⁢(xn+yn)

Using the AM-GM Inequality

Note that the AM-GM Inequality applied to {xn,xn,…,xn⏞k⁢ copies,yn,yn,…,yn⏞n-k⁢ copies} yields xk⁢yn-k≤k⁢xn+(n-k)⁢ynn Applying this to the Binomial Theorem yields (x+y)namp;=∑k=0n(nk)xkyn-kamp;≤∑k=0n(nk)k⁢xn+(n-k)⁢ynnamp;=xn∑k=0n(nk)kn+yn∑k=0n(nk)n-knamp;=xn∑k=1n(n-1k-1)+yn∑k=0n-1(n-1k)amp;=2n-1(xn+yn)

Negative Values

The inequality is true if x,y≥0 or n is even. The proofs above were written assuming that x,y≥0; however, xn is convex over all ℝ when n is even, so the Jensen proof works as is. Furthermore, if n is even, changing the sign of x and/or y leaves the right side of \text ⁢ ( A ⁢ M - G ⁢ M ) alone (and positive), but might change the sign of the left side to be negative. In either case, \text ⁢ ( A ⁢ M - G ⁢ M ) holds for all x,y∈ℝ when n is even.
