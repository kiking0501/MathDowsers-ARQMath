Countability of truth-valued functions

Consider an arbitrary set A:={a1,a2,a3,…} of cardinality ℵ0. I'm trying to determine if the set of functions F:={f|f:A→{0,1}} is countably infinite. I don't see an obvious bijection to the natural numbers which leads me to think it might not be. If so, then it seems like you could construct a diagonalization argument based on introducing a g that differs from some listed fn by only one out-put -- taking g⁢(an)=1-fn⁢(an) to arrive at a contradiction. Is that the general idea? Guidance would be appreciated.

Do you mean ".. of cardinality ℵ0"? Writing ℵ without the zero subscript is sometimes used to mean the cardinality of ℝ.
Your set F is essentially the powerset of A, so by a well known argument such as the one you've provided, it's not countable

The set of functions F is essentially analogous to the set of real values between 0 and 1, expressed in binary. (It's the set of reals in the unit interval because A is infinite.) So F is uncountable.
