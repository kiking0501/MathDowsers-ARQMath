Prove that if n is even then n2 is even and if n is odd then n2 is odd?

How can I prove that if n is even then n2 is even and if n is odd then n2 is odd?

number-theory
even numbers are of the form 2⁢n and odd numbers 2⁢n+1.
Hint: all even integers can be written as 2⁢k ∀k∈ℤ and all odd integers can be written as 2⁢k+1 ∀k∈ℤ. What do you get when you square each?

Suppose n is even. Let p=n/2.

n2=4⁢p2 which must be even since p is a whole number.

Suppose n is odd and let p=n+1.

p2 is therefore even by the above rule.

n=(p-1)2=p2-2⁢p+1.

p2-2⁢p must be even so n is therefore odd.

The square of a number is even if and only if the number itself is even
