you are only interested in proving convergence, so you will probably be needing basic analysis tools for convergence of sequences.

Let ϕ⁢(x)=xp-(1+x) and notice that ϕ⁢(0)=-1, ϕ′⁢(0)=-1, ϕ′′⁢(x)≥0 and limx→∞⁡ϕ⁢(x)=∞.

In particular, there is a unique x*>0 such that ϕ⁢(x*)=x*, and for 0≤x≤x*, we have ϕ⁢(x)<0 and for x*≤x, we have ϕ⁢(x)>0.

Since x↦xp is strictly increasing, we see that η⁢(x)=x-1+xp also satisfies the same conditions regarding sign.

Let f⁢(x)=1+xp and note that f is strictly increasing.

From above we see that there is a unique x*>0 such that f⁢(x*)=x*.

If 0≤x≤x* then x≤f⁢(x)≤f⁢(x*)=x*.

If x*≤x then x*=f⁢(x*)≤f⁢(x)≤x.

Now consider the iteration xn+1=f⁢(xn) starting from any x0.

If 0≤x0≤x*, then xn is a non decreasing sequence that is bounded above, hence xn→x~ and since f is continuous, we have x~=f⁢(x~) and so x~=x*.

A similar analysis shows that if x*≤x0, then xn→x*.
