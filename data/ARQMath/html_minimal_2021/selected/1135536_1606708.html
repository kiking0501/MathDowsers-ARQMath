This problem has an interesting history from the early days of real analysis attached to it. I really like this story and the proof it contains of the problem in this question is also different from the other answers posted.

In 1827 a M. Oliver published a proof in Crelle's Journal (Journal für die reine und angewandte Mathematik) of the following theorem:

Theorem M. Oliver: A series of positive terms converges if and only if limn→∞⁡n⁢an=0.

Not long after this paper had been published Niels Henrik Abel responded with a short note noting that the theorem "does not seem to hold true" as the series ∑n=2∞1n⁢log⁡(n) diverges but limn→∞⁡n⋅1n⁢log⁡(n)=0 contradicting the theorem above.

Abel then followed it up by another short note showing that there cannot exist any function ϕ⁢(n) such that

∑k=1∞an⁢converges⇔limn→∞⁡ϕ⁢(n)⁢an=0

The proof is remarkably simple and uses the lemma below (which coincides with the problem this question asks for). I will here present Abel's original proof. It's from memory so it's not word for word but the main idea behind the proof should be intact.

Lemma: if a series of positive terms ∑k=1∞ak diverges then so does the series ∑k=1∞aka1+a2+…+ak-1.

Proof: Let sn=∑k=1nak, then

log⁡(sksk-1)=log⁡(1+aksk-1)<aksk-1

since log⁡(1+x)<x. Summing over k=2,3,…,n the left hand side telescopes to give

log⁡(sns1)<∑k=1naksk-1

Now if sn=∑k=1nak diverges then so does the logarithm of sn and it follows that ∑k=1naksk-1 also diverges.

Proof of Abel's theorem above: Assume there exist a function ϕ⁢(n) with the properties that ∑ak converges if and only if limn→∞⁡ϕ⁢(n)⁢an=0. Then ∑k=1∞1ϕ⁢(k) diverges since limn→∞⁡ϕ⁢(n)⁢1ϕ⁢(n)=1. By the lemma above this implies that ∑k=1∞1ϕ⁢(k)1ϕ⁢(1)+1ϕ⁢(2)+…+1ϕ⁢(k-1) also diverges but limn→∞⁡ϕ⁢(n)⁢1ϕ⁢(n)1ϕ⁢(1)+…+1ϕ⁢(n-1)=limn→∞⁡11ϕ⁢(1)+…+1ϕ⁢(n-1)=0

which would indicate that it is convergent. No such function ϕ⁢(n) can therefore exist.
