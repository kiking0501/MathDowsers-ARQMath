I suspect PMar is correct about the interpretation of the question: we should show there exists a set of primes S such that infinitely many terms in the arithmetic progression have exactly S as their set of prime divisors. In fact, if a is relatively prime to b, then you should be able to show the set of powers {a,a2,a3,⋯} has infinite intersection with {a,a+b,a+2⁢b,⋯}. (Hint: the sequence of powers is repeating mod b and must return back to a since a is a unit mod b.)
