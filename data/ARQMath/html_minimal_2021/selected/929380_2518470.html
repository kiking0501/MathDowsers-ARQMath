What is wrong with the following u-substitution?

We will calculate ∫02⁢πx⁢𝑑x. Let u=sin⁡(x), and observe that sin⁡(2⁢π)=0 and sin⁡(0)=0. We also have that d⁢ud⁢x=cos⁡(x)=1-u2. Hence, ∫02⁢πx⁢𝑑x=∫00sin-1⁡(u)1-u2⁢𝑑u=0. This is very obviously wrong, but I am not sure how to explain the error formally.

Edit: Thanks for the responses and in particular the link below to the related problem! The error is indeed caused by the substitution x=sin-1⁡(u). The integration is performed over [0,2⁢π] which is outside the range of the sin-1 function.

Remark The error is slightly better disguised when calculating ∫-112⁢x1+x2⁢𝑑x.

Let u⁢(x)=1+x2, and observe that u⁢(1)=u⁢(-1)=2. Then since d⁢x=12⁢x⁢d⁢u, we have that ∫-112⁢x1+x2⁢𝑑x=∫221u⁢𝑑u=0. This time, no trigonometric substitution is used, but it is still an incorrect proof for the same reason as above. A correct proof can be obtained by using the fact that x↦2⁢x1+x2 is odd.

This example is more disturbing because the procedures above are entirely intuitive and yield the correct result. It seems to me that students when taught integration by substitution of definite integrals should also be taught that great care be exercised in checking the range of integration, particularly when the (apparent) substituting function is not invertible in that range.

Many things. For one, we are really letting x=arcsin⁡u. But there is no u whose sine is \gt⁢1.
See a similar question (http://math.stackexchange.com/q/669233/72031) and its answer via the substitution u=sin⁡x (http://math.stackexchange.com/a/669276/72031).
I believe I answered the fundamental idea behind this question [in this answer](https://math.stackexchange.com/a/2010204/9754) to another question concerning u-substitution. This is in particular very closely related to your second example.

1. Let u=sin⁡x

so that d⁢u=cos⁡x⁢d⁢x.

Now, for x∈[0,π2], \begin{align}\cos x &= \sqrt{1-u^2},\\x&=\arcsin u;\end{align} for x∈[π2,3⁢π2], \begin{align}\cos x &= -\sqrt{1-u^2},\\x&=\pi -\arcsin u;\end{align} for x∈[3⁢π2,2⁢π], \begin{align}\cos x &= \sqrt{1-u^2},\\x&=2\pi +\arcsin u.\end{align}

Therefore \begin{align}\int^{2\pi}_0 x \,dx&=\int^\frac{\pi}{2}_0 x \,dx + \int^\frac{3\pi}{2}_\frac{\pi}{2} x \,dx + \int^{2\pi}_\frac{3\pi}{2} x \,dx\\&=\int^1_0 \frac{\arcsin u}{\sqrt{1-u^2}}\,du + \int^{-1}_1 \frac{\pi-\arcsin u}{-\sqrt{1-u^2}} \,du + \int^0_{-1} \frac{2\pi + \arcsin u}{\sqrt{1-u^2}} \,du\\&=2\pi\int^0_{-1} \frac{\,du}{\sqrt{1-u^2}}-\pi\int^{-1}_1 \frac{\,du}{\sqrt{1-u^2}}\\&=2\pi \bigg[\arcsin u\bigg]_{-1}^0 -\pi\bigg[ \arcsin u\bigg]_1^{-1}\\&=2\pi^2.\end{align}

2. Re: the Remark in the original question: the integration-by-substitution proof for ∫-112⁢x1+x2⁢𝑑x supplied by the OP is actually perfectly valid (so the example isn't disturbing!), even though the substitution function u=1+x2 is not injective (and certainly not invertible) on [-1,1].

Integration by substitution does not intrinsically require that the substitution function be invertible/bijective - or even injective or monotonic - on the interval of integration; but if the substitution function is not injective on the interval of integration, the integral may (or may not) need to be split up over subintervals before the substitution can be simply applied. (This applies to indefinite integrals too.)
