For all this assume f≠0.

First, if x∈M then f⁢(x)=1⁢≤⁢∥f∥⁢∥x∥ and thus ∥x∥⁢≤⁢1∥f∥. The inequality holds when you pass to the infimum.

Next note that ∥f∥=supx∈E,∥x∥=1⁡|f⁢(x)|. For that reason for any ϵ>0 there exists a norm one vector xϵ∈E so that ∥f∥⁢≥⁢|f⁢(xϵ)|⁢≥⁢∥f∥-ϵ. There is an ϵ after which f⁢(xϵ) is not allowed to be 0. Now ∥xϵf⁢(xϵ)∥=1|f⁢(xϵ)|, which is sandwiched by 1∥f∥ and 1∥f∥-ϵ. Further it is clear that xϵf⁢(xϵ)∈M. Because you can do this for any ϵ you retrieve that infx∈M⁡∥x∥⁢≥⁢1∥f∥.
