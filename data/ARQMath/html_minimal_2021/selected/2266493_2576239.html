We can write this version of the identity principle (IP) as follows: Let f:ℝn→ℝ be real-analytic. If f=0 on a set E of positive Lebesgue measure, then f=0 everywhere in ℝn.

Proof for n=1: We have m1⁢(E∩[-r,r])>0 for some r>0. For such an r the set E∩[-r,r] is infinite, hence has a limit point in [-r,r]. We then have f=0 on ℝ by the standard IP.

For higher dimensions we need a simple fact: Write points in ℝn+1 as (x,t), with x∈ℝn and t∈ℝ. If f is real analytic on ℝn+1, then for each fixed t∈ℝ, the function x→f⁢(x,t) is real analytic on ℝn, and for each fixed x∈ℝn, the function t→f⁢(x,t) is real analytic on ℝ.

We proceed by induction. First, suppose we have the IP for n. Assume we have a real analytic f on ℝn+1 such that f=0 on a set E⊂ℝn+1 with mn+1⁢(E)>0. By Fubini,

mn+1⁢(E)=∫ℝ∫ℝnχE⁢(x,t)⁢𝑑mn⁢(x)⁢𝑑m1⁢(t).

Since mn+1⁢(E)>0, we have the inner integral above positive for t belonging to some set A⊂ℝ with m1⁢(A)>0. By the induction hypothesis, for each t∈A, x→f⁢(x,t) vanishes identically on ℝn. Thus we have f≡0 on ℝn×A. Now fix x∈ℝn. Then t→f⁢(x,t)=0 for all t∈A. By the IP for n=1, this function vanishes on ℝ. It follows that f=0 on ℝn×ℝ=ℝn+1, and we're done.
