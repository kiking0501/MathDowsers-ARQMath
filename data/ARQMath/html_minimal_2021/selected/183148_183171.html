One problem with Euler's function

Let n=100. Is ϕ⁢(n) the smallest number λ such that if gcd⁡(a,n)=1 then aλ≡n1 ? Generalize this observation.

I suppose the answer is no. I've read something about ordn⁢(a) but group theory rather goes beyond my discrete mathematics course. I don't know what exactly author of this task had on mind, but maybe what is the simplest way to find such smallest λ?

number-theory totient-function

Let λ be the smallest positive number such that aλ≡1(modn) where (a,n)=1.

By Euler's totient theorem, aϕ⁢(n)≡1(modn)

Let ϕ⁢(n)=r⁢λ+s where 0 ≤s<λ

So,1⁢≡⁢aϕ⁢(n)=ar⁢λ+s=(aλ)r⋅as⁢≡⁢as

So, there comes another s<λ, as⁢≡⁢1(modn), which is impossible as λ is the smallest=>s=0.

=> λ|ϕ⁢(n)

Unfortunately, we need to iterate the divisors of ϕ⁢(n) to find the smallest positive λ which is called ordn⁢a.

We know, n has a primitive root if it is of the form 2,4,pa,o⁢r⁢2⁢pa, where p is an odd prime and integer a≥0.

So, we don't need to traverse up to ϕ⁢(n) if n is not of any the above form.

For example, ϕ⁢(100)=40. As 100=4*25, it can not have a primitive root, no a co-prime to 100 can have order 40,o⁢r⁢d100⁢a must be one of 1,2,4,5,8,10,20.

According to Carmichael Function, we can off-course, make reduction of the traversal. The reduction will increase with the number of unique prime divisors.

Using Carmichael Function, o⁢r⁢d100⁢a must divide lcm(ϕ⁢(4)⋅ϕ⁢(25))=l⁢c⁢m⁢(2,20)=20.

We don't really need to to iterate all divisors of $\phi(n)$.
That ord100⁢(a) is a proper divisor of 40 is not adequate grounds to conclude that ord100⁢(a)∣20 (even though it is true by @lhf's correct answer).
