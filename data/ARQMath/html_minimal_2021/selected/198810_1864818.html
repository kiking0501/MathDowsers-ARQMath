Proof that every repeating decimal is rational

Wikipedia claims that every repeating decimal represents a rational number.

According to the following definition, how can we prove that fact?

Definition: A number is rational if it can be written as pq, where p and q are integers and q≠0.

Here's an explanation via an application to a conrete example. x=0.15⁢504⏞⁢ 540 540 540⁢…  (“540” repeats.) Since the repetend has three digits, we multiply by 1000 by moving the decimal point over three places: 1000⁢xa⁢m⁢p;=a⁢m⁢p;1a⁢m⁢p;5a⁢m⁢p;5a⁢m⁢p;.a⁢m⁢p;4a⁢m⁢p;0a⁢m⁢p;5a⁢m⁢p;4a⁢m⁢p;0a⁢m⁢p;5a⁢m⁢p;4a⁢m⁢p;0a⁢m⁢p;5a⁢m⁢p;4a⁢m⁢p;0a⁢m⁢p;…xa⁢m⁢p;=a⁢m⁢p;a⁢m⁢p;a⁢m⁢p;0a⁢m⁢p;.a⁢m⁢p;1a⁢m⁢p;5a⁢m⁢p;5a⁢m⁢p;4a⁢m⁢p;0a⁢m⁢p;5a⁢m⁢p;4a⁢m⁢p;0a⁢m⁢p;5a⁢m⁢p;4a⁢m⁢p;0a⁢m⁢p;…Now subtract: ⁢999⁢xa⁢m⁢p;=a⁢m⁢p;1a⁢m⁢p;5a⁢m⁢p;4a⁢m⁢p;.a⁢m⁢p;2a⁢m⁢p;5 So we have 999⁢x=155.25=155+14 4×999⁢x=(4×155)+1 3996⁢x=621 x=6213996=27×2327×148=23148=integerinteger⏞.

And of course it should be clear why this would still work with any repeating decimal expansion that you choose.
