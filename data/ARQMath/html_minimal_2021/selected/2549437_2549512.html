Ok, you can see easily that f⁢(x) is continuous at x=1 because f⁢(1)=-2 and limx→1-⁡f⁢(x)=limx→1+⁡f⁢(x)=-2

A function f⁢(x) is continuous at x=c if it is defined at x=c and limx→c=f⁢(c)

Now we differentiate the function in each piece. If the result is continuous then the function is differentiable

f′⁢(x)={2⁢x,x<10,x=112⁢x-1,x>1

we have

limx→1-⁡f′⁢(x)=2;limx→1+⁡f′⁢(x)=+∞ the limit doesn't even exist! So the function is not differentiable at x=1.

Bonus

The derivative which tends to +∞ has an interesting geometric meaning. It means that the tangent to the graph of the function is vertical at that point. As you can see in the picture below, the graph has two tangents at point (1,-2), which means it is not differentiable, indeed.
