Write f⁢(x)=x3-3⁢x2+5⁢x=(x-1)3+2⁢(x-1)+3. From (1) we see that f⁢(x)-3 is a sum of odd powers of x-1, so f is rotationally symmetric about (1,3). This can be expressed f⁢(1+x)+f⁢(1-x)=6⁢for every x. Since f is strictly increasing, (2) implies that(*) f⁢(u)+f⁢(v)=6 To answer the original question, note that f⁢(α)+f⁢(β)=17+(-11)=6.

Proof of (*): If f⁢(u)+f⁢(v)=6, then let x:=u-1. Then f⁢(1-x)=(2)6-f⁢(1+x)=6-f⁢(u)=f⁢(v). Since f is one-to-one, we deduce 1-x=v and so u+v=(1+x)+(1-x)=2.
