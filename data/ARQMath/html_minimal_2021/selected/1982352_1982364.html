Using those is easy, notice that the least common multiple of n1,n2,…⁢nk is equal to the product if and only if they are pairwise coprime.

So if n1,n2⁢…⁢nk are not pairwise coprime we have that l<|G|, and we have by the first result that the order of every element in G is smaller than |G|, so the group cannot be cyclic.

On the other hand if n1,n2,…⁢nk are pairwise coprime then l=|G|, so by result two we have that there is an element g∈G with o⁢(g)=|G|, and so the group is cyclic.
