ARCH(1) process is a Markov process

I have a question about the ARCH(1) process. Let (Ω,ℱ,P) be a probability space, let (Zt)t∈ℤ be a sequence of i.i.d. real-valued random variables with mean zero and variance one. A stochastic process (Xt)t∈ℤ is an ARCH(1)-process if it is strictly stationary and if, for all t and some process (σt) with σt>0 for every t, one has Xt=σt⁢Zt and σt2=α0+α1⁢Xt-12, where α0>0 and α1≥0. Let ℱt be the natural filtration of the process (Xt), i.e. ℱt=σ(Xs;s≤t).

I want to prove that (Xt) has the Markov property, i.e. for each B∈ℬ⁢(ℝ) and for all s,t∈ℤ with s<t, one has P[Xt∈B∣ℱs]=P[Xt∈B∣Xs]. Is that possible?

For s < t we can write:

Xt=Zt⁢α0+α1⁢Xt-12=Zt⁢α0+α0⁢α1⁢Zt-12+α12⁢Zt-12⁢Xt-12=…=Zt⁢∑k=0t-s-1α0⁢α1k⁢∏i=1kZt-i2+α1t-s⁢∏i=1t-sZt-i2⁢Xs2 By construction Zt,…,Zs+1 is independent to ℱs, so for a measureable function g≥0 it holds:

𝔼⁢[g⁢(Xt)|ℱs]=𝔼⁢[g⁢(Zt⁢∑k=0t-s-1α0⁢α1k⁢∏i=1kZt-i2+α1t-s⁢∏i=1t-sZt-i2⁢Xs2)|ℱs]=𝔼⁢[g⁢(Zt⁢∑k=0t-s-1α0⁢α1k⁢∏i=1kZt-i2+α1t-s⁢∏i=1t-sZt-i2⁢x2)]x=Xs=𝔼⁢[g⁢(Xt)|Xs]

Here the notation x=Xs denotes that we don't integrate over Xs anymore, but set the value of x in the expection by evaluating Xs first. In the last step we just go back the steps before but with σ-Algebra Xs. This shows ℙ[Xt∈B|ℱs]=ℙ[Xt∈B|Xs], but it is not the full Markov property. For the Markov property you need additionally ℙ[Xt∈B|Xs]=ℙ[Xt-s∈B|X0=x]x=Xs, but this follow from the calculations above, because the Zt are i.i.d.
