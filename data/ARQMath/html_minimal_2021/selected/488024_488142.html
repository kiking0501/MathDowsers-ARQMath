Is it true that every element of 𝔽p has an n-th root in 𝔽pn?

It is not hard to prove that every element of 𝔽p has a square root in 𝔽p2: take any a∈𝔽p and consider the polynomial f=X2-a. If f has a root in 𝔽p, then we are done. Otherwise f is irreducible over 𝔽p, let β∈𝔽p¯ be a root of f, then the extension 𝔽p⁢(β) has degree 2 over 𝔽p, and hence by the uniqueness of finite fields we have 𝔽p⁢(β)=𝔽p2, so β∈𝔽p2. With exactly the same proof we can see that every element of 𝔽p has a cube root in 𝔽p3.

But I don't know if a similar statement holds for an arbitrary n. If n>3, the fact that Xn-a has no root in 𝔽p is no longer equivalent to being irreducible. So the thing that we would need to show in this case is that Xn-a has an irreducible factor whose degree is a divisor of n. I don't see how to do that.

The problem would be solved if we could prove that 𝔽pn contains an element β whose multiplicative order is n⁢(p-1), since in that case βn is a primitive root mod p. But, from the fact that 𝔽pn× is a cyclic group, if follows that the previous condition is equivalent to n∣pn-1+pn-2+…+p+1, which does not necessarily hold (it does in some special cases, such as p≡1(modn)).

I think that the following extension to prof. Lubin's argument settles the question in the affirmative.

First let's write n=n1⁢n2, where all the prime factors of n1 are also factors of p-1, and gcd⁡(n2,p-1)=1. For all a∈𝔽p the equation xn2-a has a root y in the prime field, so it suffices to show that y has an n1th root in 𝔽pn1⊆𝔽pn.

Lemma. Assume that q is a prime, and that the finite field K contains a primitive qt⁢h root of unity ζ. Let α∈K be arbitrary. Then the polynomial f⁢(x)=xq-α has a root in the unique degree q extension L of K.

Proof. If f⁢(x) has a root in K, then that root is also in L. If no such root exists in K, then such a root β exists in K¯. Because the other zeros of f⁢(x) are gotten from β by multiplying it with a power of ζ, we see that K⁢[β] is the splitting field of f⁢(x). Let σ be a generator of the cyclic Galois group Gal⁡(K⁢[β],K). Then σ⁢(β)=β⁢ζℓ for some exponent ℓ coprime to q. As q is a prime and ζ is fixed by σ, we see that σ is of order that is a multiple of q. Therefore its order is exactly q, and we can conclude that K⁢[β]=L. Q.E.D.

The claim follows easily from the Lemma. If q is any prime factor of n1, then q∣p-1, so the prime field already contains the necessary roots of unity. Hence so do all its extensions, and the Lemma bites. More precisely, if d⁢∣d⁢q∣⁢n1, where q∣p-1 is a prime, then, by induction hypothesis the equation xd=a has a solution x∈𝔽pd. By the Lemma, the equation yq=x has a solution y∈𝔽pd⁢q, and y is then a solution of yd⁢q=a. Repeating this step enough many times gets us to n1 settling the claim.
