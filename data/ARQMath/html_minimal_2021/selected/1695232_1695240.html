Yes, the answer is correct.

limx→a⁡(f⁢(x)+g⁢(x))=limx→a⁡f⁢(x)+limx→a⁡g⁢(x) if limx→a⁡f⁢(x) and limx→a⁡g⁢(x) exist.

This can be shown as follows:

Let limx→a⁡f⁢(x)=L1 and limx→a⁡g⁢(x)=L2 Given ϵ⁢\gt⁢0, there exist δ1,δ2⁢\gt⁢0 such that |f⁢(x)-L1|⁢\lt⁢ϵ and |g⁢(x)-L2|⁢\lt⁢ϵ for all |x-a|⁢\lt⁢min⁡(δ1,δ2).

Thus, |f⁢(x)+g⁢(x)-L1-L2|≤|f⁢(x)-L1|+|f⁢(x)-L2|<ϵ+ϵ=2⁢ϵ for all |x-a|⁢\lt⁢min⁡(δ1,δ2).

Hence, limx→a⁡(f⁢(x)+g⁢(x))=L1+L2

Note that the converse is not true.

It can be similarly shown that given the individual limits, limx→a⁡f⁢(x)⁢g⁢(x)=limx→a⁡f⁢(x)×limx→a⁡g⁢(x) And, limx→a⁡k⁢f⁢(x)=k⁢limx→a⁡f⁢(x) for a constant k.
