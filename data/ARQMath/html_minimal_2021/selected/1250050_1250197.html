Yes, one can give a nicer description of a base for the relative topology on P.

  • HINT: Let p∈P; what is p⁢ℤ∩P?

For the second part of the problem, let’s ask what integers are not in cl⁡P. An integer a is not in the closure of P if there is a b∈ℤ such that (a+b⁢ℤ)∩P=\varnothing. Dirichlet’s theorem tells you that if a and b are relatively prime, then (a+b⁢ℤ)∩P is infinite, so we need to look for some b that is not relatively prime to a; how about a itself? Then a+b⁢ℤ=a⁢ℤ; if a is composite, are there any primes in a⁢ℤ? (Be a little careful here: there are actually two cases, one in which a is, so to speak, really composite, and one in which it’s simply by definition not prime.)
