  • The supremum in the definition of uniform integrability supn∈ℕ⁡∫{|Xn|≥R|Xn|⁢𝑑P is actually ∫{|X1|≥R}|X1|⁢𝑑P which converges to 0 when R→+∞, using monotone convergence for example.
  • Let j a fixed integer, and Ej,n:={Xn≥j}. Since the series ∑n≥1P⁢(Ej,n) is divergent and the events {Ej,n} are independent, by Borel-Cantelli lemma, P⁢(lim supn⁡Ej,n)=1. Hence for almost all ω, there exist Sω,j⊂ℕ infinite such that for each element n of this set Xn⁢(ω)≥j. Hence lim supn→+∞⁡Xn⁢(ω)≥j. Since it's true for almost all ω, we conclude that lim supn→+∞⁡Xn is not integrable.
