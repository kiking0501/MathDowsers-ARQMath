In addition to the excellent answers below, I will add that this group is isomorphic to S3.
@Shalab Indeed, and this can be seen by how the functions act as permutations on the three-element set {0,1,∞} if we extend their definitions in the obvious manner.
Some post discussing that this group is isomorphic to S3 can be found [here](http://math.stackexchange.com/questions/434912/show-that-the-set-of-functions-under-composition-is-isomorphic-to-s-3) and [here](http://math.stackexchange.com/questions/668122/isomorphisms-between-group-of-functions-and-s-3).

First of all, composition of functions is always associative (in a way this is "the mother of all associativities"), because for all x ((f∘g)∘h)⁢(x)=(f∘g)⁢(h⁢(x))=f⁢(g⁢(h⁢(x)))=f⁢((g∘h)⁢(x))=(f∘(g∘h))⁢(x).

Thus, what you need to show about your set X of functions is

  • if f,g∈X, then f∘g in X.
  • there is a neutral element (which is f1, obviously)
  • for each f∈X there is an inverse element, i..e some g with g∘f=f∘g=f1

One can also try this the other way around: The set of bijections ℚ∖{0,1}→ℚ∖{0,1} is certainly a group G. If we start with f2,f3∈G, there is certainly a smallest subgroup ⟨f2,f3⟩ of G that contains both f2 and f3. To find this group (noting that f2,f3 are their own inverses), we can simply keep playing with composing found functions with f2 and f3 until we no longer find new functions - a process that fortunately terminates and gives us exactly the set X.

Another interesting point to note is that all elements of X have the form f⁢(x)=a⁢x+bc⁢x+d with a⁢d-b⁢c≠0 (where we ignore the fact that f is not defined at x-dc, say we view these as functions on ℝ∖ℚ). This reminds us of invertible matrices A=(abcd) with det⁡A=a⁢d-b⁢c≠0. Do you see how the composition of two fucntions of this "matrix type" relates to a well-known operation among the corrsponding matrices?
