Smooth map S1→S2 can not be surjective

Why cannot a smooth (or piecewise linear) map S1→S2 be surjective? There are space-filling curves, but the usual examples have very "twisty" definitions.

UPD A bit of background for this problem. It's part of he proof that all the normal vector fields on S1⊂ℝ4 are homotopic. Once it's proved that every map S1→S2 is homotopic to piecewise linear or smooth that is what is left to prove the statement.

This is an immediate consequence of Sard's theorem (see [wikipedia][1]). Namely, the critical values of a map f:S1→S2 are a null set in S2. At a point, the derivative is a linear function from a 1 dimensional vector space to a 2-dimensional vector space, so every point in the image is a critical value. [1]: https://en.wikipedia.org/wiki/Sard%27s_theorem

@StanO.: Sard's theorem *is* the analogue of the PL answer given below. The point of that answer is essentially that a single linear lower-dimensional simplex has measure zero, and you're taking the union of these measure zero things. You are not going to show that the image of a smooth curve in S2 is nowhere dense without it.
