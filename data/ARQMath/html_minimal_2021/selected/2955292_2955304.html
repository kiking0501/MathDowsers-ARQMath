I think a proof by contradiction is not necessary.

Since γ is regular, we can reparametrise it to a unit speed curve. By assumption, for every t∈ℝ there exists a λ⁢(t)∈ℝ (λ depends on t) such that γ⁢(t)+λ⁢(t)⁢γ′⁢(t)=P. Deriving this gives (1+λ′⁢(t))⁢γ′⁢(t)+λ⁢(t)⁢γ′′⁢(t)=0. Since γ has unit speed, it follows that γ′′⁢(t)⋅γ′⁢(t)=0, so γ′′⁢(t) is a normal vector.

If γ′′⁢(t0)≠0 at some point, then by continuity it is non-zero in some interval I containing t0. Then λ⁢(t)=0 and thus γ⁢(t)=P for t∈I, which would contradict the fact that γ is regular. Hence γ′′⁢(t)=0 must hold for all t. Integrating this twice gives that γ is a line or line segment.

Edit: The proof above uses that γ∈C2. User Laz suggested in the comments that this statement should also hold if γ∈C1. Here is an argument for the case γ∈C1.

Write γ⁢(t)=(x⁢(t),y⁢(t)) and P=(a,b). Then \eqref{equa} becomes x⁢(t)+λ⁢(t)⁢x′⁢(t)=a,y⁢(t)+λ⁢(t)⁢y′⁢(t)=b.

If λ⁢(t0)≠0 in a point t0, then λ⁢(t)≠0 on a open interval containing t0. On this interval we can integrate to obtain x⁢(t) This is a parametrisation of a line. This local argument shows that γ piecewise consists of line segments.

If λ⁢(t)=0 on some (open) interval, we get as before that γ⁢(t)=P on the interval and this contradicts the regularity. So λ⁢(t)=0 only in isolated points. Now, note that the curve γ⁢(ℝ) is connected (γ is continuous and ℝ is connected). Moreover, γ′⁢(t) is continuous so the slopes of all the line segments have to be the same. Hence γ is a line.
