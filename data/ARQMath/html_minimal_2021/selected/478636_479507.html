Assume that we are choosing n objects from {a,⋯,a,b,⋯,b,c1,⋯,cn}
where there are n a's and n b's.

If we choose k elements from 0,1,2,3,4,…, there are k+1 ways to do this;

and then we still have to choose n-k elements from {c1,⋯,cn},
which can be done in (nn-k)=(nk) ways.

Therefore the total number of choices is given by ∑k=0n(k+1)⁢(nk).

Since (x+1)n=∑k=0n(nk)⁢xk, x⁢(x+1)n=∑k=0n(nk)⁢xk+1; so differentiating gives (x+1)n-1⁢[(n+1)⁢x+1]=∑k=0n(k+1)⁢(nk)⁢xk.

Now substituting x=1 gives that ∑k=0n(k+1)⁢(nk)=2n-1⁢(n+2).

Another way to evalutate this would be to split up the sum as ∑k=0nk⁢(nk)+∑k=0n(nk) to get n⁢2n-1+2n=2n-1⁢(n+2).
